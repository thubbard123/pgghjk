Opening Zipped Projects

After you have zipped a project, anyone who also uses can open and unzip the file, retuning the project to its normal state so that it can be edited.

how to open a zipped project
  1. Double-click the zipped FLPRJZIP file. The application opens with the Unpack dialog displayed.
  2. Click the Browse button and select a folder to store the project, clicking OK when you are finished.
  3. Click Unpack. The project is unzipped and the files are placed in the location you specified. A message asks if you want to open the project. Click Yes if you do. Otherwise, you can open the project later from that location.
 note

If you unpack a project that is bound to source control, the files in the unpacked project become writable.
