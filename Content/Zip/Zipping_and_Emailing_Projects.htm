Zipping and Emailing Projects

This topic deals with the second method. Also see Zipping Projects.

how to zip and email a project
  1. Open a project.
  2. In the Save section select Zip Project>Zip and Email Project.

    compresses the project and places it into a single file with an.flprjzip extension. The file is attached to an email message, ready for you to send to others.

    When individuals receive the email, the can unzip the file by double-clicking it (if they also have installed). See Opening Zipped Projects.

 note

Alternatively, you can use a more robust export feature, with additional options, to zip a project (or parts of one). See Exporting Projects.
