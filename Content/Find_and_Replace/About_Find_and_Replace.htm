About Find and Replace

From time to time, you may need to search for (and replace) specific text or elements in your project.

From time to time, you may need to search for (and replace) specific text in your documents.

provides the following find and replace features (available from the Home ribbon or the Edit menu).

You can open the Quick Find widget, which gives you the flexibility to search through the current document, to match the case of the text, and to use regular or wildcard expressions. For more information see Using the Quick Find Widget.

find and replace options
  • Quick Find and Quick Replace Widgets See Using the Quick Find and Quick Replace Widgets.
  • Find and Replace in Files See Using the Find and Replace in Files Feature.
  • Find Results 1 From the Home ribbon, you can click the down arrow next to Find and Replace in Files and select Find Results 1. This opens the Find Results 1 window pane, which lets you see results from the Find and Replace in Files window pane. You can also click a button to save the results to a CSV file.
  • Find Results 2 From the Home ribbon, you can click the down arrow next to Find and Replace in Files and select Find Results 2. This opens the Find Results 2 window pane, which is a secondary window pane that lets you see results from the Find and Replace in Files window pane. You can also click a button to save the results to a CSV file.
  • Find Elements You can open the Find Elements window pane, which is used for finding files containing particular elements—custom tags, MadCap-specific features (e.g., drop-down text, concepts, equations), markup, inline styles, and style classes. You have the flexibility to look in all files in the project or only in the open documents. The results of the search are displayed in either the Find Results 1 window pane or in the Find Results 2 window pane, depending on your selection. For more information see Using the Find Elements Feature.
