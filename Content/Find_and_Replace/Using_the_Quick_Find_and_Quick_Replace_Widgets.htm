Using the Quick Find and Replace Widgets

how to open the quick find widget
  1. Open a file.

    • Ribbon In the Find and Replace section select Quick Find.

    • Keyboard Shortcut Press CTRL+F.
 how to open the quick replace widget
  1. Open a file.

    • Ribbon In the Find and Replace section select Quick Replace.

    • Keyboard Shortcut Press CTRL+H.
 how to use the quick find and quick replace widgets

Click to select any of the following filter options:

  • Whole word
  • Match case

  • Regular Expressions If you select this option, you can use regular expressions when you look for content.

    about regular expressions

  • Wildcards If you select this option, you can use wildcards when you look for content.

    about wildcards
  • Source text finds only occurrences of the text that appear in source (untranslated) documents.
  • Target text finds only occurrences of the text that appear in target (translated) documents.
  • Find and Replace in Files This option opens the Find and Replace in Files window pane, which lets you perform more advanced searches, including global searches throughout the project. See Using the Find and Replace in Files Feature.

If you do not see any search results, be sure that a filter is selected.
