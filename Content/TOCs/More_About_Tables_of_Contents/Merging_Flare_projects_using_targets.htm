Merging Flare Projects Using Targets

how to merge multiple flare projects on your computer

  1. Open the TOC in the Flare project that will serve as the parent project.

  2. In the TOC Editor, select the TOC entry or book where you want to link the output from another Flare project. (The TOC of the linked project will be inserted at that spot in the "master" TOC.)
  3. In the local toolbar of the TOC Editor, click . The Properties dialog opens.
  4. Select the General tab.
  5. Click Select Flare Project and Target. The Link to Flare Project and Target dialog opens.

    • Click Browse for Project. Then click Project File and find/select a Flare project to which you want to link.

      OR

    • Click Select Recent Project. Then select a recently opened Flare project from the list on the right.
  7. From the Select Target field, choose a specific target in the Flare project. Make sure you select a target that uses the same output type (e.g., , , ) as the target that you will build in the current project.
  9. (Optional) If you want the merged TOC to replace the entry that you selected, do the following.
    1. Select the Advanced tab.
    2. Click the check box labeled When merging, replace node with merged TOC.
