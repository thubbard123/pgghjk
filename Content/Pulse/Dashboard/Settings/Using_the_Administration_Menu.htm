Using the Administration Menu

Administration Menu

In the Pulse dashboard, the Administration menu is available to members of the Administrators group. With the commands in this menu, you have the ability to modify the Pulse web server settings or to run reports.

settings

The options in the Administration>Settings menu include:

  • Database View the Pulse database settings. See Database.
  • General View and configure the general settings for the Pulse web server. See General Settings.
  • Groups Manage the default user groups and create new groups. See Managing Groups.
  • Suggested Feeds Configure suggested feeds for your user groups. See Suggested Feeds.
  • Users Add new users and perform user administration tasks. See Managing Users.
  • Profiles Configure the elements and fields that are available for user profiles. See Configuring Profile Settings.
  • Moderation Configure the moderation feature in Pulse. You can add reasons for new posts being rejected by moderators. See Configuring Moderation—Rejection Reasons.
  • Log View Pulse server events, errors, processes and tasks. See Log.
  • Communities View a list of Flare communities that integrate with your Pulse systems. See Communities.
  • Synonyms Manage a list of synonyms to enhance the search capabilities of a Flare community. See Managing Synonyms.
 reports

The functions in the Administration>Reports menu give you access to a variety of usage reports for both the Pulse system and your Flare communities. See Reports.
