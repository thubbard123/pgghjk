Internal Analyzer

You can use the internal Analyzer by selecting View>Project Analysis. This opens a submenu that allows you to analyze your project for the following:

  • Broken Bookmarks See Viewing Broken Bookmarks.
  • Broken Links See Viewing and Fixing Broken Links.
  • Database Errors You can See Viewing Database Errors.
  • Files With Changes See Viewing Files with Changes.
  • Files With Annotations See Viewing Files with Annotations.
  • Topics Not In Index You can See Viewing Topics Not Indexed.
  • Topics Not In Selected TOC See Viewing Topics Not in a TOC.
