  3. You can move a block of content by dragging and dropping the appropriate tag bar, or by using options from the context menu.drag and drop

    1. Click the tag bar that is shown the left of the content you want to move and drag the paragraph or other block tag up or down to the place where you want to move it. A blue arrow helps guide you, showing you where the content will be inserted when you release the mouse button.

      You can move the content to another place as long as it is at the same level as the current tag.

    2. Release the mouse button to drop the content to its new location.
    context menu
    1. Right-click the tag bar that is shown the left of the content you want to move.
    2. In the context menu, select Move and then choose one of the following options in the submenu to move the paragraph or other content.

      • Move Up This moves the content above the block of content that is just before it (on the same level).

      • Move Down This moves the content below the block of content that is directly under it (on the same level).

      • Move Left This moves the content out of the parent tag that contains it. It then moves that child tag to the left and therefore below the original parent tag.

      • Move Into Previous This moves content above and to the right, so that it is contained in the tag that is just above it. It therefore becomes a child tag within that parent tag. You might use it, for example, to move <p> tags within <div> tags.

      The options that are shown in the context menu depend on the tag bar that you have selected. In some cases, none of the options may be available. Instead, you might need to use the drag-and-drop method, or perhaps cut and paste the content into the new location.
