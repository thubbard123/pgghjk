Opening a Collection

There are multiple ways to open a Mimic movie collection that you have created.

how to open a collection using the open dialog

  1. Do one of the following:

    • Ribbon Select File>Open.

    • Keyboard Shortcut Press CTRL+O on the keyboard.

  2. In the Open dialog, navigate to the MIPRJ file and select it. Unless you selected a different location when creating the collection, a Mimic collection file is stored by default in the Documents\My Mimic Projects folder.

  3. Click Open. The collection is loaded in the Collection Editor.
 how to open a collection by dragging a file from windows
  1. Open Windows File Explorer and navigate to a folder containing a collection file.
  2. Launch .

  3. Drag the collection file from Windows File Explorer to the application window.

    You can also use this method to open any file type that is supported in .
