About Movie Collections

A movie collection is a way to combine two or more individual movies into a group of related, linked movies. You might find this useful, for example, if you are creating a video tutorial for your users and want one lesson (i.e., movie) to smoothly lead into the next.

When you create a collection, a file with an .miprj extension is created (e.g., MyMovieCollection.miprj). When you want to access the collection in the future, this is the file to open. See Mimic Files.

note

Creating a movie collection in Mimic is optional. Alternatively, you can produce individual movies, separate from any collection. See Recording Movies.
