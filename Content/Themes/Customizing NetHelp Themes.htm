Customizing NetHelp Themes

The following themes are included with NetHelp targets. These themes are the starting point for creating your customized themes.

Table of Themes

Themes Available1

Description

Responsive2 (default)
  • Designed to work well on both desktop and mobile platforms
  • Adapts to the window size automatically
  • Works on Android 4.0 or later and iOS; on the desktop, IE9 or later
  • Language will be English by default
  • Style Sheets available include jquerymobile\dark, jquerymobile\default, jquerymobile\green, and jquerymobile\light

Responsive Logo2
  • Identical to Responsive theme, but includes a default logo in theme settings

Tabs
  • Tripane window with Contents, Index, Search tabs at left
  • Breadcrumbs above content
  • Previous, Next, Home, Print, Email buttons
  • Language will be English by default; the default stylesheet jquery-ui\green-sheen

Tabs Logo
  • Identical to Tabs theme, but includes a default logo in theme settings

Accordion
  • Tripane window with Contents, Index, Search accordion buttons  at left
  • Breadcrumbs above content
  • Previous, Next, Home, Print, Email buttons
  • Language will be English by default; the default stylesheet jquery-ui\green-sheen

Accordion Logo
  • Identical to Accordion theme, but includes a default logo in theme settings

1The Responsive and Responsive Logo themes are based on jQueryMobile 1.3.2. The Tabs, Tabs Logo, Accordion, and Accordion Logo themes are based on jQuery UI.

2The Responsive and Responsive Logo themes adapt to the current window size (screen resolution). If the window size is too small, the side panel is closed automatically so that only the topic is displayed. In addition, visual elements such as breadcrumbs, buttons, and related topics change their appearance and size. These themes also include a Favorites tab, which allows end users to save favorite topics (stored on the user side).
How to Create a New Theme
  1. On the Home ribbon, click the down arrow next to the Theme button and choose Add New Theme. The Add New Theme dialog opens.
  2. Name the new theme, and choose the Source theme (Responsive, Responsive Logo, Tabs, Tabs Logo, Accordion, or Accordion Logo).
  3. The Theme Designer opens.

  4. Customize the theme using the Theme Editor. See Using the NetHelp Theme Designer.
