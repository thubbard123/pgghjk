Using the NetHelp Theme Designer

The NetHelp Theme Designer makes it easy to create themes visually.

What You Can Change

What You Can Change

You can change the following (and more) quickly and easily. Please note that the options in the Theme Designer vary based on the theme chosen. The Responsive theme options are slightly different than the options for Accordion or Tabs. See NetHelp Theme Designer Settings (Responsive) and NetHelp Theme Designer Settings (Tabs and Accordion).

  • Header Add a logo, expand/reduce the height of the header.

  • Toolbar buttons Add/delete toolbar buttons, change the button order and alignment.

  • Icons Select a new icon for any button, upload your own, or display no icon. See Adding Custom Icons to a NetHelp Theme.

  • Table of contents, index, search Change the title that appears in each, the messages, and the visibility of the area.

  • Topic breadcrumbs Turn breadcrumbs on/off, add a label, include the current topic.

  • Language settings Choose a different language for the theme (14 are included), add your own language configuration file, or edit any of the strings in the selected configuration file. See Using the Settings List Editor for Localization and More.

  • Colors and fonts used All you need to do is choose a different stylesheet, download a new one from the jQuery ThemeRoller (Tabs or Accordion Themes) or  jQuery Mobile ThemeRoller (Responsive theme), or create a new one with the appropriate ThemeRoller.

 Notes

Customized themes are stored by default in \\My Doc-To-Help Projects\Doc-To-Help\Themes. You can change this location using the Doc-To-Help Options dialog (File>Tools>Options>Files).

You can preview how your changes will look and behave by clicking the Preview button in the Theme Designer. You can’t edit the theme in Preview mode.
