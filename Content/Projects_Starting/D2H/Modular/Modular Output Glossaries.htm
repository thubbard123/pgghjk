Modular Output Glossaries

Glossaries may be used in modular hub projects, but HTML Help targets require a few adjustments to avoid duplicate items in the hub table of contents and index.

In NetHelp projects, all child project glossaries are merged in the hub and duplicate glossary entries are automatically purged.

How to Work with Glossaries in HTML Help Files
  1. Verify all child projects, and the hub project, contain the same Glossary document file.
  2. Open a child project.
  3. Select the Home tab. At the bottom of the Target group, click . The Help Targets dialog opens.
  4. In the Advanced section, select the Skip glossary check box.
  5. Build the output.
  6. Repeat this for all child projects.
