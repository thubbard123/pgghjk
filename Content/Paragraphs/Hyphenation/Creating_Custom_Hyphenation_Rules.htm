Creating Custom Hyphenation Rules

In addition to specifying hyphenation settings, you can create custom hyphenation rules for words if you want them to break in specific locations. See Setting Hyphenation.

how to create custom hyphenation rules
  2. Place your cursor on the word for which you want to create a special rule.
  3. In the Hyphenation section select Hyphenation Window. The Hyphenation window pane opens.

  4. Click Add Custom Rule. The Add Custom Hyphenation Rule dialog opens, showing the word you selected and the current places where the word will be hyphenated at the end of lines. Each rule is separated by a semi-colon.

  5. If you want to add a new rule for a word, you can click in the Rule field and type the word next to the others, adding a hyphen in a different location inside the word to indicate where it will break. Make sure to separate new text from other rules with a semi-colon.
  6. A message indicates that the new rule has been added.
  7. This creates a file with an .mchyph extension and places it in the Advanced folder of the Project Organizer. You can make changes to existing rules by opening this file. Also, you can have one MCHYPH file for each language that you need. See Managing Custom Hyphenation Rules.
