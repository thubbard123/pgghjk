Setting Page and Column Breaks

 

If you are working on a document in the Print Layout mode, you may see icons in the margins next to text. These icons indicate where a paragraph has been moved to another page or column as a result of a page or column break, or due to widow or orphan control.

 notes

If you are using a multi-column page layout and set an "Inside" break to "Avoid," you must specify this in both the "Page Break" and "Column Break" fields.

It is also possible to use styles to set breaks on lists and tables. See Setting Page and Column Breaks for Lists and Setting Page Layout Breaks on Tables.

The method to insert page breaks independent of an HTML element is not supported for lists or tables. However, you can split a list or table into two separate lists or tables and then insert an independent page break in an empty paragraph between the two.
