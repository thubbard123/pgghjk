Creating Initial Caps Effects

If you want to create an initial caps effect in a paragraph, causing the first letter to look different than the rest of the paragraph, you can use 's first-letter pseudo class.

  5. In the local toolbar, click New Selector. The New Selector dialog opens.
  6. Expand the Advanced Options section, and from the Pseudo Element drop-down, select first-letter.

  7. The new class is added under a "(Pseudo Classes)" heading and is automatically selected.

 notes

If you want to modify the initial letter even further in order to produce a drop caps effect, you need to adjust the float and margin settings. See Creating Drop Caps Effects.

In order to use this feature, you must ensure that the first letter of the paragraph is not preceded by any other content (such as an image).

You can also use the first-child pseudo class, which lets you apply style settings to the first child of another element (e.g., the first <p> tag that occurs within a <div> tag).
