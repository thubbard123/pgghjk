Importing Dictionaries

You can see which dictionaries are installed on your computer by opening the Options dialog and selecting the Dictionaries tab. See .

You can see which dictionaries are installed on your computer by opening the Options dialog and selecting the Spelling tab. See .

Green dots indicate which dictionaries are supported by spell checkers and hyphenators. A hyphenator automatically inserts separators in an editor based on particular word patterns, as outlined in the dictionary file. Hyphenators are important only for certain languages.

You can also use the Options dialog to import more dictionaries that you have downloaded.

how to import a dictionary
  1. Download a dictionary from OpenOffice.org.

  2. Select File>Options.

    The Options dialog opens.

  3. Select the DictionariesSpelling tab.

  4. In the Installed Dictionaries section, click Import Dictionaries.

  5. In the dialog that opens, locate and double-click the dictionary that you downloaded. The dictionary is added to the Options dialog.
