Choosing the Location for Global Dictionaries

How to Choose the Location for Global Dictionaries
  1. Select File>Options. The Options dialog opens.
  2. Select the Spelling tab.
  3. In the Select Global Dictionary Location section, choose one of the following:
    • Use default location Select this if you want your global dictionaries to be stored in the App Data folder on your computer.
    • Use custom location Select this if you want to place your global dictionaries in any folder you choose on your computer or on a network drive. After selecting this option, click and choose the folder location. If you select a location on your network, any authors with access to that location can use the same dictionary.
