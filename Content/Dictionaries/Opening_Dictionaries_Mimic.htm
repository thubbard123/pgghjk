Opening Dictionaries

You can add words to a custom dictionary when you spell check movies and click the "Add to Dictionary" button for highlighted words. When a word is added to the custom dictionary, it is not flagged as a questionable spelling in the future;Mimic simply moves on to the next misspelled word.

A separate dictionary is created for each language used in your collection. If you want to make changes to a custom dictionary, you can open it using the steps below.

how to open a custom dictionary

  1. In the Language section, select Edit Dictionary.

    The Open Dictionary dialog displays.

  2. Select the dictionary that you want to open.
  3. The Dictionary Editor opens.
