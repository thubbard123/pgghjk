Editing Custom Dictionaries

how to Edit a Custom Dictionary
  1. Select Tools>Edit Dictionary.
  2. From the drop-down, select the kind of dictionary you want to edit: Global or Project. Any global or project dictionaries that you've created in various languages are listed below.
  3. From the grid, select the dictionary you want to edit.
  4. The Dictionary Editor opens.

  5. Edit the dictionary by doing any of the following:

    edit an existing word
    1. Click a word in the dictionary.
    2. Retype the word.
    3. Press Enter.
         add a new word
    1. In the local toolbar, click . A new row is added with temporary text.
    2. Click on the temporary text.
    3. Type the word that you want to add to the dictionary.
    4. Press Enter.
         add an internal comment
    1. Click in the Comment column next to the word.
    2. Type the internal comment.
    3. Press Enter.
         delete a word
    1. Click the row with the word that you want to delete.
