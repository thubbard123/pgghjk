Inserting Horizontal Rules

lets you insert rules (horizontal lines) into topics. For example, you might use a rule as a design element to separate your topic title from the content.

You can always change the look of the rule by modifying the hr style. If you want to have different rules for your topics, you can open add new style classes for each different horizontal line. For example, if you want a short rule, you can create a new style class and name it "short." The full name for you new style would be hr.short. You can then adjust the style property settings for hr.short to meet your needs.

how to insert a rule into a topic

  4. Select hr.
note

The hr tag may render differently from browser to browser. For a more consistent look across all browsers, it is a good idea to set both the color and the background-color properties of the hr tag.
