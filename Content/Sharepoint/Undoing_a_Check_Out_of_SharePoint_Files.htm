Undoing a Check-Out of SharePoint Files

If you have files checked out from SharePoint but do not want them checked out anymore, you can use the "Undo Check Out" option instead of checking in the files.

how to undo a check-out of sharepoint files
  1. In the Resources section select SharePoint Explorer.

    The SharePoint Explorer opens. See SharePoint Explorer.

  2. Locate the file(s). You can use the "multi-view" to do this.

  3. Select the file(s). If the window pane is split into two halves, you can select the appropriate folder on the left side and select the files on the right side.
  4. Right-click the file(s) and from the context menu select Undo Check Out.

  5. In the dialog that opens, click Undo Check Out.

    By default there is a check mark next to all files that were selected. If necessary, you can click any check box to remove the check mark, which means that file will still be checked in when you are finished.
