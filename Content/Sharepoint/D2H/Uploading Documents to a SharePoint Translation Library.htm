Uploading Documents to a SharePoint Translation Library

You must create a Translation Management Library in SharePoint before uploading your documents. See Creating a SharePoint Translation Library.

How to Upload Documents to a SharePoint Translation Library
  1. Open Doc-To-Help, then open the project you would like to upload to the SharePoint Translation Library.
  2. On the Home ribbon, select the Translate button. The Create Translation Project Wizard opens.
  3. Click Next.
  4. Choose the SharePoint Translation Library.
  5. Confirm the documents to share and click Next. Because the project was created with the default language of English, Doc-To-Help will automatically assign the documents to that language.
  6. Click Next.
  7. Confirm the settings and click Next again.
  8. Doc-To-Help shares the documents and creates a project for each language represented in the list of translators chosen for the SharePoint Translation Library. Click Finish.
  9. If the project has auxiliary files (such as graphics) Doc-To-Help will ask if you would like to synchronize those also. Click OK. After synchronization, click Close.

When you open the SharePoint Translation Library, there will now be a version of each document for each language.

In addition, a Doc-To-Help project is created for each language.
