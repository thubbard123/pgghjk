Editing a SharePoint Wiki

Once your NetHelp is in a SharePoint wiki, it can be edited by anyone with the proper permissions.

How to Edit a SharePoint Wiki
  1. Click the Page tab.
  2. Click the Check Out button.

  3. Click the Edit button. The page switches to edit mode and editing can begin.

    The Format Text and Insert ribbons display a complete set of Word-style formatting options, as well as spell check. You can edit anything, add images, and insert links to URLs.

    In Edit Mode, links to other topics within the project become wiki links ([[Documents/football|Football]]). This syntax is unique to wikis and makes it easy for anyone to edit existing links and create new ones.  

  4. Make your edits and save your work.
  5. On the Format Text tab, click the Check In button.
