Associating Conditions with Targets

how to associate condition tags with a target

  3. Double-click the target that you want to associate with one or more condition tags. The Target Editor opens to the right.
  4. Click the Conditional Text tab.

  5. There are two ways to use the Conditional Text tab—Basic and Advanced. By default when you open this tab for the first time, the Basic option is selected. You can click Advanced to switch to that mode if you are experienced at writing condition tag expressions by hand.
