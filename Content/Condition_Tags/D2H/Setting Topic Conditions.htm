Setting Topic Conditions

You can set specific platform, target, and/or attribute conditions for a topic.

How to Set Topic Conditions
  1. In the Topics window, select the topic.
    • Open the Topics ribbon, and from the Condition group, select the condition(s) using the drop-downs.

    • Open the Topics ribbon and click the Properties button. In the Topic Properties dialog, select the conditions in the Condition section.

No matter which method is used to set the conditions, they will always be displayed in both the Topic Properties dialog and the Condition ribbon group.

 Note

Attributes must be specified in the Help Targets dialog (see Creating and Deleting Targets) to be properly included/excluded from the final output. See Creating and Using Attributes.
