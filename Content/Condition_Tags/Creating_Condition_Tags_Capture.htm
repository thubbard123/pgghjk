Creating Condition Tags

See Linking to Flare Projects.

You can create condition tags using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to create condition tags for a single image only. Use the Profiles Editor if you want to create condition tags for a profile, which can be used when capturing future images. See About Profiles.

how to create condition tags using the file properties dialog
  1. Open the image.
  2. Double-click the image. The File Properties dialog opens.
  3. Click the Condition Tags tab.
  4. Click in an empty Name cell and type a name for the tag.
  5. Click the Background drop-down arrow and select Pick Color. The Color Picker dialog opens.
  6. Select a new color.
  8. (Optional) Click in the Comment cell and type an internal comment that helps you identify the tag. Press Enter when you are finished.
 how to create condition tags using the profiles editor
  1. Open the profile.
  2. In the Profiles Editor, select the Condition Tags tab.
  3. Click in an empty Name cell and type a name for the tag.
  4. Click the Background drop-down arrow and select Pick Color. The Color Picker dialog opens.
  5. Select a new color.
  7. (Optional) Click in the Comment cell and type an internal comment that helps you identify the tag. Press Enter when you are finished.
