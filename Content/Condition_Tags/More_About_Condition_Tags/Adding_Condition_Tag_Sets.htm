There are many tasks that you can perform when it comes to condition tags. Some of these tasks may be used quite frequently.

Adding Condition Tag Set Files

A condition tag set file has an .flcts extension and is stored in the Project Organizer under the Conditional Text folder.

how to add a condition tag set file
    • Ribbon In the Content section select New>Condition Tag Set.
    • Right-Click In the Project Organizer, right-click on the Conditional Text folder and from the context menu select Add Condition Tag Set.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Condition Tag Set is selected.
  5. In the File Name field, type a new name for the condition tag set.
  6. (Optional)
  7. (Optional) See Applying Conditions to Content.
  8. (Optional) See Associating Tags with Files.
  9. Click Add. The condition tag set is added to the Conditional Text folder in the Project Organizer. The Condition Tag Set Editor opens to the right, with the page for the new condition tag set shown. As with the default condition tag set, provides you with two condition tags (PrintOnly and ScreenOnly) to help get you started.
