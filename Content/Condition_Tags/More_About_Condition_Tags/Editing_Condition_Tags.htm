Editing Condition Tags

If necessary, you can edit the name, color, or comments for a condition tag that you have created.

how to edit a condition tag
  1. In the Project Organizer, expand the Conditional Text folder.
  2. Double-click the condition tag set holding the tag that you want to modify.
  3. rename a tag
    1. Click the tag and press F2 on your keyboard. The tag name is highlighted.
    2. Type a new tag name.
    3. Press Enter on your keyboard.
         change the color of a tag
    1. Click the Background drop-down arrow and select Pick Color. The Color Picker dialog opens.
    2. Select a new color.
         change or add comments for a tag
    1. Click in the Comment cell of the tag.
    2. Press F2 on your keyboard.
    3. Type comments for the tag.
    4. Press Enter on your keyboard.
