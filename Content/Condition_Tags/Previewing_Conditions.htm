Previewing Conditions

how to preview conditions
  1. Open an image a frame that you want to preview.
  2. Make sure you have already applied conditions to objects on the imageframe and associated the imagemovie, collection, or target with condition tags (including and excluding). See Applying Conditions and Associating Condition Tags with Images.See Applying Condition Tags and Associating Condition Tags.

  3. In the Object section, click the Preview Conditions check box .

    The objects that have condition tags set to be shown still appear, but the objects that have condition tags set to be excluded are hidden.

  4. In the Object section, click the Preview Conditions check box .
