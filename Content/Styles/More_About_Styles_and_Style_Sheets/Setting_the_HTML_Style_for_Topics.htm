Setting the HTML Style for Topics

You can set the HTML style for one topic at a time or for multiple topics at once.

how to set the HTML Style for a Single Topic
  1. Find and select the topic.
  2. In the local toolbar of the Content Explorer, click . The Properties dialog opens.
  3. Select the Topic Properties tab.
  4. In the Topic Style Class field, click the down arrow and select a class from the list.
 how to set the HTML Style for multiple topics

    • Keyboard Shortcut Press CTRL+SHIFT+J.

 note
