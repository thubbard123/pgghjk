Additional Regular Stylesheet Tasks

In addition to the primary tasks that you can perform related to regular stylesheets, you can also do the following:

  • Print Output—Setting Styles See Setting Styles for Print Output in a Regular Stylesheet.
  • Selectors—Renaming See Renaming Selectors.
  • Styles—Comments See Adding Comments to Styles.
  • Styles—Deleting See Deleting Styles.
  • Styles—Disabling You can See Disabling and Hiding Styles.
  • Styles—Generated Pages See Styles for Generated Pages.
  • Styles—Importing See Importing Styles.
  • Stylesheets—Importing See Importing Stylesheets.
  • Stylesheets—Linking See Linking Stylesheets.
  • Stylesheets—Opening When you create a new stylesheet for your project, the stylesheet opens automatically in the Stylesheet Editor or Table Style Editor. If you want to make changes to a stylesheet that is closed, you can open it. You can open stylesheets that you create or that are part of new standalone documents that you create. However, you cannot open the stylesheets that are associated with contribution templates or topics sent to you by Flare authors. See Opening Stylesheets.
  • Tag Groups—Creating See Creating Tag Groups.
  • Topics—Setting HTML Style See Setting the HTML Style for Topics.
  • Units of Measurement—About When changing the look of content, you may often need to select a unit of measurement (UOM). See About Units of Measurement.
