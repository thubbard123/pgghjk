Deleting Concept Links

There are two parts to a concept link—(1) the concept link itself and (2) the concepts inserted into topics that associate the topic with a particular concept link. The following steps show you how to delete the concept link.

how to delete a concept link from a topic

If you no longer need a concept link that you have previously inserted into a topic, you can easily delete it from a topic.

  2. Right-click the concept link that you want to delete.
  3. From the context menu, select Delete. The concept link is removed from the topic.
