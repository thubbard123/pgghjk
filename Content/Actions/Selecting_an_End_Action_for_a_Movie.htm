Selecting an End Action for a Movie

If you want a specific action to take place when a movie finishes playing, you can apply an end action to the movie (e.g., go to the next movie in the collection, go to the first frame of the movie, open a website). One common use for this is to create a looping action for the movie so that it plays over and over until it is stopped. See Looping Movies.

how to select an end action for a movie
  1. Open the movie.

  2. In the Properties section, select Properties.

    The Movie Properties dialog opens.

  3. Select the End Action tab and select the action that you want to apply to the end of the movie:
