Creating Breadcrumbs

To insert a proxy, open a master page (see Creating Master Pages), place your cursor where you want to insert the proxy, and select Insert>Proxy>Breadcrumbs Proxy.

hyperlinked breadcrumbs

If the books in the TOC are linked to topics in the project, the breadcrumbs will be hyperlinked in the output; if the books are not linked in the TOC, the breadcrumbs will not be hyperlinked.

 how to link a toc book to a topic
  1. Open the TOC.
  2. Select the TOC book.
  3. In the local toolbar of the TOC Editor, click . The Properties dialog opens.
  4. Click Select Topic. The Link to Topic dialog opens, displaying all the topics in your project.
  5. Select the topic to which you want to link the entry and click Open.
  6. The book is now linked to the topic.
 notes

After you insert a Breadcrumbs proxy into a master page, you can modify the look of the breadcrumbs using styles. See Using Styles to Determine the Look of Breadcrumbs.

If you are generating an target that uses a master page with a Breadcrumbs proxy, Flare will automatically remove the proxy when building the output. This is because the Eclipse development platform provides its own breadcrumb support. See the Platform Plug-in Developer Guide from Eclipse at http://help.eclipse.org/kepler/index.jsp.
