Creating a Topic Menu

how to insert Topic Menu
  1. Open a master page that you will use when generating the target.
  2. Place your cursor where you want the menu to be shown in the output. If you are working in a master page, you may need to press the down arrow on your keyboard and then press Enter at the correct location.
  3. Select Insert>Proxy>Insert Menu Proxy. The Menu Proxy dialog opens.

  4. You can select the following options:

  5. The proxy is added.
 Editing a Menu Proxy

After inserting a Menu proxy, you can edit it. To do this, open the topic or master page where the proxy is inserted, right-click on it, and select Edit Menu Proxy.

 What's Next?

If you inserted the proxy into a master page, make sure the master page is associated with your target. See Associating Master Pages with Targets.
