About Palettes

After you create a movie, you can add various types of objects (such as shapes, lines, buttons, and cursors) to the individual frames. A palette is an element that lets you store objects for later use.

example

Example

Let's say you create a button and modify it (adding text, color, shadow, an action), with the intention of using it in most frames. It would be time-consuming to redesign that button each time you want to use it in a frame. So instead, you can save the button to a palette. And the next time you want to add the button to a frame, you just double-click it or drag it from the palette.

Steps for Using a Palette

Here are the basic steps for using a palette in .

  1. Open provides you with an initial palette that you can use (called "MyPalette"). Whenever you want to add objects to a palette or use objects from it, the first step is to open the palette. See Opening Palettes.
  2. (Optional) Create In addition to the initial palette, you can create more palettes if you have many objects that you want to save and organize in different groups. See Creating Palettes.
  3. (Optional) Link to External Palettes In addition to the initial palette and those that you create locally, you can link to external palettes. You might want to do this if you share a palette with a team or if you have a palette that you need to use for a specific project. See Linking to External Palettes.
  4. Add Objects to Palettes After you create or open a palette, you can add objects to it. For more information see Adding Objects to Palettes.
  5. Organizing Palettes You can rearrange the order of objects in a palette by clicking on them and dragging them up or down in the Palette window pane. See Using Objects from Palettes.
  6. Use Objects from Palettes When a frame requires a particular object that you've saved, you can add it to the frame. See Using Objects from Palettes.
 note

Palettes are especially convenient because they are stored in your "Documents\My Palettes" subfolder. This means that, once created, the palette and its objects become available to all of your Mimic collections and movies. Not only that, but they are also available to any images that you edit in Capture.
