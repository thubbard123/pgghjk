Using Objects from Palettes

After objects are added to a palette, you can use them in images whenever necessary.

how to use an object from a palette
  1. Open an image.
  2. Open the palette containing the object that you want to use.
  3. (Optional) You can rearrange the order of objects in each palette to make them easier to use. Click and drag the objects in the palette up or down to change their order.
    • Click the object in the palette, drag it to the image, and drop it.
    • Double-click the object in the palette.
  5. You can then move, resize, or double-click the object to make property changes to it.
