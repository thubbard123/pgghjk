Opening Page Layouts

There are multiple ways to open page layouts that you have added to your project. You can open layouts from the Content Explorer or from an open topic associated with a page layout. See About Page Layouts.

You can open page layouts that you have created or that exist in a folder on your computer. However, you cannot open the page layouts that are associated with contribution templates or topics sent to you by Flare authors.

 how to open a page layout from an open topic
  1. Open a topic.

  2. In the XML Editor, make sure you are working in Print Layout mode. In the local toolbar, if , , or is displaying, click the down arrow next to the current layout and select Print Layout from the drop-down. This changes the button display to , which means the topic is shown in Print Layout mode. See About Layout Modes.

  3. Double-click on the margin of the page (i.e., not on a frame). Alternatively, you can right-click on the margin of the page (i.e., not on a frame). Then in the menu, select Edit Page Layout Document.

    The page layout opens in the Page Layout Editor.

 how to open a page layout that is part of a new document

The following steps pertain to new documents that you create without using a contribution template from a Flare author.

  1. Open the document that you created.

  2. Right-click on the margin of the page (i.e., not on a frame). Then in the menu, select Edit Page Layout Document.

    The page layout opens in the Page Layout Editor.
