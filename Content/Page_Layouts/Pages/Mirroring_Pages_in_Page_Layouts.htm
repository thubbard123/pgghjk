Mirroring Pages in Page Layouts

There are various ways to add new pages to a page layout.  See About Page Layouts.

how to mirror a page in a page layout
  1. Open the page layout. The Page Layout Editor opens.
  2. Open the page that you want to mirror. (You can click a small rectangle to the right of the layout to open its different pages.)
  3. In the local toolbar, click the down arrow next to the Page Properties button .
  4. From the drop-down select Mirror Page.A new page is added to the page layout. A small rectangle is added to the right of the page layout, which can be used to open the new page.
 notes

You can also insert, duplicate, or copy a page. See Inserting Pages in Page Layouts, Duplicating Pages in Page Layouts, and Copying Pages in Page Layouts.

If you have created mirror margins on your pages and are compiling Microsoft Word output, you need to enable the mirror margins in the Target Editor. See Enabling Mirror Margins in Word Output.
