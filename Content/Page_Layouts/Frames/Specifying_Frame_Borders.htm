Specifying Frame Borders

For more information see About Page Layouts and Editing Pages and Frames.

how to add borders to a frame
  1. Select the Borders tab.
  2. Set the options in the Borders section.

  3. If you want the frame to have rounded corners, set the options in the Border Radius section. This is supported only in PDF, XPS, and XHTML outputs.

  4. In the Frame Properties dialog, click OK.
 note

Microsoft Word output does not support frame borders.
