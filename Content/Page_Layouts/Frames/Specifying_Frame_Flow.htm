Specifying Frame Flow

 

If you add more than one frame of the same type to a page, they are numbered in the order you add them (1, 2, 3, and so on), and arrows connect them to indicate the flow, starting at the first frame. When you generate output, if the first frame becomes completely filled with content, the content then continues in the second frame, then the third, etc. If you remove one of the frames, the remaining ones are renumbered (e.g., if you have four body frames and remove the second one, third frame is renumbered 2 and the fourth one is renumbered 3).

For more information see About Page Layouts and Editing Pages and Frames.

how to specify the flow for a frame
  1. Select the Flow tab.
  2. Select one of the options to determine the flow of the frame.
    • Body
    • Header
    • Footer

    • Decoration

  3. In the Frame Properties dialog, click OK.
