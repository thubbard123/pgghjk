Creating Columns

how to create columns
  1. Open a page in a page layout.
  2. Double-click the frame for which you want to add or configure columns.
  3. Select the Columns tab.
  4. Set any of the options on the tab for the columns or the divider.columns
    • Count Enter the number of columns to be used in the frame to display text.
    • Gap Enter the amount of space that you want between each column.
    column divider
    • Style Select the type of vertical line that you would like to display between columns (either a solid or double line). If you do not select a style, the gap between the columns will be empty.
    • Width Specify the width of the single solid line or the double lines that serve as the column divider.
    • Margin Top Specify the amount of space between the top of the column divider and the top edge of the frame.
    • Margin Bottom Specify the amount of space between the bottom of the column divider and the bottom edge of the frame.
    • Color Use this field to apply a color to the column divider. Click
note

When you insert an Index proxy, you can select the number of columns for the index in your output. However, this applies to Microsoft Word and Adobe FrameMaker output only (see Creating a Topic for a Print Index). For PDF, XHTML, and XPS output, the number of columns in an index is determined by the settings in the page layout, rather than by the setting in the Index proxy.
