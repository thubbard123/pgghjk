Specifying a Frame Background

 For more information see About Page Layouts and Editing Pages and Frames.

how to add a background to a frame
  1. Select the Background tab.
  2. Use the tab to specify the settings that you want for the background.set a color for the background
    1. In the Color section, select a color direction for the frame background.
    2. In the Fill Start field, click the down arrow and select a color to use at the beginning of the background color gradient.
    3. In the Fill End field, click the down arrow and select a color to use at the end of the background color gradient.
    add an image to the background
    1. In the Image section, click the Browse button. The Insert Image dialog opens.
    4. (Optional) In the Repeat field, select whether you want the image to repeat in the background.
    5. (Optional) In the X field, select or enter the horizontal position of the background image in the frame.
    6. (Optional) In the Y field, select or enter the vertical position of the background image in the frame.
  3. In the Frame Properties dialog, click OK.
