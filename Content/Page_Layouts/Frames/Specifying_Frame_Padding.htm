Specifying Frame Padding

 For more information see About Page Layouts and Editing Pages and Frames.

how to specify the padding for a frame
  1. Select the Frame tab.
  2. Use the Padding section to enter settings.
    • Left Enter the amount of space that you want to be added between the left edge of the frame and any text or content within it.
    • Right Enter the amount of space that you want to be added between the right edge of the frame and any text or content within it.
    • Top Enter the amount of space that you want to be added between the top edge of the frame and any text or content within it.
    • Bottom Enter the amount of space that you want to be added between the bottom edge of the frame and any text or content within it.
