Changing the Vanity URL

See Setting Builds to Live or Keep. For more information, see the Central online Help.

How to Change the Vanity URL
  1. In the main navigation framework, click your user name, then click Settings. The Settings dialog opens.
  2. Click the Server tab. Your current vanity URL is displayed in the Vanity URL field.
  3. Below the Vanity URL field, click Edit. The Change Vanity URL dialog opens.
  4. In the New Prefix field, enter the prefix you want to use for your vanity URL (only alpha characters and numbers are allowed). A preview of your vanity URL ([prefix].mcoutput.com) appears below the field.
  5. Click Save.
