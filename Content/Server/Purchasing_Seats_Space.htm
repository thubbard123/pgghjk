Purchasing User Seats and Storage Space

Users who have the appropriate permissions can manage certain server settings.

Note

How to Purchase User Seats or Storage Space

    • Home Page Click . The Home page opens. In the Storage and Usage widget, below the Users graph, click Add Users, or below the Storage graph, click Add Storage.

      OR

    • Settings Dialog In the main navigation framework, click your user name, then click Settings. The Settings dialog opens. Click the Server tab. Below the Users graph, click Add Users, or below the Storage graph, click Add Storage.

    The Add to License dialog opens.

  2. In the dialog, use the drop-downs to select the products you want to add to your license, and to select the type of renewal you want.

    As you add or remove user seats and/or storage space, Central calculates the total cost and displays it next to the Total Due Today field.

    The total shown does not include tax.

  4. Click Continue. The Confirm Purchase dialog opens. A message shows you the number of user seats and/or the amount of storage space you are purchasing, as well as the total amount of the purchase (exclusive of tax).
  5. Click Complete Purchase. Your card will be charged.
