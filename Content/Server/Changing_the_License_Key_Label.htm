Changing the License Key Label

Users who have the appropriate permissions can manage certain server settings.

How to Change the License Key Label
  1. In the main navigation framework, click your user name, then click Settings. The Settings dialog opens.
  2. Click the Server tab. Your current license key label is displayed in the Key Label field.
  3. Below the Key Label field, click Edit. The Change Key Label dialog opens.
  4. In the New Key Label field, enter the label you want to use for your license key.
  5. Click Save. Your license key label is updated throughout MadCap Central.
 Notes

Changing the license key changes it for all users.
