What is a Primary Target?

Use the following steps if you want to quickly build and publish the primary target for your project.

So what is the big deal with the primary target? It is simply about convenience. Although you can use more than one target for your project—you will most likely use just one target most of the time. So lets you designate one target as your primary target and gives you shortcuts to perform certain tasks with it, including building the output from the primary target.

 how to build and Publish the primary target
    • Ribbon In the Build section select Build Primary (to build only) or Publish Primary (to publish a target that has already been generated).
    • Local Toolbar Open the target. In the local toolbar of the Target Editor click (to build only) or (to publish a target that has already been generated).
    • Keyboard Shortcut Press F6 (to build only) or CTRL+F6 (to publish a target that has already been generated).
