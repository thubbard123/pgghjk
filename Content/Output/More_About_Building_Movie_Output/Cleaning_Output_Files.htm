Cleaning Output Files

When you build a movie or collection, the output files are placed in an Output subfolder. You can clean these output files anytime you want. This means that the Output subfolder (and all output files contained within it) are deleted. The next time you build the movie or collection, the Output subfolder will be re-created for you automatically.

how to clean output files for a collection
  1. Open the collection.

  2. In the Collection Output section, select Clean Output.

    A message displays, asking you to confirm the deletion of the Output folder and its files.

 how to clean output files for a standalone movie
  1. Open the movie.

  2. In the Movie Output section, select Clean Movie Output.

    A message displays, asking you to confirm the deletion of the Output folder and its files.
