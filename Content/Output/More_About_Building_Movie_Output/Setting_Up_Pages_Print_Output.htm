Setting Up Pages for Print-based Output

Before you build output using the PDF format, you can adjust settings such as the page size, orientation, and margins.

how to set up pages for print-based output
  1. In the Collection Editor or Movie Properties dialog, select PDF as the movie format. See Selecting the Movie Format.
  2. Click the Page Setup button to the right. The Page Setup dialog opens.
  3. Complete the fields in the dialog as necessary.
