Saving Publishing Logs

After you publish movies, you can view a log, which provides a summary of the actions performed for the publishing process. If you want to keep a record of the log, you can save it as an XML file.

how to save a publishing log
  1. Follow the steps for publishing a collection or movie. See Publishing Output.
  2. After the collection or movie files are published, a dialog opens, asking if you want to view the log. Click Yes. The Publishing Log dialog opens.
  3. Click Save Log File.
  4. In the Save As dialog, navigate to a location and provide a name for the log file.
  5. Click Save.
