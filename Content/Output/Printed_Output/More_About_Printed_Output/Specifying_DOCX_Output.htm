Specifying DOCX Output Via Microsoft Word

DOCX is Microsoft Word's platform-independent, open XML format. In , you can send output directly to that format when generating a Word target.

how to specify docx output for microsoft word
  1. Open a Word target. If you have not yet created a Word target, see Adding Targets.
  2. In the Target Editor, select the General tab.

  3. In the Output File field, enter the name that you want to use for the main output file and add the .docx extension at the end of it.

    For example, if you want your main output file to be called "MyNewManual," you would enter the following in this field: MyNewManual.docx

 note

In order to send output to DOCX via Microsoft Word, you need to have Microsoft Word 2007 installed. Previous versions of Word do not support this functionality.

 What's Next?
