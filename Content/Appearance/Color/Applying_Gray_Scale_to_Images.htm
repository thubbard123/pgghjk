Applying Gray Scale to Images

how to apply gray scale to an image using the file properties dialog
  1. Capture or open the image.
  2. In the Capture Editor, double-click the image. The File Properties dialog opens.

  3. Select the Format tab.

    more about the Format Tab
  4. From the Medium drop-down, select the medium whose gray scale settings want to want to edit (i.e., print, web, custom). If necessary, select Enable Format to enable the medium. See Selecting a Medium for an Image.
  5. Select Gray Scale so that a check mark displays in the box.
 how to apply gray scale to an image using the profiles editor
  1. Open the profile.

  2. In the Profiles Editor, select the Format tab.

    more about the Format Tab
  3. From the Medium drop-down, select the medium whose gray scale settings want to want to edit (i.e., print, web, custom). If necessary, select Enable Format to enable the medium. See Selecting a Medium for an Image.
  4. Select Gray Scale so that a check mark displays in the box.
 note

You can also add a gray scale effect around an object in an image. Doing this will display the image in gray, except for the area inside the object, which remains displayed in color. For more information see Adding Gray Scale Effects.
