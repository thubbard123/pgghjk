Editing Torn Edge Effects

After you add a torn edge effect to an image, you can edit the effect in the following ways.

  • Modify the Wave Length and Height
  • Modify which Edges have Torn Edges
  • Remove the Torn Edge Effect

See Adding Torn Edge Effects.

how to modify the wave length and height
  1. Open the image.
  2. Double-click the image.
  3. In the File Properties dialog, select the Edge Effects tab.
  4. Change the numbers in the Wave Length and/or Wave Height fields. The higher the number, the wider or deeper the wave in the torn edge.
 how to modify which edges have torn edges
  1. Open the image.
  2. Double-click the image.
  3. In the File Properties dialog, select the Edge Effects tab.
  4. In the Edges section, click in the check boxes to add or remove check marks. This determines the specific edges (top, bottom, left, right) where you'd like the effect to be applied in the image. If a check mark is displayed, the effect will be applied to that edge.
 how to remove the torn edge effect
  1. Open the image.
  2. Double-click the shape.
  3. In the File Properties dialog, select the Edge Effects tab.
  4. Click the down arrow next to the Edge Effect field and select (default).
