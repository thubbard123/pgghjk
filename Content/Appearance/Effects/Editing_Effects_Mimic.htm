Editing Effects

You can edit the start time of the effect, the duration of the effect, and the way effect is applied to the object. You can also modify some blur and opacity effects on all objects to which those effects are applied in a single frame or a whole movie.

How to Edit an Object Effect
  1. Open a movie that contains objects to which effects have been applied.

  2. In the Timeline window pane (View>Timeline), do one of the following:

    • For all standard object types, click Frame View. Any action objects converted to objects will also be in this view.
    • For action object types, click Action View. Any objects converted to action objects will also be in this view.

  3. In the Timeline window pane, right-click an effect bar and select Properties.

    The Effect Properties window pane opens.

  4. In the Effect Properties window pane, do the following, if necessary:

    Change the Position and Size of the Effect Area
    • Expand the Object Layout section and make the desired adjustments:
      • Position Select a different position on the frame for the effect area. The first field sets the horizontal position, and the second field sets the vertical position. Manually change the position of the effect area by clicking on the red effect area frame and dragging it to a new place in the Frame Editor.
      • Size Select a different size for the effect area. The first field sets the width, and the second field sets the height. Manually change the size of the effect area by clicking on the edges of the red effect area frame and dragging them to the desired height or width.
         Change the Direction of the Effect Area
    • Expand the Mask section and select from the following:
      • Inside Everything inside of the red effect area frame will have the effect applied to it.
      • Outside Everything outside of the red effect area frame will have the effect applied to it.

    Masks are not available for clip effects. You can only clip inside the red effect area frame.

         Change the Mask Properties

    The mask properties are only applicable to blur and opacity effects.

    • Opacity Expand the Opacity Mask section and drag the slider to the desired percentage. The higher the percentage, the more opaque the effect area will become.
    • Blur Expand the Blur Mask section and enter a number in the Blur Radius field. The higher the number, the more blurred the effect area will become.
how to modify An Effect applied to All Objects in a Single Frame

You can modify the amount of blur or opacity on all objects within a frame at once. The settings here only apply to objects to which the blur or opacity effect has been applied.

  1. Open the frame.
  2. In the Frame Editor, double-click the frame (not the shape).
  3. In the , expand the Frame Image Effects section.
  4. Change the number in the Blur Factor field. A higher number applies a larger amount of blur.
  5. Change the number in the Opacity field. The higher the number, the more opaque the object effect will be.
  6. (Optional) Click Make same as movie if you want to use the same setting that is entered for the entire movie (in the Movie Properties dialog).
 how to modify an Effect applied to All Objects in a movie

You can modify the amount of blur or opacity on an entire movie. This will make it so all existing objects that have a blur or opacity effect are set to a consistent effect level. Additionally, the blur or opacity will automatically be set whenever adding those effects to any new objects.

  1. Open the movie.

  2. In the Properties section, select Properties.

    The Movie Properties dialog opens.

  3. Select the Image Effects tab.
  4. Change the number in the Blur Factor field. A higher number applies a larger amount of blur.
  5. Change the number in the Opacity field. The higher the number, the more opaque the object effect will be.
