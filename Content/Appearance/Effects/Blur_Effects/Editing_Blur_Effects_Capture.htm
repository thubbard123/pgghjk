Editing Blur Effects

After you add a blur effect to an image, you can edit the effect in the following ways.

  • Modify the Amount of Blur Applied You can do this using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to edit blur effects for a single image only. Use the Profiles Editor if you want to edit blur effects for a profile, which can be used when capturing future images. See Creating Profiles.
  • Remove the Blur Effect
How to Modify the Amount of Blur Applied Using the File Properties Dialog
  1. Open the image.
  2. Double-click the image (not the shape).
  3. In the File Properties dialog, select the Image Effects tab.
  4. Change the number in the Blur Factor field. The higher the number, the more the image is blurred.
 How to Modify the Amount of Blur Applied Using the Profiles Editor
  1. Open the profile.
  2. In the Profiles Editor, select the Image Effects tab.
  3. Change the number in the Blur Factor field. The higher the number, the more the image is blurred.
 How to Remove the Blur Effect
  1. Open the image.
  2. Double-click the shape (not the image).
  3. In the properties dialog for the object, select the Image Effects tab.
  4. Click in the Enable Blur Effect field to remove the check mark.
 note

If you want to edit the blur effect within a shape, instead of around it, see Editing Blur-Inside Effects.
