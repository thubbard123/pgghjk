Editing Shading Effects

After you add a shading effect to an image, you can edit the effect in the following ways.

  • Modify the Amount of Shading Applied You can do this using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to edit shading effects for a single image only. Use the Profiles Editor if you want to edit shading effects for a profile, which can be used when capturing future images. See Creating Profiles.
  • Remove the Shading Effect
how to modify the amount of shading applied using the file properties dialog
  1. Open the image.
  2. Double-click the image (not the shape).
  3. In the File Properties dialog, select the Image Effects tab.
  4. Change the number in the Shade Factor field. The higher the number, the darker the shading.

The shade color is based on the background color from the Appearance tab.

 how to modify the amount of shading applied using the profiles editor
  1. Open the profile.
  2. In the Profiles Editor, select the Image Effects tab.
  3. Change the number in the Shade Factor field. The higher the number, the darker the shading.

The shade color is based on the background color from the Appearance tab.

 how to remove the shading effect
  1. Open the image.
  2. Double-click the shape (not the image).
  3. In the properties dialog for the object, select the Image Effects tab.
  4. Click in the Enable Shade Effect field to remove the check mark.
