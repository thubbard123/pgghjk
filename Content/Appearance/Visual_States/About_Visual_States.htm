About Visual States

You can add different visual states to an object in order to change the object's appearance if certain actions are performed.

Tasks Associated with Visual States
  • Add a Visual State to an Object You can add a visual state to an object when you want the object to have a different look after a specific action is performed. See Adding Visual States to an Object.
  • Edit a Visual State Once you have added a visual state to an object, you can easily edit it. See Editing Visual States.
  • Remove a Visual State You can remove a visual state from an object, or you can completely remove the visual state from the movie. See Removing Visual States.
