Rotating Objects

You can rotate objects after adding them to images frames. You can rotate objects manually, or you can rotate them using preset increments—1 degree, 45 degrees, or 90 degrees at a time.

example how to rotate an object
  1. Open an image to which you have added objects.
  2. Click the object that you want to rotate.

    • Ribbon In the Rotation section, select one of the following.

      Rotates the selected object 45 degrees clockwise. Click the down arrow to select 90, 45, or 1 degree clockwise and counter-clockwise rotations. See step 5 below.

      Returns the object to its original setting before it was rotated.

    • Tool Strip Select Object>Rotate.
    • Format Toolbar Click . (You can open the Format toolbar by selecting View>Toolbars>Format or by clicking in the Frame Editor's local toolbar.)
    • Keyboard Shortcuts Click on an object and press CTRL+Left Arrow or Right Arrow on your keyboard to rotate an object to the left or right one degree at a time. Click on an object and press CTRL+SHIFT and drag the object to rotate it 15 degrees at a time.
    • Right-Click Right-click on the object and select Rotate.

  4. Depending on the option you choose, you can select one of the following rotation options.
