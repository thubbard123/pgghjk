Additional Topic Tasks

In addition to the primary tasks you can perform related to topics, you can also do the following:

  • Broken Bookmarks See Viewing Broken Bookmarks.
  • Broken Links You can See Viewing Broken Links.
  • Closing You can close any single topic page that you want. You can also close all open topics (and other open editors, such as targets and TOCs) at the same time. See Closing Topics.
  • Deleting You can delete a topic any time you want. See Deleting Topics.
  • HTML Style See Setting the HTML Style for Topics.
  • Meta Descriptions See Adding Meta Descriptions for Topics.
  • Not in a TOC You can See Viewing Topics Not in a TOC.
  • Not Indexed You can See Viewing Topics Not Indexed.
  • Page Layout in XML Editor See Selecting a Page Layout to View Content in the XML Editor.
  • Paste—Default Behavior See Setting the Default Paste Behavior.
  • Paste—Text from Other Programs See Pasting Text from Other Programs.
  • Previewing See Previewing Topics.
  • Print Glossary See Creating a Glossary Manually for Print.
  • Print Index See Creating an Index Manually for Print.
  • Print TOC See Creating a TOC Manually for Print.
  • Printing See Printing Topics.
  • Renaming You can rename any of the topics in the Content Explorer. See Renaming Topics.
  • Reviews—Accepting Topics See Accepting Reviewed Files.
  • Reviews—Merging Topics See Merging Reviewed Files.
  • Reviews—Sending Topics You can send files for review. See Sending Files for Review.
  • Reviews—Viewing Topics in Inbox See Viewing Reviewed Files in the Inbox.
  • Reviews—Viewing Topics Sent See Viewing Files Sent for Review.
  • Search—Including or Excluding Topics See Including or Excluding Topics in Search.
  • Startup Topic See Specifying the Startup Topic in a Target.
  • Titles—Changing You can change the name of a topic title. See Changing Topic Titles.
  • Zoom/Scale See Using the Zoom/Scale Feature.
