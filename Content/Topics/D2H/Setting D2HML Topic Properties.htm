Setting D2HML Topic Properties

Specific topic properties may be set using D2HML (Doc-To-Help Markup Language). See About Doc-To-Help Markup Language.

How to Set D2HML Topic Properties
  1. In the Word source document, select the Doc-To-Help ribbon.
  2. In the D2HML Styles group, click Topic Properties. The D2HML Topic Properties dialog opens.

  3. Make changes in the different sections of the dialog.

    General
    • See About Context-sensitive Help. See the online Help or the Context-sensitive Help Guide.
    Appearance
    • See Creating a Table of Contents.

         Condition

    • Disable Comments If checked, DISQUS commenting will be disabled for this topic. See Adding DISQUS Commenting to NetHelp Targets. See the online Help or the Targets Guide.
Notes
