Exporting the Topic List

Exporting your topic list is useful if you’d like to sort or filter your topics, or if you’d like to provide a list of topics and context IDs to your software development team for the implementation of context-sensitive Help. See About Context-sensitive Help. See the online Help or the Context-sensitive Help Guide.

How to Export the Topic List
  1. Open the Topics window.
  2. Choose the Topics ribbon.
  3. Click the Print and Export button.
  4. From the Export to… list, choose Excel or Text.
  5. The Save As dialog opens.
  6. Choose the file location and name.
 Note

Only the columns displayed will be exported. For information on how to display/hide topics, see Topics Window.
