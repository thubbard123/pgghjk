Touring the Workspace

's workspace is flexible, uses a modern Multiple Document Interface (MDI), and gives you several options to work the way that you want.

Main Sections of the Interface

main sections of the interface

's user interface consists of the following major sections:

  • Left The left side of the interface is the default location for the File Map window pane, which can be used to view and align files in the project you are translating.
  • Middle The large middle section of the interface is the default location for the main Alignment Tool, which is used to align individual text segments in a file. The middle area also displays the Start Page, which is used for quickly performing high-level tasks and accessing information.
  • Right The right side of the interface (like the left side) is the default location for various window panes (e.g., Dynamic Help window pane).

Start Page

start page

The Start Page in this application is a handy window pane that displays when you first launch and remains open unless you close it. You can use the Start Page to open and create projects, access Help resources, and more.

File Map Window Pane

file map window pane

The File Map window pane lets you view all of the files in your project and align them as necessary. You can limit the files that are displayed by using the file type filter. For example, if you want to see only topic files, you could select the "htm" option from the Filter drop-down list. See Aligning Files.

Alignment Tool

alignment tool

The Alignment Tool displays after you open a file from the File Map window pane. The Alignment tool lets you view and align text segments within a file as necessary. See Aligning Segments.

Ribbons

ribbons

The ribbon layout consists of a File drop-down with common tasks such as creating and opening alignment projects. This is followed by the Home ribbon, which is the default, and several others related to various features in . See Ribbons.

The ribbon view also lets you take advantage of the Quick Access toolbar, which lets you add your favorite ribbon options to it. See Quick Access Toolbar.
