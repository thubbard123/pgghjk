Window Panes

The Movie Viewer can have multiple window panes in the interface that are used primarily for viewing the online Help. If more than one window pane is open, the panes are organized in an accordion structure. This means that they are stacked on top of each other, with the active window pane on the very top. Accordion bars for each window pane are located at the bottom, and you can click any bar to bring that particular window pane to the very top.

The Movie Viewer contains the following window panes.

  • Dynamic Help window Used to automatically display the Help topic related to the active element in the interface.
  • Help Favorites window Used to view your favorite Help topics and searches in your online Help.
  • Help Glossary window Used to view the glossary terms and definitions in your online Help.
  • Help Index window Used to view the index in your online Help and open topics.
  • Help Search window Used to search for information in your online Help.
  • Help TOC window Used to view the TOC in your online Help and open topics.
