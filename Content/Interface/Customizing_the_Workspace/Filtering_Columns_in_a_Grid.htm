Filtering Columns in a Grid

How to Filter Columns in a Grid
  1. In Central, open a page with a grid—Projects, Tasks (Calendar, Archive, or Backlog), Teams, Users.

  2. In the grid, click next to the column you want to filter. A popup opens.

    If you do not see next to a column, you cannot filter that column.

  3. In the Show items with value that field, select a filter type from the first drop-down, then enter a value in the field beneath the drop-down. This is your first filter condition.

    You can select from the following filter types:

    • Is equal to
    • Is not equal to
    • Starts with
    • Contains
    • Does not contain
    • Ends with

  4. (Optional) If you want to filter by multiple conditions, in the And/Or field, select one of the following:

    • And Include both filter conditions.
    • Or Include either filter condition.
  5. (Optional) If you want to filter by multiple conditions, select a filter type from the drop-down, then enter a value in the field beneath the drop-down. This is your second filter condition. You can select from the same filter types as the previous field.
  6. (Optional) To reset the filters, click Clear.
  7. Click Filter. The columns on the page will be updated.
