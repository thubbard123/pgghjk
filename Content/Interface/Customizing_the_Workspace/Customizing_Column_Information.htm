Customizing Column Information

You can use a context menu to simply hide or show certain columns in a window pane, or you can open the Columns dialog to customize columns even more.

how to hide or show columns from the context menu
  1. In the window pane, right-click on any of the column headings.
  2. From the context menu, click next to any column name to remove or add the check mark next to it. If a check mark appears next to a column name, it will be shown in the window pane. If a check mark does not appear next to it, the column will not be shown.

 how to customize from the columns dialog
  1. Open the Columns dialog.

    • In the Statistics window pane or Pending Changes window pane, click the Columns button in the local toolbar.

      OR

    • In the TOC Editor, make sure you are in Grid View. Then in the local toolbar, click .
    1. Open the window pane where you want to customize column information (e.g., Review Packages window pane).
    2. In the local toolbar of the window, click the Columns button .
  2. Use the options in the dialog to do any of the following.hide a column
    • In the Show column, click the check box to remove the check mark next to the column that you want to hide in the interface.
    include information in a chart
    • In the Chart column, click the check box to include the data in a chart.
    resize the width of a column
    • Double-click in the appropriate Width cell and type the new width in pixels.
    set the font color for the text in a column
    1. In the appropriate Color cell, click the down arrow and select Pick Color. The Color Picker dialog opens.
    2. Select the color.
    set the background color for a column
    1. In the appropriate Background cell, click the down arrow and select Pick Color. The Color Picker dialog opens.
    2. Select the color.
    change the order of the columns
    1. Select any of the items in the dialog. The row for that item is highlighted.

    2. Click the Move Up or Move Down button to adjust the position of the row.

      When you are finished, the item at the top of the dialog will display on the far left in the final interface. The item at the bottom will display at the far right.

 Note

You can also customize column information in the Builds window pane, but the features are slightly different than those described here. See Customizing Column Information in the Builds Window Pane.
