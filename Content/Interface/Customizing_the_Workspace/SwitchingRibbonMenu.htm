Switching Between Ribbons and the Tool Strip

By default ribbons are used when you install and launch for the first time. However, you can use the Options dialog to switch back to the classic tool strip, where toolbars and menus are shown.

how to switch between ribbons and the tool strip
  1. Select File>Options.

    The Options dialog opens.

  2. Select the Interface tab.
  3. In the Menu Style section, select either Ribbon or Tool Strip.
