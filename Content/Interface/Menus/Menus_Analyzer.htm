Project Properties

Opens the Project Properties dialog, which lets you do the following.

  • Select a master table of contents (TOC) for the entire project. The TOC will be used for all targets in the project. However, you can also set a master TOC at the target level. If you do that, the TOC specified at on the target takes precedence over the master TOC set at the project level.
  • Select a master page layout for the entire project.
  • Select a master stylesheet for the entire project.
  • Select a language to be used for the project.
  • Bind a project to source control.
