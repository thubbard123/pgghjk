Proxy

Opens a submenu, which lets you select a proxy to insert into a topic. A proxy is a placeholder for a certain type of content.

  • Insert Body Proxy Opens the Body Proxy dialog, which lets you insert a Body proxy into a master page. You can then enter text above the proxy (for a header) or below the proxy (for a footer). In order for this option to be enabled, you must first open a master page. See Using the Body Proxy.
  • Insert Breadcrumbs Proxy Opens the Breadcrumbs Proxy dialog, which lets you insert a Breadcrumbs proxy into a master page or an individual topic. If a proxy is included in a master page or topic, the output will display a "trail of breadcrumbs" composed of the table of contents (TOC) entries above the current topic in the TOC hierarchy. See Using the Breadcrumbs Proxy.
  • Insert Topic Toolbar Proxy Opens the Topic Toolbar Proxy dialog, which lets you insert a topic toolbar proxy into a master page or an individual topic. You can use the topic toolbar proxy to generate a web toolbar anywhere in any topics. This is an alternative (or an additional option) to the regular toolbar that can be included in all online outputs (via the Help Viewer or through the use of skins). See Inserting Topic Toolbar Proxies.
  • Insert Mini-Toc Proxy Opens the MiniTOC Proxy dialog, which lets you insert a "mini-TOC" proxy into a master page or topic. If this proxy is included in a master page or topic, the output will display links to TOC entries below the current topic in the TOC hierarchy (a "mini-TOC"). It's a good way to create a small TOC at the beginning of chapters in print-based output. See Creating a Mini-TOC.
  • Insert Glossary Proxy Opens the Glossary Proxy dialog, which lets you insert a Glossary proxy into a topic. Glossary proxies are used when creating print-based output from your project. When you build the printed output, the Glossary proxy is replaced with the generated glossary from your project. See Creating a Glossary Manually for Print.
  • Insert Index Proxy Opens the Index Proxy dialog, which lets you insert an Index proxy into a topic. Index proxies are used when creating print-based output from your project. When you build the printed output, the Index proxy is replaced with the generated index from your project. See Creating an Index Manually for Print.
  • Insert TOC Proxy Opens the TOC Proxy dialog, which lets you insert a TOC proxy into a topic. TOC proxies are used when creating print-based output from your project. When you build the printed output, the TOC proxy is replaced with the generated TOC from your project. See Creating a TOC Manually for Print.
  • Insert Endnotes Proxy Opens the Endnotes Proxy dialog, which lets you insert an Endnotes proxy into a topic. You would use this proxy if you want to generate a list of your footnotes on a separate page of your output, perhaps at the end of the manual. When you build the output, the Endnotes proxy is replaced with the generated list of footnotes from your project. See Using an Endnotes Proxy.
  • Insert Concepts Proxy Opens the Concepts Proxy dialog, which lets you insert a Concepts proxy into a topic. You would use this proxy if you want to generate a list of the concepts in your project. When you build the output, the Concepts proxy is replaced with the generated list of concepts from your project. See Creating a List of Concepts.
  • Insert List-Of Proxy Opens the List-Of Proxy dialog, which lets you insert a proxy into a topic based on a particular style class. This allows you to create a list of elements, such as a list of images or tables in your project. When you build the output, the proxy is replaced with the generated list. See Creating a List of Elements.
  • Insert Relationships Proxy Opens the Relationships Proxy dialog, which lets you insert a proxy into a topic or master page. This allows you to create a link that is based on a relationship table. When you build the output, the proxy is replaced with the generated link, which lets users quickly navigate to related topics. See About Relationship Tables.
  • Search Bar Proxy Opens the Search Bar Proxy dialog, which lets you insert a proxy into a topic or master page. This allows you to create a field that can be used to perform searches in your output. When you build the output, the proxy is replaced with the generated search bar. See Using the Search Bar Proxy.
  • Menu Proxy Opens the Menu Proxy dialog, which lets you insert a proxy into a topic or master page. This allows you to create a menu based on your TOC file, allowing users to navigate to other topics. This proxy can be especially useful if you are creating HTML5 Top Navigation output and you want a context-sensitive menu to display next to each topic. When you build the output, the proxy is replaced with the generated menu. See Using the Menu Proxy.
  • Search Results Proxy Opens the Search Results Proxy dialog, which lets you insert a proxy into a topic. This proxy works with the Search Bar proxy and allows you to create a container that is used to display the results of end user searches in your output. When you build the output, the proxy is replaced with the generated search results. See Using the Search Results Proxy.
  • Insert Page Header Opens the Page Header dialog, which lets you insert a Page Header proxy into a topic. Page header proxies are used when creating printed output from your project. When you build the printed output, the Page Header proxy is replaced with the page header from the master page your project (see "Body Proxy" above). A page header is simply a block of text that you can format freely and include variables such as page numbers. See Adding Page Header and Footer Proxies.
  • Insert Page Footer Opens the Page Footer dialog, which lets you insert a Page Footer proxy into a topic. Page footer proxies are used when creating printed output from your project. When you build the printed output, the Page Footer proxy is replaced with the page footer from the master page in your project (see "Body Proxy" above). A page footer is simply a block of text that you can format freely and include variables such as page numbers. See Adding Page Header and Footer Proxies.
