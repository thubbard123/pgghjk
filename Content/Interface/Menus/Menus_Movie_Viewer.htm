Menus

The Movie Viewer's interface includes a menu bar at the top of the program window, containing several menu options.

file menu

Open

Opens a dialog, which lets you search for and select a generated Mimic movie file to open.

Page Setup

This option is disabled.

Print

This option is disabled.

Print Preview

This option is disabled.

Recent Files

Opens a submenu, which lets you select one of your most recently opened movie files. The maximum number of recent files saved in the registry is 30.

Exit

Closes the Movie Viewer.
