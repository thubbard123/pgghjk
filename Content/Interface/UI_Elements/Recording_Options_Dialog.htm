Recording Options Dialog

Use this dialog to set options pertaining to the recording mode, sound, callouts, and cursors before you record a movie. You can do the following in this dialog.

  • Callouts (Automatically Add) See Automatically Adding Callouts to Recorded Movies.
  • Callouts (Select Style) See Selecting the Callout Style for Recording.
  • Cursors (Exclude) See Excluding Cursors from Recorded Movies.
  • Cursors (Select Style) See Selecting the Cursor Style for Recording.
  • Magnifier (Show/Hide) See Enabling and Disabling the Magnifier.
  • Recording Mode (Select) See Selecting the Recording Mode.
  • Sound (Enable/Disable) See Enabling and Disabling Sound for Recording.
  • Sound (Select Audio Input Source for Narration) See Selecting the Audio Input Source.
  • Typing Box Shapes (Enable/Disable) See Adding Typing Box Shapes for Recording.
  • Optimize Recording Region (Enable/Disable) See Enabling and Disabling Window-Based Recording Optimization.
