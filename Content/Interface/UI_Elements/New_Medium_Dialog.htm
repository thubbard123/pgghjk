New Medium/Media QueryDialog

This dialog opens when you select Options>Add Medium in the local toolbar of the Stylesheet Editor. You can use the New Medium dialog to create mediums and media queries. See About Mediums and Media Queries.

  • Mediums See Creating Stylesheet Mediums.For more information see the Flare online Help.
  • Media Queries See Creating Media Queries.For more information see the Flare online Help.
Buttons and Sections

Button/Section

Description

Medium

Select this option if you want to create a new medium. Enter the name of the medium in the field.

Media Query

Select this option if you want to create a new media query. The name of the media query will populate based on your selections in the drop-downs.

[Media Features]

Select the media feature you want to apply to the query (e.g., width, height, aspect-ratio).

For more information about the option in this field, and the others related to media queries, see https://www.w3.org/TR/css3-mediaqueries/.

[Other Fields]

Depending on your selections in the previous fields, other fields will display, allowing you to add more information (e.g., size value, unit of measurement).
