Condition Tags Dialog

This dialog opens whenever you attempt to apply condition tags to content or when you attempt to map an undefined condition tag to an existing condition tag. You can perform the following actions from this dialog:

  • Apply Conditions Select the condition tag(s) you want to apply to your content. See Applying Condition Tags to Content.
  • Create/Edit Condition Tags Open the Condition Tag Set Editor, where you can create new condition tags or edit existing condition tags. See Creating Condition Tags.
  • Map a Condition Tag If you have undefined condition tags in your project, you can select a condition tag to map to the undefined tag. See Viewing Undefined Condition Tags.

Button/Section

Description

Condition Tag Sets

If you want to see condition tags from a different condition tag set, select the condition tag set to display it on the right side of the grid.

Condition Tags

Select the condition tag(s) you want to apply to your content or map to the undefined condition tag.

Comment

Additional information about the condition tag.

Edit Set

Click this button if you want to edit the selected condition tag set. This opens the Condition Tag Set Editor.

Unselect All

Click this button if you want to unselect all of the selected condition tags.

Select All

Click this button if you want to select all of the condition tags in the currently selected condition tag set.
