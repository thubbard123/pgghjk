Users Page

The Users page displays all users—both administrators and regular users—who have been added to the MadCap Central license. See About Users.

All active users will be able to see this page, but only administrators with the "User Administration" and/or the "Delete Users" permissions can perform certain tasks (e.g., change the user status and permissions; delete or reinvite users; reset passwords).

What You Can Do on This Page

Following are the primary tasks that you can perform on the Users page:

  • Activity (view) You can view all of the activity for a specific user. See Viewing the Activity for a User.
  • Columns (filter) You can filter any of the columns in the Users grid to limit the information shown. For example, you can filter the Online column to display only the users who are currently logged on to MadCap Central. See Filtering Columns in a Grid.
  • Columns (show/hide) Some columns are displayed by default when you first open the Users page. However, you can add other columns or hide existing ones. See Showing and Hiding Columns.
  • Messages (send) You can use the Message Center to send messages to other users. See Using the Message Center.
  • Permissions (set) Administrators can set permissions for other users. See Setting Permissions.
  • Profile (view) You can view the profile for any user. This shows you various pieces of information about the user, such as contact information, department, and teams/projects to which the user is assigned. See Viewing User Profiles.
  • Status (set) Administrators can change the status of users. This includes the ability to deactivate, reactivate, or reinvite the user. You can also reset the user's password. See Setting a User's Status and Password.
  • Tasks (create and assign) You can create a new task and assign the selected user to it. See Creating and Assigning Tasks.
  • Users (add) Administrators with "User Administration" permission can add new users, inviting them to join the MadCap Central license. See Adding Users.
  • Users (assign to teams/projects) You can associate users with teams and/or projects in Central. See Assigning Users to Teams and Projects.
  • Users (delete) Administrators with the appropriate permission can delete users from the Central license. Deleting a user frees up that seat so that is can be assigned to a new user. See Deleting Users.

Button/Section

Description

Opens the Delete User dialog. If you delete a user, that person will permanently be removed from the license. All access will be denied to the user, and all data associated that person will be deleted. See Deleting Users.

If you want to preserve all of the data for a person you wish to remove, you can deactivate the user instead. All access will be denied to the user on the license until the user is reactivated. Deactivating a user frees up a seat just as deleting a user would. However, all of the user data will be preserved. See Setting a User's Status and Password.

If you delete a user who is active on another Central license key, the user stays in the system (on the other license). If you delete a user who is not active on another key, that user is completely removed from the system.

Shows how many user seats are occupied on the license, followed by how many seats were purchased (e.g., 93/100 means that 93 of 100 available seats are being used).

Lets you find users based on their names or email addresses.

Columns/Grid

Only some of the columns will be on by default. If you do not see some of the following columns on your Users page, you can turn them on by clicking Column Options in the toolbar.

If you click on a row—but not on a linked area—the row will be selected. You can then use toolbar options for that user, or you can use the right-click menu. If you double-click a row, the User Profile dialog opens (see Viewing User Profiles).

You can sort the information alphabetically by clicking any of the column headers.

Column

Description

Opens a drop-down that lets you filter any of the columns in the Users grid to temporarily limit the information shown. See Filtering Columns in a Grid.

Name

Displays the name and avatar for the user from the account settings. This area of the row is also interactive.

Email

Displays the user's email address from the account settings. You can click the address to send the user an email.

Status

Displays the status of the user: Invited, Active, or Deactivated.

If you click on the status for a user, the Set User Status dialog opens. Depending on the current status, this lets you deactivate, reactivate, or reinvite a user. See Setting a User's Status and Password.

Online

Indicates whether the user is online: Yes, No, Away (logged on but idle for 30 minutes).

Last Sign In

Displays the last time the user logged on.

Last Sign Out

Displays the last time the user logged out.
