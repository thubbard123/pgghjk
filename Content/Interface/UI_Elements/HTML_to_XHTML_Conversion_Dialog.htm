HTML to XHTML Conversion Dialog

This dialog is used to convert a single HTML file to XHTML when you import a RoboHelp project or try to open a non-XML file in a project (e.g., if you have copied and pasted an HTML file to your project folder in Windows and then attempt to open it within ).

  • If you select to "Convert all topics at once" during the RoboHelp project import, this wizard is not necessary. However, if you do not select to convert the topics at the time of the import, this wizard can be opened when you try to open an imported topic. See Opening Topics.

    OR

  • Attempt to open a non-XML file in the application.
  1. In the Dynamic Reports section select More Reports>Non-XML Topics.

    The Non-XML Topics window pane opens.

  2. Click the Apply check box next to a single topic row.
  3. In the local toolbar, click Convert Topics.

Button/Section

Description

Original File

Displays the appearance of the topic file before it is converted.

Converted File

Displays the appearance of the topic file after it is converted.

Show Source Code

Displays the source code of the original and converted topics.

Create Backup File

Ensures that a backup copy of the original topic file is created and saved when you convert the file. The backup is stored in the Backups window pane. See Viewing Backup Files.

note

Convert

Converts the file to XML.

Cancel

Closes this wizard without converting the file.
