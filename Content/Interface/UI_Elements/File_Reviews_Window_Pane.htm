File Reviews Window Pane

This window pane lets you work with review documents (topics and snippets) that you have received from others. See About Reviews and Contributions.

There are two different views in the window pane. The information in the window pane changes, depending on the view you select.

  • Inbox This view shows files that reviewers have returned to you.
  • Sent Topics This view shows files that you have sent for review.

To switch from one view to another, click the drop-down field at the top of the window pane and make your selection.

Following are the primary tasks that you can perform in this window pane.

  • Reviewed Files (View and Accept) See Viewing Reviewed Files in the Inbox and Accepting Reviewed Files.
  • Reviewed Files (Import) You can import reviewed files into the inbox. After that, you can view and/or accept the files. You might use this feature, for example, if you have received a file from a reviewer, but rather than immediately sending the file to your inbox from the email message, you saved it to your local drive. See Importing Reviewed Files.
  • Sent Files (View) You can view a list of all files that you have sent to others for review. If you double-click a file from the list, it opens in the XML Editor. See Viewing Files Sent for Review.
