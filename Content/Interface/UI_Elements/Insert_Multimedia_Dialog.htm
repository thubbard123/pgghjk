Insert Multimedia Dialog

This dialog is used to insert multimedia files (YouTube, Vimeo, Flash, Windows Media, Quicktime, WebM, Universal 3D) into topics or snippets.

This dialog is used to insert multimedia files into documents.

  • 3D Models You can insert U3D files. See Inserting 3D Models.
  • Audio You can insert Windows Media audio files. See Inserting Audio Files.
  • Movies You can insert several types of video files, including YouTube and Vimeo movies. See Inserting Movies.See Inserting Movies.
