Feedback Service End User License Agreement Dialog

Use this dialog to review the MadCap Feedback Service license agreement and to enter the license key that you have been issued. This enables you to take advantage of the MadCap Feedback features and to configure a target to Feedback Explorer. See Configuring a Target with Feedback Explorer.

note

This dialog is used only if you are using 's hosted service, as opposed to your own server.
  1. Open the target for which you want to see feedback.
  2. Select the Community tab.
  3. Click Enable Pulse/Feedback Server.
  4. Click the Enter Key button.

Button/Section

Description

[License Agreement]

Displays the Feedback end user license agreement that you should review.

Feedback Service License Key

Enter or paste the Feedback license that you have been issued from .

Accept

Closes the dialog and adds the license key to the Target Editor.

Cancel

Closes the dialog without adding the license key to the Target Editor.
