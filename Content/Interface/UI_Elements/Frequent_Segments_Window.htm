Frequent Segments Window Pane

This window pane is used to do any of the following with frequent segments in your project.

  • View You can view a list of segments that occur frequently in your project. See Viewing Frequent Segments.
  • Convert to Snippets You can convert frequent segments to snippets. See Converting Frequent Segments to Snippets.
  • Convert to Variables You can convert frequent segments to variables. See Converting Frequent Segments to Variables.
  • Customize You can open a dialog that lets you customize the number of occurrences and the number of words in a segment to qualify it for inclusion in the window pane. See Customizing Frequent Segments Shown.
