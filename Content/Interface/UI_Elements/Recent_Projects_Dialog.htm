Recent Projects Dialog

Use this dialog to remove projects from the list of recently opened projects. This allows you to keep the list shorter and more manageable.

  • Select File>Manage Recent Projects.

Button/Section

Description

Recent Projects

Lists the names of the most recently opened project files.

Path

Provides the location of the project.

Open File

Opens the first selected project in the list.

Remove All

Removes all projects from the list.

Remove

Removes the selected project(s) from the list. You can select multiple projects in the list by holding down the SHIFT or CTRL keys and clicking a range of projects or individual ones.
note

When you use this feature, the project is merely removed from the list, but the actual project is not deleted.
