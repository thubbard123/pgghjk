Review Packages Window Pane

This window pane lets you view the files that are part of a review package file sent from an author. You can double-click on a file name to open it so that you can review and edit it. When you are finished and you save your changes, you can click the "Return to Sender" button in the local toolbar to email it back to the author.

  • Open a review package file received from an author.
  • In the Packages section select Review Packages.
