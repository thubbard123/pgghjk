Contents Window Pane (Doc-To-Help)

Note

You can use the Contents window pane to view and edit your table of contents (TOC).

Right-Click

You can right-click on a TOC item to do the following:

  • Convert that topic to a book.

  • Sort the topic list alphabetically.

 Note

You can create custom TOCs for each target. See Creating a Table of Contents. See the online Help or the Documents, Templates, and Content Guide.
