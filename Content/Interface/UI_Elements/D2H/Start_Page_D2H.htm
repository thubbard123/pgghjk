Start Page

From the Start Page window you can do the following:

  • Create/Open/Convert Project You can create a new project, open an existing project, or convert a legacy project. See Creating and Converting Projects. See the online Help or the Getting Started Guide.
  • Team Authoring You can select team authoring actions. See About Team Authoring. See the online Help or the Managing Your Project Guide.
  • Help You can open the online Help.
  • PDFs You can open a topic with access to various PDF guides. See PDF Guides.
  • Videos You can open the video page on the madcapsoftware.com website.
