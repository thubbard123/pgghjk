Variables Window (Doc-To-Help)

Note

The Variables window is used to create and display all the text and rich content variables available in your project.

Variables allow you to manage content in one place for reuse across your project. See Inserting a Variable. See the online Help or the Documents, Templates, and Content Guide.

How to Open the Variables Window
  1. Open the Project ribbon.
  2. In the Project group, choose the Variables button.
