Index and Groups Window Pane (Doc-To-Help)

Note

You can use the Index and Groups pane to view and edit your index or groups. The display in this pane varies based on the toolbar button chosen.

Click (default) in the toolbar to display the project keywords (Index); click to display the project groups. See About Indexes and About Groups. See the online Help or the Documents, Templates, and Content Guide.
