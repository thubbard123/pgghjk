Topics Window

To make your Word source document more useful for online output, it is broken up into smaller elements called "topics," which are based on the heading styles in the document. After you create a Doc-To-Help project and build some output from it, Doc-To-Help automatically creates multiple topics as a result, structuring the content for online documentation. Heading 1 styles in your Word document automatically become parent topics, and all of the Heading 2 styles under a Heading 1 style become subtopics. Just keep in mind that these topics are not created until you build at least one target in your project.

The Topics window displays all of the topics that were created as a result of the most recent build. There are several tasks you can perform in this window.

Drag-and-Drop

From this window, you can select topics that can be dragged and dropped into the Contents, Index and Groups, or Related Topics panes. You can also drag topics into source documents to create links. See Drag-and-Drop Linking. See the online Help or the Documents, Templates, and Content Guide.

If you are working in Word, you can also drag and drop topics from the Contents pane in the Doc-To-Help Project panel to create links in your source documents. See Doc-To-Help Project Panel and Drag-and-Drop Linking.

 Right-Click

You can right-click in the Topics window and select several options, including the following:

  • Open Topic Open the source document and highlight the heading where the topic starts.
  • Show Filter Row Show/hide the filter row.
  • Clear Filter Clear the filter.
  • Show Deleted Topics Show/hide deleted topics.
  • Purge Deleted Topics Purge deleted topics.
  • Default Topic Specify a topic as the default for the project (the first displayed in the online output).
  • Reset Column Layout Return to the default column display.
  • Columns Specify the columns to display.
  • Properties Open the Topic Properties dialog, which lets you choose various settings for the topic. This includes the ability to apply platform, target, or attribute conditions to topics. See About Conditions. See the online Help or the Documents, Templates, and Content Guide.
 Arrange Columns and Sort Information

Columns may be rearranged by selecting the column header and dragging to the desired position. Click on a column header to sort by that column.

 Linking to Deleted Topics

Furthermore, in the Link dialog, you can create links to such a deleted topic without first rebuilding a target that is conditioned to include it.
