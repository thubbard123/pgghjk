Frame Properties Window Pane

The Frame Properties window pane is divided into sections. Each section is used to modify the selected frame's appearance or behavior. If the Object, Sound, Keyframe, or Effect Properties window pane is open, it will change to the Frame Properties window pane as soon as you click on the Frame Editor.

Changes made in the Frame Properties window pane apply only to the current frame. To do the same for all frames in a movie, see Movie Properties Dialog.

Frame Appearance

The Frame Appearance section controls the look of each individual frame.

  • Name You can label the frame, making it easier to identify in the Frames window pane.
  • Background You can also set a background image by choosing an image stored on your computer or by taking a screen capture. See Setting Backgrounds for Frames.
  • Border See Adding Borders.
 Frame Transition

The Frame Transition section controls the length of time a each frame will display during the movie and the way one frame transitions to the next.

  • Time Span See Setting Timing for Frames.
  • Transition See Setting Transitions for Frames.
 Frame Sound

The Frame Sound section controls the audio added to or recorded for each frame in the movie.

  • Sound File You can also record audio for a frame. See Adding Audio to Frames and Recording Audio.
 Frame Image Effects

The Frame Image Effects section controls the amount or blur or shade applied to each frame in the movie.

  • Image Effects After adding a blur or shade effect to a frame, you can edit those effects, customizing the effect factor for each frame. See Editing Effects.
 Master Frames

The Master Frames section controls the master frames being used for each frame in the movie.

  • Master Frames See Applying Master Frames.
 Conditions

The Conditions section controls the conditions applied to each frame in the movie.

  • Conditions See Applying Condition Tags.
 note

If you see a film strip image on a frame in the Frames window pane, that means the frame is using full-motion video (FMV), which is commonly used for dragging and similar actions. If you open the properties for a frame that is using FMV, there are some things that cannot be done in that frame, such as modifying the timing of the frame.
