Dictionary Editor

This editor is used to edit a custom dictionary that you have created while spell checking content. See About Dictionaries and Spell Checking Manually.

  1. Select Tools>Edit Dictionary.
  2. In the Open Dictionary dialog, click in the drop-down field and select Global or Project, depending on the kind of dictionary you want to edit.
  3. In the grid below, select the dictionary you want to edit.

Button/Section

Description

Adds a new dictionary word row to the editor. You can enter the word and (if you choose) a comment for it.

Cuts the selected word row. You can then paste the row into a different dictionary.

Copies the selected word row. You can then paste the copy of the word row in the editor, thus creating a new word. You can also paste the copy of the row into a different dictionary.

Pastes the row that you have cut or copied.

Deletes the selected word row.

Word

Displays the word. To change the text, simply click in the appropriate cell and type a new word.

Comment

Lets you add an optional internal comment next to each word.
