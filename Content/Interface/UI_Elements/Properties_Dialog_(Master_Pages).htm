Properties Dialog (Master Pages)

You can use this dialog to view basic properties (name, location, size, date created, date last modified) of a master page file. You can also use the dialog to perform the following tasks.

  • Condition Tags You can apply condition tags to the master page file. See Developing Targets and Applying Conditions to Content.
  • File Tags You can associate file tags with the master page. See About File Tagging and Associating Tags with Files.
  • Snippet Conditions You can create snippet conditions in a master page. See Using Snippet Conditions, About Snippets, Creating Conditions, and Applying Conditions to Content.
