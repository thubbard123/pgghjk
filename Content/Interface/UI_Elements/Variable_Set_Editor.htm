Variable Set Editor

This editor can be used to do the following with variables:

  • Create You can create custom basic or custom date/time variables within a variable set. You can also use multiple definitions as a way to override variables on a target. See Creating New Variables and Creating Custom Date/Time Variables.
  • Edit After you create a variable, you can easily edit it in the Variable Set Editor. If you change the definition for a variable that has been inserted into topics, the changes will automatically be reflected in all those topics. See Editing Variables and Editing Date/Time Variables.

For more information, see the Flare online Help.
