External Resources Window Pane

The External Resources window pane lets you select and maintain groups of external files that you want to share among projects. The paths of these files are written to the registry so they will be available for all your projects. See About External Resources.

External resources can be virtually any local or network files that you have access to (e.g., images, PDF files, project files). The external resources feature is ideal for shared files that you expect to change over time (e.g., logo images, PDFs, stylesheets), as opposed to, say, a template file that is simply copied into your project and changed only in that project.
