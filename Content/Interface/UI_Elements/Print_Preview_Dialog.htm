Print Preview Dialog

This dialog is used to see how a chart and its data will look when you print it. This includes the ability to zoom in and out.

  1. Create a chart by sending statistics to the Chart window pane. See Creating Charts.
  2. In the local toolbar of the Chart window pane, click the Print Preview button .
