Conditional Text Dialog

This dialog opens whenever you attempt to associate condition tags with content. You might see this dialog when setting condition tags on a target, or when setting condition tags on an import file. You can use basic or advanced conditional expressions to filter your content.

You can create tag expressions when performing any of the following tasks:

You can perform the following tasks from this dialog:

  • Apply Conditions Select the condition tag(s) you want to apply to your content. See Applying Condition Tags to Content.
  • Edit Condition Tags Open the Condition Tag Set Editor, where you can create new condition tags or edit existing condition tags. See Editing Condition Tags.
  • Create Basic and Advanced Condition Tag Expressions When you create tag expressions, you have the choice of working with conditions in Basic or Advanced mode. You can switch between the two modes by clicking Basic or Advanced in the user interface. You can use Basic if you are newer to condition tag expressions, or you can use the Advanced mode if you are experienced at writing condition tag expressions by hand. Buttons and Sections

Button/Section

Description

Condition Tag Sets

If you want to see condition tags from a different condition tag set, select the condition tag set to display it on the right side of the grid.

Condition Tags

Select the condition tag(s) you want to apply to your content.

Include

Select this check box to include the condition tag in your content.

Exclude

Select this check box to exclude the condition tag from your content.

Edit Set

Click this button if you want to edit the selected condition tag set. This opens the Condition Tag Set Editor.

Basic

Select this option to use the Include and Exclude check boxes to set your condition tag expressions.

Advanced

Select this option to write expressions manually, rather than clicking check boxes. You can type the names of the condition tags as well as any of the following tokens: OR, AND, NOT, (). This allows you to produce more complex, robust expressions to control your output (i.e., "and" statements, as well as "or" statements). See Creating Basic and Advanced Tag Expressions.See Creating Basic and Advanced Tag Expressions.

Copy from Basic

Click this button if you want to copy your expression from the Basic section to the Advanced section.

Target Expressions

Select a Flare target from the drop-down. The appropriate conditions are applied for you, using your settings from the Target Editor.
