Select Bookmark Dialog

When creating a link, this dialog is used to select a specific location in a topic—a bookmark or a particular style (including classes and IDs). It is used when inserting a link such as a hyperlink (see Inserting Text Hyperlinks) or cross-reference (see Inserting Cross-References). It is also used in the Properties dialogs for tables of contents (TOCs) and browse sequences when linking an entry to a topic (see Linking TOC Entries and Linking Browse Sequence Entries). If necessary, a bookmark will be created at that location if one does not exist.

Button/Section

Description

Bookmarks

Lists all of the bookmarks in the topic. You can select any of the bookmarks as the destination for the link. For more information see Inserting Bookmarks into Topics.

Headings

Lists all of the headings in the topic (paragraphs associated with an <h1> through <h6> tag). You can select any of the headings as the destination for the link.

Paragraphs

Lists all of the paragraphs in the topic. You can select any of the paragraphs as the destination for the link.

Ordered List Items

Lists all of the ordered (i.e., numbered) lists in the topic. You can select any of the list items as the destination for the link.

Unordered List Items

Lists all of the unordered (i.e., bulleted) lists in the topic. You can select any of the list items as the destination for the link.

Table Content

Lists all of the content found in any tables in the topic. You can select table content as the destination for the link.

Dropdowns

Lists all of the drop-down content (heading and body) in the topic. You can select any parts of the drop-down effect as the destination for the link.

Expanding Text

Lists all of the expanding text content (heading and body) in the topic. You can select any parts of the expanding text effect as the destination for the link.

Styles

Lists all of the style classes and IDs (along with the associated content) in the topic. You can select any of the style classes or IDs as the destination for the link.

Select this to clear a bookmark in the link.
