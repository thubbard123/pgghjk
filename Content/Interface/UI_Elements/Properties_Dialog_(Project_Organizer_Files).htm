Properties Dialog (Project Organizer Files)

This dialog is used to do the following for project files.

  • Display basic properties (name, location, size, date created, date last modified) of a file in the Project Organizer.
  • Mark the file as "Read Only."
  • Set condition tags on a project file. See About Condition Tags and Applying Condition Tags to Content.
  • Associate file tags with a project file. See Associating Tags with Files.
