Start Page

The Start Page is a handy window pane that displays when you first launch and remains open unless you close it. You can use the Start Page to open and create projects, access Help resources, and more.

how to open the start page

The Start Page automatically opens when you first launch . If you close the Start Page, you can reopen it.

To open the Start Page, do the following:

  • In the Pane section, select Start Page.
 Getting Started

This contains links to open an existing project (see Opening Projects) or create a new one (see Creating Alignment Projects).

 Help resources

This tab contains links to additional resources, such as the Help system, the online community, and the MadCap blog.

 recent projects

This lists projects that you have already created. They are listed in order of how recently they were opened. Click on a project link to open that project.

If you want to remove links from this list, you can do so from the Recent Files dialog. See Recent Files Dialog.

In the Start Page, you can pin your favorite projects to keep them displayed at the top of the list.

To pin a project, hover over any recent project and click the little pin. It will then be moved to the top of the list.

To unpin a project, just click the pin again and it will be moved back to the bottom section of the list.

The pinned projects are sorted in order, with the top being the most recently used.

If you want to remove links from this list, you can do so from the Recent Projects dialog. See Recent Projects Dialog.

If you manage your recent projects, and remove them from the Start Page (See Recent Files Dialog), those that are pinned are not removed. If you want to remove a pinned file from the Start Page, you need to unpin it in order for it to be removed.
