Teams Page

The Teams page displays all teams that exist for the current Central license. See About Teams.

All active users will be able to see this page, but only administrators with the "Manage Teams/Projects" and/or the "Delete Teams" permissions can perform certain tasks (e.g., create new teams; edit teams; delete teams).

What You Can Do on This Page

Following are the primary tasks that you can perform on the Teams page:

  • Activity (view) You can view all of the activity for the team. See xxx (create team activity topic??)
  • Columns (filter) You can filter any of the columns in the Teams grid to limit the information shown. For example, you can filter the Online column to display only the users who are currently logged on to MadCap Central. See Filtering Columns in a Grid.
  • Columns (show/hide) Some columns are displayed by default when you first open the Team page. However, you can add other columns or hide existing ones. See Showing and Hiding Columns.
  • Messages (send) You can use the Message Center to send messages to the entire team. See xxx (create team message topic?? combine with user topic??)
  • Profile (view) You can view the profile for any team. This shows you various pieces of information about the team, which users are on the team, which projects are assigned to the team, and the team description. See xxx (create viewing team profile topic)
  • Teams (create) Administrators with "Manage Teams/Projects" permission can create new teams. See Adding Users.
  • Teams (edit) You can assign projects to a team or change the users associated with it in Central. See Editing Teams.
  • Teams (delete) Administrators with the appropriate permission can delete teams from the Central license. See Deleting Teams.
  • Teams (toggle) You can toggle the teams that are displayed on the Teams page to show either all teams or only the teams you belong to. See xxx what?? filter??

Columns/Grid

Only some of the columns will be on by default. If you do not see some of the following columns on your Teams page, you can turn them on by clicking Column Options in the toolbar.

If you click on a row—but not on a linked area—the row will be selected. You can then use toolbar options for that team, or you can use the right-click menu. If you double-click a row, the Team Profile dialog opens (see xxx).

You can sort the information alphabetically by clicking any of the column headers.
