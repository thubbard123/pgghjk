Index Links Dialog

This dialog opens whenever you add a new index link to your index link set. An index link is an entry in a generated index that points to another entry. There are three kinds of index links—"See," "See Also," and "Sort As." See Creating Index Links.

Button/Section

Description

Term

Type the term that you anticipate users may look for in the generated index. Please note that the fields in this dialog are case-sensitive; therefore, if you enter a term in all lowercase letters, that is how it will be displayed in the output.

If you want to create a multi-level index entry, separate the first-level term and the second-level term with a colon (e.g., a single-level entry might be "Pasta"; a multi-level entry might be "Pasta:rigatoni").
