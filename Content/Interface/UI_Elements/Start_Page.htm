The Start Page

how to open the start page

The Start Page automatically opens when you first launch . If you close the Start Page, you can reopen it.

  • Quick Launch Bar Press CTRL+Q to switch focus to the Quick Launch Bar in the upper-right corner of the workspace. Begin typing Start Page and select it from the list when it appears.
  • Ribbon In the Common section, select Start Page.

Sections of the Start Page

Following are explanations of the different sections in the Start Page.

getting started

This contains links to open an existing project (see Opening a Project) or create a new one (see Creating a New Project).

 recent projects Home

This area contains links to Getting Started information designed for new users.

templates

This area lets you browse project templates and click to start a new project based on any of them.
