Link to Topic Dialog

This dialog is used to create a link to a topic from several different locations (e.g., from a table of contents entry).

  • When you perform a task such as creating a TOClinking a TOC item to a topic, click the Topic, Select a Topic, or browse button. See Creating a Table of Contents.

Button/Section

Description

Toggles between showing or hiding the subfolders holding the topic files.

Splits the dialog into two halves. The right half shows additional details for the files. When you click a file on the right side, a red circle shows the location on the left side.

Moves up one folder.
