Accept Imported Documents Dialog

This dialog lists the files to be imported as a result of your selections in any of the following.

  • Project Import Editor See Global Project Linkingâ€”Importing Files from Other Projects.
  • Import Microsoft Word Wizard See Importing Word Files.
  • Word Import Editor See Importing Word Files.
  • Import FrameMaker Wizard See Importing FrameMaker Files.
  • Frame Import Editor See Importing FrameMaker Files.
  • Import DITA Wizard See Importing DITA Files.
  • DITA Import Editor See Importing DITA Files.

The dialog is slightly different for Flare project import files than it is for Word, FrameMaker, or DITA import files. The dialog provides you with a last look at the files to be imported before they are actually brought in to the project.

In the case of Flare project import files, you also have the opportunity to select individual files to include or exclude from the import. You can also see the status of individual files, which lets you see whether the imported file is newer than your local file, or vice versa. For more information about project import files, see Rules for Global Project Linking.
