Project Analysis Window Pane

This window pane is used to display information collected by the built-in Analyzer. See About Analyzer.

Following are the types of information you can see in this window pane. Click in the drop-down field at the top of the window pane to select a different view.

Project Analysis Information
  • Broken Bookmarks See Viewing and Fixing Broken Bookmarks.
  • Broken Links You can See Viewing and Fixing Broken Links.
  • Database Errors You can See Viewing Database Errors.
  • Files With Changes See Viewing Files with Changes.
  • Files With Annotations See Viewing Files with Annotations.
  • Topics Not in Index You can See Viewing Topics Not Indexed.
  • Topics Not in Selected TOC You can See Viewing Topics Not in a TOC.
