Convert Files to XML Wizard

This wizard is used to convert multiple HTML files to XHTML (e.g., if you add HTML files to your project folder in Windows and then attempt to open them within ).

    • Ribbon In the Explorer section select File List.
    • Keyboard Shortcut Press CTRL+SHIFT+J.

    The File List window pane opens.

  2. Select multiple non-XML (HTM) files. You can hold down your SHIFT or CTRL key and select a range or individual files.
  3. Right-click and select Convert to XML.

Button/Section

Description

Open File

Opens the selected HTML file in a browser window.

Preview Conversion

Opens a window that lets you see how the selected file will look after the conversion to XHTML.

Create Backups of Original Files

Ensures that backup copies of the original topic files are created and saved when you convert the files. The backup is stored in your Documents\Conversion Backups subfolder.

note

Remove

Removes the selected HTML file from the wizard.

Convert

Converts the files to XML.

Cancel

Closes this wizard without converting the files.
