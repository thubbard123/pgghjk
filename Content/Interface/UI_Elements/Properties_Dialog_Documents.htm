Properties Dialog (Documents)

This dialog displays basic properties (name, location, size, date created, date last modified) of the document. It also lets you set various other properties for the document.

Following are the primary tasks performed in this dialog.

  • Disable Empty Page (Auto-end on Left Page) In a Flare project an author can use empty pages to ensure that chapters always start on a right (odd) page. If an author does not want to have an empty page before certain chapters, that feature can be disabled. If this affects the document that you are contributing to an author, select the Page Properties tab, click the arrow in the Auto-end on 'left' page field, and select Disabled.

  • First Page Type On the Page Properties tab you can assign one of the following types to determine how that your contribution document behaves in an author's output: Normal, Title, First, Empty, Left, Right.

  • Language On the Language tab you can select a language to be used for the document.
  • Page Layout On the Page Properties tab you can see the page layout currently associated with the document.
  • Stylesheet On the Page Properties tab you can see the stylesheet currently associated with the document.
  • Title On the Page Properties tab, you can change the name of the topic title.
