The Start Page

how to open the start page

The Start Page automatically opens when you first launch . If you close the Start Page, you can reopen it.

  • In the Common section, select Start Page.
 getting started

This contains links to open an existing file or create a new one (see Creating New Documents).

 recent Files

This lists recently opened files. They are listed in order of how recently they were opened. Click on a link to open that file.

If you want to remove links from this list, you can do so from the Recent Files dialog. See Recent Files Dialog.

In the Start Page, you can pin your favorite files to keep them displayed at the top of the list.

To pin a file, hover over any recent file and click the little pin. It will then be moved to the top of the list.

To unpin a file, just click the pin again and it will be moved back to the bottom section of the list.

The pinned files are sorted in order, with the top being the most recently used.

If you manage your recent files, removing them from the Start Page (See Recent Files Dialog), those that are pinned are not removed. If you want to remove a pinned file from the Start Page, you need to unpin it in order for it to be removed.
