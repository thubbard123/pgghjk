Frame Contents Window Pane

Use this window pane to add or edit text and other content in a page layout frame. You must have a page open in the Page Layout Editor with a header, footer, or decoration frame selected for text edit in order to enter content into this window pane. This window pane functions much like the XML Editor that is used for editing topics, only on a smaller scale. See About Page Layouts and Adding Text and Content to Frames.

Opens a dialog that you can use to insert a snippet. See Inserting Snippets.
