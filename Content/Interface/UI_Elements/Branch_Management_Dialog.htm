Branch Management Dialog

This dialog is used to create, delete, switch (i.e., select), and publish branches.

  • Create You can create new branches. See Creating Branches.
  • Delete You can delete existing branches. See Deleting Branches.
  • Switch You can switch to another branch so you can commit files to that branch. See Selecting Branches.
  • Publish You can publish a new branch so you can push to, pull from, and synchronize that branch. See Publishing Branches.
