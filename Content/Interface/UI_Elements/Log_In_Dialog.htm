Log In Dialog

This dialog may open if you are binding your project to source control. Use the dialog to enter the user name and password required for accessing the server where the source control files are located. See About Source Control and Multi-Authoring.

Button/Section

Description

Server

Displays the server where the source control database is located.

User name

Enter the appropriate user name to access the server. (You may need to check with your network administrator.)

Password

Enter the appropriate password to access the server. (You may need to check with your network administrator.)

Save Password

Click this check box if you want to remember your password so that you do not need to re-enter it in the future.
