Properties Dialog (Generic)

This dialog displays when you have selected more than one type of file (e.g., topic file, stylesheet file, image file)—perhaps from the File List window pane—and then opened the properties. The general, file tag, and conditional text properties for the selected files are shown in this dialog.

This dialog displays translation, file tag, and general properties for the selected file.

 translation tab

Use this tab to set the translation status of a particular file. The status that you select affects the "Status" column in the File List window pane.

Button/Section

Description

Translation Status

Select the translation status for the selected file.

  • Untranslated
  • In Progress
  • Completed
  • Reviewed
  • Approved
  • Sent for Translation
  • Sent for Review
file tags tab

If you are translating a MadCap Flare project, use this tab to view and edit file tag values. See About File Tagging.

 general tab

Use this tab to mark the files as "Read Only" so that they cannot be modified. Other information is also shown on this tab, although the word "multiple" will often display in many of the fields (rather than the actual data) if more than one file has been selected.

file tags tab

Use this tab to associate files with tags. See Associating Tags with Files.

 conditional text tab

Use this tab to apply condition tags to the files. For more information about this feature, see Developing Targets and Applying Conditions to Content.

Button/Section

Description

Condition Tag Sets

Lists all the condition tag sets in your project. After you select a condition tag set in the list, the condition tags within it are displayed to the right.

Condition Tags

Lists the condition tags in the set. Click the check box next to each tag that you want to associate with the file. For more information about this feature, see About Conditions and Applying Conditions to Content.

note

If multiple files are selected, you might see a check box filled with color instead of a check mark. This indicates that the condition tag in question has been applied differently to the various files that you have selected (i.e., one file might have this condition tag selected while another file does not).

Edit Set

Opens the Condition Tag Set Editor, so that you can make changes to the condition tags in it.

Unselect All

Removes check marks from all condition tags above.

Select All

Adds check marks to all condition tags above.
