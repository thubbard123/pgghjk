Properties Dialog

This dialog displays basic and conditional text properties for the selected file.

general tab

Use this tab to mark the files as "Read Only" so that they cannot be modified. Other information is also shown on this tab, although the word "multiple" will often display in many of the fields (rather than the actual data) if more than one file has been selected.

conditional text tab

Use this tab to apply condition tags to the files.

Button/Section

Description

Condition Tag Sets

Lists all the condition tag sets in your project. After you select a condition tag set in the list, the condition tags within it are displayed to the right.

Condition Tags

Lists the condition tags in the set. Click the check box next to each tag that you want to associate with the file. For more information about this feature, see About Conditions, Developing Targets, and Applying Conditions to Content.

note

If multiple files are selected, you might see a check box filled with color instead of a check mark. This indicates that the condition tag in question has been applied differently to the various files that you have selected (i.e., one file might have this condition tag selected while another file does not).

Edit Set

Opens the Condition Tag Set Editor, so that you can make changes to the condition tags in it.

Unselect All

Removes check marks from all condition tags above.

Select All

Adds check marks to all condition tags above.
