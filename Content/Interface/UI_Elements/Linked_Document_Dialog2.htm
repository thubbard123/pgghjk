Linked Document Dialog

This dialog opens if you try to edit content that is linked to other project files.

  1. Import files from another project. See Global Project Linking—Importing Files from Other Projects.
  2. Open a file that you imported and attempt to make a change.

Button/Section

Description

Link

Displays the source file.

Continue with operation

Closes the dialog, keeps the link to the source project, and lets you continue editing in the file. If you try to save the file, you will again see a dialog asking what you want to do.

Remove the link and continue the operation

Removes the link to the source project and lets you continue editing in the file.

Remove the link and cancel the operation

Removes the link to the source project and adds that file to the Removed Links tab of the Project Import Editor. Therefore, the file will not be included in future imports from the source project.
