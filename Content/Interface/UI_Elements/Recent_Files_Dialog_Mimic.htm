Recent Files Dialog

Use this dialog to remove files from the list of recently opened files. This allows you to keep the list shorter and more manageable.

  • Select File>Manage Recent Files.

Button/Section

Description

Recent Files

Lists the names of the most recently opened files.

Path

Provides the location of the file.

Open File

Opens the first selected file in the list.

Remove All

Removes all files from the list.

Remove

Removes the selected file(s) from the list.

notes

When you use this feature, the file is removed from the list, but the actual file is not deleted.

The maximum number of recent files saved in the registry is 30.
