Attributes Window Pane

This window pane is used to set attributes for a tag. Attributes let you provide additional information about tagged content and how that content should behave.

example

example

You can use the "dir" attribute to change the base direction of a block of text. The default value for the "dir" attribute is "ltr" (left-to-right), but you can change the value to "rtl" (right-to-left). If you apply the "rtl" value to tagged content, that content will start at the far right edge of the window (as opposed to the left) and move to the left as the amount of content increases. You would set this attribute by clicking on an area of tagged content, clicking in the right column next to dir, and selecting rtl from the drop-down.

  • Ribbon In the Attributes section select Attributes Window.
  • Keyboard Shortcut Press SHIFT+F12.

Description

[Tag]

Displays the tag (e.g., <h1>, <p>, <td>) that is applied to the content where the cursor is currently placed. The changes that you make in the Attributes window pane will be applied to that tagged content in that topic.

Displays all of the attributes in the window pane.

Displays only the attributes provided by . Your custom attributes are not displayed.

Displays only the custom attributes that you create. Attributes provided by are not displayed.

Opens the Enter Custom Attribute dialog, which lets you create a new attribute and provide a value for it.

Selects the content of the parent tag each time you click this button.

example

example

Let's say you have applied bold to a word in your topic. If the cursor is located on that word and you click this button, the content in the parent tag (e.g., <p>) will be selected. If you click the button yet again, the content in the <body> tag will be highlighted. And if you click the button once more, the content in the <html> tag will be selected.

[Attribute]

The left column in the window pane displays the attributes applicable to the tag where your cursor is placed.

[Value]

Use the right column in the window pane to set the value for the attribute in the left column.
