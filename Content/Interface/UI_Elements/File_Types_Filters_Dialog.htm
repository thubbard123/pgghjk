File Types Filters Dialog

When you create a Lingo project from a folder, you can create filters for any Word, Excel, XML, or text files that may be in the folder so only certain parts of the file are imported for translation. You can also add filters to unsupported file types using a text file type filter. See About File Filters.

How to Open the File Types Filters Dialog
  1. Select File>New Project.
  2. Begin the process of creating a project. See Creating New Projects.

  3. On the second page of the wizard, click Add Folder. In the dialog that opens, select the folder you want to add to the project.

  4. If you want to create file type filters for the files within the folder, from the Filter drop-down select Customize Folder. The File Types Filters dialog opens.

How to Use the File Types Filters Dialog

  1. In the Translate column, select the file types you want to import. You can click the Select All check box to add all the file types at once.

  2. From the Filter drop-down, select the filter you want to use for each translatable file type.

    You cannot select a filter for file types that cannot be translated, such as image files.

  3. If you want to create a new filter, from the Filter drop-down, select Create New. See About File Filters.

 Buttons and Sections

Button/Section

Description

File Type

The file types available in your folder.

Translate

When this check box is selected, the file type is included for translation. If the check box is not selected, the file type will not be translated.

Select All

When this check box is selected, all available file types are included for translation. You can also deselect the check box to clear the Translate check boxes for all file types.

Filter

A list of existing filters for each file type. You can also select Create New to create a new file filter. See About File Filters.
