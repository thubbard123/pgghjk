Incoming Topic Editor

This editor can be used to view and accept an MCCO contribution topic. See About Reviews and Contributions.

  1. In the Resources section select Contributions.

    The Contributions window pane opens.

  2. Click in the drop-down field at the top of the window pane, select Contributions. All contributed topics that you have brought into your project, but have not yet been accepted, are listed in the pane.
  3. Double-click one of the rows.

Button/Section

Description

Deletes the selected item.

Opens the selected item.

Lets you import a contribution document.

Lets you accept the contribution topic. See Accepting Contribution Files.
