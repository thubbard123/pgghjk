Create Glossary Link Dialog

This dialog opens when you add a glossary term link to your content. From this dialog you can insert glossary terms and create new glossary terms. See Inserting Glossary Term Links.

Button/Section

Description

Select Glossary File

If you have more than one glossary file, select the glossary you want to use.

Select Glossary Term

Select the glossary term you want to link to.

Definition

The definition of the selected glossary term appears in this section.

Add New Term

Click this button if you want to add a new term to the selected glossary. The Create Glossary Term dialog opens. See Create Glossary Term Dialog.
