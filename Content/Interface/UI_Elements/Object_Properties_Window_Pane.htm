Object Properties Window Pane

The Object Properties window pane opens by default on the right side of Mimic. This window pane is dynamic and changes depending on the type of object you have selected.

Arrow

You can control the length, width, curvature, and direction of an arrow. See Setting the Shape of an Arrow.

Arrow
  • Tail You can control the last half of the arrow's body. This allows you to create different tail shapes, such as a scalloped end.
  • Head Tail You can control the sides of the arrow's head. This allows you to create different shapes for the head's tail, such as very long sides.
  • Head You can control the arrow's head. This allows you to create a rounded arrow head as well as other head shapes.
  • Sides You can change the length and width of the sides of the arrow's body. Additionally, you can choose whether or not the sides of the arrow's body should be curved, and how much they should curve.
  • Rotation You can change the direction of the different arrow parts. This allows you to create, for example, an arrow body that goes straight up with an arrow head that points to the right.
  • Overall You can change the curve factor for the entire arrow.
 Button

Button properties allow you to modify the appearance and behavior of a button that has been added to a movie frame. See Adding Buttons.

Bubble

Callout properties allow you to modify the appearance and behavior of a callout that has been added to a movie frame. See Adding Callouts.

Cursor

Cursor properties allow you to edit the look of cursors. See Adding Cursors.

Cursor
  • Cursor You can select a cursor type. See Selecting Cursor Types.
 Failure Caption

You can modify the appearance and behavior of an input failure caption. See Adding Input Captions.

 Hint Caption

Hint caption properties allow you to modify the appearance and behavior of a hint caption. See Adding Input Captions.

 Image Object

Image object properties allow you to modify some aspects of an image you have inserted into the movie frame. See Adding Images as Objects.

Image
  • Source Image See Replacing Image Objects.
 Rectangle
  • Anchor See Setting Object Anchors.
 Input Box

You can use input box properties to modify appearance and behavior of an input box. See Adding Input Boxes.

Input Box
  • Input Box You can enter one or more correct entries for the input box. Additionally, you can determine how many chances a user gets to input a successful entry. See Adding Input Boxes.
 Loop

Loop properties allow you to modify the shape and thickness of a loop object.

Loop
  • Coordinates Adjust the Start, Top, Left, Bottom, Right, Inner Top, and End coordinates to change how the loop looks. See Setting the Coordinates for a Loop.
  • Thickness Adjust the thickness of the sections of the loop, or make changes to the Scale All field to adjust the thickness of the entire loop at once. See Setting the Thickness for a Loop.
 Oval

Oval properties allow you to modify a circle shape.

Polygon

Polygon properties allow you to modify the appearance of a polygon shape.

Polyline

Polyline properties allow you to set the appearance of a line, including adding or removing arrow parts. See Adding Lines.

Polyline

When you add a line to a frame, an arrow is automatically added at its head. You can choose to remove the arrow head from your polyline. However, if the line will be used as an arrow, you can also set various arrow head and tail options. See Setting the Arrows for a Line.

  • Arrow Heads You can completely remove the arrow head from the line, change the position where the head will display on the line, or you can add multiple arrow heads. Additionally, you can change the position, size, and color of the arrow head.
  • Arrow Tails You can add an arrow tail to a line, change the position where the tail will display on the line, or you can add multiple arrow tails. Additionally, you can change the position, size, and color of the arrow tail.
 Rectangle

Rectangle properties allow you to set the appearance of a rectangle shape.

 Rollover Caption

Rollover caption properties allow you to modify the rollover caption appearance. See Adding Rollover Captions.

 Success Caption

Success caption properties allow you to set the appearance of a success caption. See Adding Input Captions.

 Text Rectangle

Text rectangle properties allow you to modify the appearance and content of a text rectangle object.

 Typing Box

Typing box properties allow you to set the appearance and content of a typing box. See Adding Animated Typing Boxes.

Typing Box
  • Start Text You can enter text here if you want the text to display within the typing box as soon as the typing box appears in the movie.
  • Typed Text You can enter text here if you want the letters to appear as if they are being "typed" into the typing box. Once the typing box appears in the movie, the letters are automatically typed one by one in the typing box.
 NoteS

If more than one object is selected, the Object Properties window pane will change to a Properties window pane. Select a single object in order to edit its properties.

Visual states show a limited version of the Object Properties window pane. For all objects, there is a General section that allows you to update the state name and an Object Appearance section that allows you to adjust the opacity of a visual state. Additionally, all objects, excluding polylines and image objects, have a Background section that allows you to change the color of an object for a visual state.
