Visual States Window Pane

The Visual States window pane opens by default on the left side of Mimic. This window pane allows you to add one or more visual states to an object. This gives an object a different look when a user performs an action (e.g., clicking a button, moving your mouse over an object). You can use this window pane to perform the following tasks:

How to Open the Window Pane

  • In the Tool Windows section, select Visual States.

    The Visual States window pane opens.

 Tasks you can Perform in this Window Pane
  • Add a Visual State You can add opacity states to all objects, and background color states to all objects except polylines and image objects. See Adding Visual States to an Object.
  • Edit a Visual State Once you have added a visual state to an object, you can edit it instead of deleting it and creating a new state from scratch. See Editing Visual States.
  • Remove a Visual State You can remove a visual state if it is no longer needed. See Removing Visual States.
