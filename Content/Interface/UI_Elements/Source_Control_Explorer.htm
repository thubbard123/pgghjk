Navigation menu

Quickly navigate to panes in the Source Control Explorer without needing to return to the Home pane.

  • Home Opens the Home pane.

  • Pending Changes Opens the Pending Changes pane.

  • Settings Opens the Settings pane.

  • Branches Opens the Branches pane. This pane is only available if you are using Git.
Home Pane

Button/Section

Description

Pending Changes

Opens the Pending Changes pane.

Settings

Opens the Settings pane.

Branches

Opens the Branches pane. This pane is only available if you are using Git.

Import Project

Opens the Import Project from Source Control Wizard dialog. See Importing a Project from Source Control.

Network Settings

Opens the Network settings dialog. This option is only available is you are using Git or Subversion. See Modifying Network Settings.
Settings Pane

Button/Section

Description

Provider Type

View the source control provider you are using for your project.

Database

View the source control database where your project is bound.

Database Folder

View the source control database folder where your project is bound.

Unbind

Unbind your project from source control. This removes project information from the Settings fields. See Unbinding a Project from Source Control.

Bind

Bind your project to source control. This fills in project information in the Settings fields. See Binding a Project to Source Control.
Pending Changes Pane

Button/Section

Description

Commit/Submit/Check In

Send your local files to source control. The terminology on this button changes depending on the source control provider you are using. See Checking In, Committing, and Submitting Source Control Files.

Revert/Undo Check Out

Return your local files to their most recent state. The terminology on this button changes depending on the source control provider you are using. See Reverting or Undoing a Checkout of Source Control Files.

Comment

Add a comment to the commit/submit/check in. You can view the comment in the file's history. See Viewing the History of Source Control Files.

Included Changes

The files in this section will be included in the next commit/submit/check in. You can right-click any file or folder to add it to the Excluded Changes section.

Excluded Changes

The files in this section will be excluded from the next commit/submit/check in. You can right-click any file or folder to add it to the Included Changes section.
Branches Pane

Button/Section

Description

Published Branches

Shows available published branches. See Publishing Branches.

Unpublished Branches

Shows available unpublished branches. See Publishing Branches.

Create Branch

Create a new unpublished branch. See Creating Branches.

Delete Branch

Delete the selected branch. Deleting the branch also deletes any commits on that branch. See Deleting Branches.

Select Branch

Switch to the selected branch. See Switching Branches.

Publish Branch

Publish the selected branch. See Publishing Branches.
Notes

The Branches pane is only available if you are using Git.
