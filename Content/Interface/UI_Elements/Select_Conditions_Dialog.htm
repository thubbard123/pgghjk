Select Conditions Dialog

Use this dialog to select the conditions you want to exclude from a Lingo project. This lets you filter the project by condition. You can filter out conditions from a new project (using the Create New Project wizard) or an existing project (using the Project Properties dialog). See Creating New Projects and Project Properties Dialog.

  1. In the Select conditions to exclude area, select a condition set from the left side of the grid. The tags in that condition set appear on the right side of the grid.
  2. On the right side of the grid, select the check boxes next to the condition tags you want to exclude from the project.
  3. Click OK.

For more information, see Dynamic Filtering.
