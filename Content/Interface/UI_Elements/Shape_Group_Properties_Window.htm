Shape Group Properties Window Pane

The Shape Group Properties window pane opens by default on the right side of Mimic. If the Object, Sound, Keyframe, Effect, or Frame Properties window pane is open, it will change to the Shape Group Properties window pane as soon as you group a set of objects or click on an existing grouped object set.

How to Open this Window Pane
  1. Open a movie that contains one or more sets of grouped shapes. See Grouping Objects.
  2. Click a shape group.

  3. In the Properties section, select Properties.

    The Shape Group Properties window pane opens.
