Manage Movie Sound Library Dialog

This dialog lists any audio files (such as MP3 and WAV files) that you have imported into the movie or recorded, as well as how many times each file has been used in the movie. You can use this dialog to do the following:

  • Add Audio to Cursors See Adding Audio to Cursors.
  • Add Audio to Frames See Adding Audio to Frames.
  • Add Audio to Movies See Adding Audio to Movies.
  • Rename Audio Files See Renaming Audio Files.
