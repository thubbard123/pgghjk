Confirm Replace Dialog

This dialog opens whenever you attempt to replace a file with the same name.

Button/Section

Description

Replace

Click this button if you want to replace the file in the destination folder with the file you are copying.

Skip

Click this button to leave the files as they are, without replacing the file in the destination folder.

Rename

Click this button if you want to copy the file to the destination folder but keep the other file as well. In this case, the file you are copying will be renamed (e.g., from "Topic" to "Topic1") so that both files can be retained.

Do this for all conflicts

Click this check box if multiple files are involved and you want the same action to be applied to all of them.
