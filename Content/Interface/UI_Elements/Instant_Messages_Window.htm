Instant Messages Window Pane

This window pane displays instant messages from other users and lets you send and configure instant messages. Following are the primary tasks that you can perform in the Instant Messages window pane.

  • Instant messages (reply)  See Replying to Instant Messages.
  • Instant messages (send)  See Sending Instant Messages.
  • Instant messages (view)  See Viewing Instant Messages.
  • Status (select)  See Selecting an Instant Messaging Status.
  • Users (view)  See Viewing Instant Messaging Users.
