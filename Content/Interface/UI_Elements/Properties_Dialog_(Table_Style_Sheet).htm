Properties Dialog (Table Stylesheet)

This dialog displays basic properties (name, location, size, date created, date last modified) of the table stylesheet file. It is also used to change some properties of the file.

Following are the primary tasks that you can perform in this dialog.

  • Name (Edit) You can change the name of the file.
  • Read Only You can mark the file as "Read Only" so that it cannot be modified.
  • File Tags (Associate) You can associate file tags with the table stylesheet. See About File Tagging and Associating Tags with Files.
  • Condition Tags (Apply) You can apply condition tags to the table stylesheet. See Developing Targets and Applying Conditions to Content.
