Confirm Merge Folder Dialog

This dialog opens whenever you attempt to move a folder into a location that contains folders and files with similar names.

Button/Section

Description

Do this for all folders

Click this check box if multiple folders are involved and you want the same action to be applied to all of them.

Yes

Click this button if you want to continue with the file merge. If you elect to continue merging the folders, you will be asked if you want to replace files within them that have with similar names.

Skip

Click this button to leave the folders as they are, without merging them.
