Select Table Style Template Dialog

You can use the Select Table Style Template dialog to create a new table stylesheet while inserting a table into a topic or splitting table cells. After you finish making your changes to the table, the new stylesheet is added to your project.

Buttons and Sections

Button/Section

Description

Template Folders

Select the folder where the template on which you want to base your new stylesheet is stored.

Templates

Select the template on which you want to base your new table stylesheet.

Source File

The file path of the source template.

New Style Name

Enter a name for the new table stylesheet.

Preview

A preview of a table using the settings from the template appears in this area.
