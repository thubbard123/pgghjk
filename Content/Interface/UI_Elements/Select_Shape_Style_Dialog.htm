Select Shape Style Dialog

Use this dialog to select a style (i.e., a look and feel) to be used for bubble callouts or cursor shapes during movie recordings. When you record a movie, callouts and cursor movements are automatically added to the movie frames (unless you specify to exclude them). See Recording Movies.

  • Callout Style See Selecting the Callout Style for Recording.
  • Cursor Style See Selecting the Cursor Style for Recording.
