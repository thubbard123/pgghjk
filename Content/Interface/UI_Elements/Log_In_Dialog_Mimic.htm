Log In Dialog

Use this dialog (when publishing generated movie output) to enter the user name and password required for accessing the server to which you are publishing. If you do not enter the user name and password ahead of time (via the Publish Movie dialog), another dialog will open later when you actually publish the output, asking for the user name and password.

if you are publishing a collection

  1. Open the collection.

    • Ribbon In the Collection Output section, select Publish.

    • Keyboard Shortcut Press CTRL+F6 on the keyboard.

    The Publish Movie dialog opens.

  3. Complete the fields in the Publish Movie dialog. Make sure you are publishing to an FTP site, rather than using "File System."
  4. Click Login Credentials.

if you are publishing a standalone movie

  1. Open the movie.
    • Ribbon In the Movie Output section, select Publish.
    • Keyboard Shortcut Press CTRL+F6 on the keyboard.

    The Publish Movie dialog opens.

  3. Complete the fields in the Publish Movie dialog. Make sure you are publishing to an FTP site, rather than using "File System."
  4. Click Login Credentials.

Button/Section

Description

Server

Displays the server where the output is being published.

User name

Enter the appropriate user name to access the server. (You may need to check with your network administrator.)

Password

Enter the appropriate password to access the server. (You may need to check with your network administrator.)
