New Dialog

This dialog lets you create a new movie or collection in any of the following ways.

  • Record Movie You can quickly begin recording a new movie, whether it is a standalone movie or part of a collection. See Recording Movies.
  • Create Movie Collection You can open a wizard to create a new movie collection. See Creating a Movie Collection.
  • Create Movie from Media File You can select an image or video and create a new movie based on it. See Creating Movies from Media Files.
  • Create Movie from Screenshot You can capture a screenshot and create a new movie based on it. See Creating Movies from Screen Captures.
  • Import PowerPoint You can create a movie by importing a Microsoft PowerPoint presentation. See Importing PowerPoint Presentations.
  • Create Blank Movie You can open a dialog, which lets you create a new movie with an empty frame. See Creating Blank Movies.
