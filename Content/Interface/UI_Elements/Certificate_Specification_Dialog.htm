Certificate Specification Dialog

This dialog opens when your project uses a Git SSH repository, and needs to access your SSH keys. You may see it when binding the project, or performing tasks such as pushing, pulling, committing, or synchronizing files.

Button/Section

Description

Public key

Enter your public SSH key. You can also click to browse for the key on your network.

Private key

Enter your private SSH key. You can also click to browse for the key on your network.

Save certificate information

Select this check box if you want to remember your key information so you do not need to enter it again later.
