Global Toolbars

Global toolbars can be seen only if you are working in the traditional tool strip view, as opposed to the ribbon view.

standard toolbar

Tools in the Standard toolbar let you perform functions such as opening projects to analyze, opening interface window panes, and sending selected files to Flare so that you can work on them. To see this toolbar, select View>Toolbars>Standard.

 

Description
