Changing Hot Keys

Mimic provides you with several different options for recording images. Some of these options have shortcut hot keys assigned to them. For example, the default shortcut for opening the Timeline is CTRL+T. If you want, you can change the hot key combinations for any of these options (e.g., you can change the CTRL+T shortcut to, say, CTRL+SHIFT+T).

how to change hot keys
  1. Select File>Options.

    The Options dialog opens.

  2. Click the Keyboard Shortcuts tab.

  3. In the Command Scope section, select the option whose hot keys you want to change.
  4. In the Key Assignment section, select the hot key that you want to assign to the option.
  5. (Optional) In the Modifier Key Assignment section, you can select the Control Key, Shift Key, and/or Alt Key check boxes if you want to use a combination for the shortcut. For example, if you select D from the list to the right and add check marks to the Shift Key and Alt Key boxes, the new shortcut for the option will be SHIFT+ALT+D.
  6. (Optional) In the Options section, you can assign a hot key to a command that does not appear in the grid above by clicking Unlisted Commands. In the Other Commands dialog, select a command, then click OK. The unlisted command appears in the grid.
  7. (Optional) In the Options section, you can return to the factory default keyboard shortcuts by clicking Reset to Factory Defaults.
  8. Click OK.
