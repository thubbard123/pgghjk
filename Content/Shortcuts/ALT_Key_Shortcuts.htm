ALT Key Shortcuts

You can use ’s ALT key shortcuts to access ribbon and menu (i.e., tool strip) options.

How To Use ALT Key Shortcuts in Ribbons
  1. Press and release the ALT key on your keyboard. You’ll see letters (representing keyboard keys) floating above different ribbon tabs.
  2. Press a key (not case-sensitive) to open the corresponding ribbon (even if the desired ribbon is already open). You'll see additional floating letters above the various options on that ribbon.
  3. Press the key(s) corresponding to the option you want to access.

Example

Let's say you want to insert a cross-reference using the option in the Insert ribbon. First, you press and release the ALT key, which shows floating letters over each ribbon tab.

Press the N key to open the Insert ribbon.

Press the R key, followed by the F key.

The Insert Cross-Reference dialog opens.

Example

Let's say you want to open the Profiles Editor using the option in the View ribbon. First, you press and release the ALT key, which shows floating letters over each ribbon tab.

Press the V key to open the View ribbon.

Press the R key.

The Profiles Editor opens.

Let's say you want to open the Timeline using the option in the View ribbon. First, you press and release the ALT key, which shows floating letters over each ribbon tab.

Press the V key to open the View ribbon.

Press the L key.

The Timeline opens.

 How To Use ALT Key Shortcuts in Menus
  1. Press and release the ALT key on your keyboard. On the menu labels, you’ll see underlined letters (representing keyboard keys).
  2. Press a key (not case-sensitive) to open the corresponding menu. You'll see additional underlined letters on the labels for most of the features in that menu.
  3. Press the key(s) corresponding to the option you want to access.

Example

Let's say you want to open the Options dialog from the Tools menu. First, you press and release the ALT key, which shows an underlined letter in each menu label.

Press the T key to open the Tools menu.

Press the O key.

The Options dialog opens.
