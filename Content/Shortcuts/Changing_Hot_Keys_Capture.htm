Changing Hot Keys

how to change hot keys

    • Ribbon Select File>Options.

    • Tool Strip Select Tools>Options.

    The Options dialog opens.

  2. Click the Hot Keys tab.

  3. In the Action section, select the option whose hot keys you want to change.
  4. In the Key Assignment section, select the hot key that you want to assign to the option.
  5. (Optional) In the Modifier Key Assignment section, you can select the Control Key, Shift Key, and/or Alt Key check boxes if you want to use a combination for the shortcut. For example, if you select D from the list to the right and add check marks to the Shift Key and Alt Key boxes, the new shortcut for the option will be SHIFT+ALT+D.
  6. (Optional) In the Options section, you can return to the factory default keyboard shortcuts by clicking Reset to Factory Defaults.
  7. (Optional) In the Options section, you can turn off the hot key feature by checking the Disable Hot Keys check box.
  8. Click OK.
