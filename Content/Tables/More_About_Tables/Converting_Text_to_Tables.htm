Converting Text to Tables

If you have a series of paragraphs or some text separated by commas or other strings, you can quickly place all of it into a table.

how to convert text to a table
  2. Drag and select the text to be converted, so that it is highlighted.
  3. In the Table section click the face of the Insert Table button (not the down arrow). The Insert Table dialog opens.

  4. In the Text to Table section, select one of the options.

    You can also complete any of the other fields in the dialog as necessary. For more information about the other settings, see Inserting Tables into Topics.
