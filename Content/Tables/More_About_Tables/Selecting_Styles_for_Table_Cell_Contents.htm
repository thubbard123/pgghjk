Selecting Styles for Table Cell Contents

There are a couple of different ways to use styles to affect the look of tables that you insert. First, you can use standard table styles (e.g., table, td, th, tr). You can modify these styles in your regular stylesheet. A second option is to use the custom table stylesheets that lets you create. These table stylesheets let you create patterns and different looks for tables. For more information about both kinds of stylesheets, see About Styles and Stylesheets.
