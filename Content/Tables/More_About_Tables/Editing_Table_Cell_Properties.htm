Editing Table Cell Properties

After a table is inserted into a , you can easily change the properties for cells within that table. This includes changing the borders, text alignment, hyphenation, background, and more.

    padding
    1. Select the Borders tab.

    2. Set the options in the Padding section.

    text alignment
    1. Select the Text tab.
    2. To set the horizontal alignment, in the Text Alignment section, select one of the options.
      • Left Read more… Aligns the text on the left side of the cell.
      • Center Read more… Aligns the text on the center of the cell.
      • Right Read more… Aligns the text on the right side of the cell.
      • Justify Read more… Aligns the text so that it is flush with both the left and right side.
    3. To set the vertical alignment, in the Alignment section, select one of the options.
      • Top Read more… The text aligns at the top of each cell.
      • Middle Read more… The text aligns in the middle of each cell.
      • Bottom Read more… The text aligns at the bottom of each cell.
    text indentation
    1. Select the Text tab.
    2. Use the First line indent section.
    text line height
    1. Select the Text tab.
    2. In the Line height section, click the down arrow in the upper-right corner and select Length.

    3. This is the amount of space that will be added between lines.

    hyphenation
    1. Select the Hyphenation tab.
    2. Use the tab to specify the hyphenation settings.
    background
    1. Select the Background tab.
    2. Use the tab to specify the settings that you want for the background.
