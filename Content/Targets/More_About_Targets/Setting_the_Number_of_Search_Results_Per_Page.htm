Setting the Number of Search Results Per Page

Search Results per Page

For more information, see Optimizing Your Site.

how to Change the Number of Search Results per Page
  1. Open a target.
  2. Click the Advanced tab of the Target Editor.

  3. In the Results Per Page field, use the arrows to set the number of results you want to display on each page. By default, the number of results per page is 20.

 What's Next?

After you set the number of search results per page, you can format the way search pagination appears in the skin. See Search Results in HTML5 Skins.
