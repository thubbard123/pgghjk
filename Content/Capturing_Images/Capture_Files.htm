Capture Files

When you use Capture to create images, two types of files are at work: image files and property files.

Image files

image files

An image file is the main file created when you capture and save an image. You can create and open images using any of the following file types: .

Property files

property files

In addition to the main image file, Capture creates an XML-based file with a .props extension. This file holds all of the property information (including object layer information). Therefore, the objects are not merged into the image, which means that you can easily change the object information later without having to start over from scratch. It also means that the XML files can be opened and modified with third-party tools. This is especially useful if you need to localize the callouts in your images (e.g., have them translated into other languages).

If you add a Capture image into a MadCap Flare project, the PROPS file is included in the project as well (although it is hidden).
