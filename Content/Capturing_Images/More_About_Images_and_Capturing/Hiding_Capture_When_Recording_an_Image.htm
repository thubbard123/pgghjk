Hiding Capture When Recording an Image

You can choose to hide the Capture application automatically when you initiate a capture method. Sometimes, this feature may not be relevant (e.g., when you perform an Active Window capture using shortcut keys). However, if you use the menu option when performing, say, a Region capture, you will want Capture to minimize when you do this so that you can easily record whatever application is behind Capture. You will probably want to leave this feature turned on most (if not all) of the time.

how to hide capture when recording an image

    • Ribbon Select File>Options.

    • Tool Strip Select Tools>Options.

    The Options dialog opens.

  2. Select the Capture tab.
  3. Select Hide on Capture.
