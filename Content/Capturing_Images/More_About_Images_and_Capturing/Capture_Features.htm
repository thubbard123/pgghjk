Capture Features

See the following for more information about features in Capture:

  • Hiding Capture You can choose to hide the Capture application automatically when you initiate a capture method. See Hiding Capture When Recording an Image.
  • Magnifier You can enable or disable the magnifier that appears when you capture new images (using the Region method). See Enabling the Magnifier.
  • Sound You can choose to hear recording sounds when you capture images, or you can turn this feature off if you do not want to listen to the sounds. See Enabling Sound When Capturing an Image.
  • Time Delay You can enable the time delay feature to be used when capturing images. With this feature, images are captured after the number of seconds that you specify. See Enabling the Time Delay Feature.
