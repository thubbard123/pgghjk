Exporting Analyzer Data to a CSV File

how to export analyzer data to a csv file
  1. Open the window whose data you want to export to a CSV file.
  2. If you want to export only some of the data, select the rows that you want to include.

    • Local Toolbar To export all data, click . To export only selected rows, click .

      OR

    • Right-click Right-click in the window pane and from the context menu, select either Export to CSV or Export selection to CSV.

    The Save As dialog opens.

  4. In the Save As dialog, select a destination and file name for the CSV file. The "Save as type" field will be pre-populated with the CSV file type.
  5. Click Save.
