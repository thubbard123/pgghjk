Exporting File Data to a CSV File

You can export data from the File List window pane and TOC Editor to a comma-separated value (CSV) file. This can be quite useful since there are several columns of data in the window pane. CSV files are typically used to transfer large amounts of spreadsheet or database information from one program to another.

how to export file data to a csv file
  1. Open the File List window pane or the TOC Editor.

  2. If you opened the TOC Editor, make sure Grid View is shown, as opposed to Tree View. Otherwise, the export buttons will not be enabled.

  3. If you want to export only some of the data, select the rows that you want to include.

    • Local Toolbar To export all data, click . To export only selected rows, click .

      OR

    • Right-Click Right-click in the window pane and from the context menu, select either Export to CSV or Export selection to CSV.

    The Save As dialog opens.

  5. In the Save As dialog, select a destination and file name for the CSV file. The "Save as type" field will be pre-populated with the CSV file type.
  6. Click Save.
