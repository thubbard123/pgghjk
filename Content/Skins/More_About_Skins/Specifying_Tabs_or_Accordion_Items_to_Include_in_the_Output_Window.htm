Specifying Navigation Elements to Include in the Output Window

how to specify tabs or accordion items to include in the output window
  1. Open a skin.
  2. Select the General tab.
  3. In the Features section, click the check box (adding a check mark) next to each navigation element that you want to include in the output. The elements available are different, depending on the type of skin you've opened—Standard, HTML5, or Mobile.
