Skin Tasks for HTML Help

There are various skin-related tasks unique to output that you can perform.

Adding a Web Toolbar to HTML Help

See Specifying Web Toolbar Settings—HTML Help Output.

how to add a web toolbar to html help output
  1. Open a Standard skin.
  2. Select the HTML Help Setup tab.
  3. Click Display toolbar in each topic.
 notes

As an alternative, you can add a web toolbar with the help of a topic toolbar proxy. This allows you to add a toolbar in specific places within topics, as well as limit the topics that have the toolbar. See Inserting Topic Toolbar Proxies.
