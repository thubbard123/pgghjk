Printing Images

You can print images that you have captured.

how to print an image
  1. Capture or open the image.

    • Ribbon Select File>Print.

    • Tool Strip Select File>Print.
    • Keyboard Shortcut Press CTRL+P.
  3. Complete the options in the Print dialog.
  4. Click OK.
 note

You can also specify the page setup for printing (e.g., paper, orientation, margins).
