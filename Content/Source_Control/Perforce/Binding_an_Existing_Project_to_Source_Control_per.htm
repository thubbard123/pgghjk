 Automatically Detecting Existing Project Bindings

You can scan a project when you open it to see if it has any bindings to source control. If a binding is detected, you can apply the binding and submit the project to source control.

 notes

You can also bind a new project to source control while creating it. See Creating a New Project.See Creating New Projects.
