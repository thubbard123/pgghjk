Importing from Microsoft SourceSafe

This topic chapter section focuses on importing a project from source control. You might use this method, for example, if you are working on a multi-author project and another member of the team has placed the project in Microsoft Visual SourceSafe. For more information see About Source Control.

Following are the steps for importing a project from Visual SourceSafe.

how to import a project from source control

    • Ribbon Select File>New Project>Import Project.

    • Source Control Explorer From the View ribbon or View menu, open the Source Control Explorer. From the drop-down, select the Home pane. Click Import Project.

    The Import Project from Source Control Wizard dialog opens.

  2. From the drop-down, select Microsoft Visual SourceSafe.
  3. Next to the Database field, click .
  4. Find the appropriate SourceSafe database and double-click the INI file.
  5. Click Next.

  6. Next to the Project file field, click Browse. The Browse Source Control Files dialog opens.

  7. Find and click on the Flare project file (FLPRJ) that you want to import. (You may need to log in with your user name and password.)
  9. Click Next.
  10. In the Project name field, the name of the project being imported is displayed. It is recommended that you leave the name as it is, especially if you are working with other authors on the project. However, you can enter a different project name if you want.
  11. In the Project folder field, either accept the default location for the new project or click to browse for and select a folder.
  12. Click Finish. The project is imported and loaded into .
