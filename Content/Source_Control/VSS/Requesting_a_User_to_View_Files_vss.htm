Requesting a User to View Files—Visual SourceSafe

note

You cannot use this feature if you have disabled instant messaging. See Enabling and Disabling Instant Messaging.

 how to request a user to view source control files

  1. In one of the window panes (e.g., Content Explorer, File List, Project Organizer, Pending Changes window pane), select the relevant file(s).

    OR

    Open a file.

    • Ribbon In the Instant Messaging section select Request View File.

    • Right-Click If you have the Content Explorer, Project Organizer, Pending Changes window pane, or File List open, right-click the file and select Source Control>Request View File.

    The View File Request dialog opens.

  3. Select Send Instant Message.

  4. (Optional) Enter a message to the other user.

    If you leave the Message field blank, sends a default message.

  6. Click Send. A popup indicates that the instant message has been sent to the other user.

When other users receive your message, they can open the specified file by clicking a button in the message prompt.
