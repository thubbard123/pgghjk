Importing from Git

This topic chapter section focuses on importing a project from source control. You might use this method, for example, if you are working on a multi-author project and another member of the team has placed the project in Git. For more information see About Source Control.

This topic focuses on importing a project from source control. You might use this method, for example, if you are working on a project with several other translators and another member of the team has placed the project in Git. For more information see About Source Control.

Following are the steps for importing a project from Git.

how to import a project from source control

    • Ribbon Select File>New Project>Import Project.

    • Source Control Explorer From the View ribbon, open the Source Control Explorer. From the drop-down, select the Home pane. Click Import Project.

    The Import Project from Source Control Wizard dialog opens.

  3. In the Remote drop-down, enter the remote repository where the project is located.
