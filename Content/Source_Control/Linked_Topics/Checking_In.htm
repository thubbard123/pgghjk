Checking In, Committing, and Submitting Source Control Files

You can check in, submit, and commit source control files in the following ways:

  • Apache Subversion Committing Source Control Files
  • Git Committing Source Control Files
  • MadCap Central Pushing Content to Central
  • Microsoft Team Foundation Server Checking In Source Control Files
  • Microsoft Visual SourceSafe Checking In Source Control Files
  • Perforce Submitting Source Control Files
