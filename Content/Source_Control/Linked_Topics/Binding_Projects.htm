Binding a Project to Source Control

You can bind a project to source control in the following ways:

  • Apache Subversion Binding a Project to Apache Subversion
  • Git Binding a Project to Git
  • Microsoft Team Foundation Server Binding a Project to Microsoft Team Foundation Server
  • Microsoft Visual SourceSafe Binding a Project to Microsoft Visual SourceSafe
  • Microsoft Visual Studio Online Binding a Project to Microsoft Visual Studio Online Using Microsoft Team Foundation Server
  • Perforce Binding a Project to Perforce
  • Third-Party Providers Binding a Project to a Third-Party Source Control Provider
