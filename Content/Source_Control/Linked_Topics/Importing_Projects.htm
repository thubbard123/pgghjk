Importing a Project from Source Control

This chapter section focuses on importing a project from source control. You might use this method, for example, if you are working on a multi-author project and another member of the team has placed the project in source control. For more information see About Source Control.

You can import projects from source control in the following ways:

  • Apache Subversion Importing a Project from Subversion
  • Git Importing a Project from Git
  • MadCap Central Importing Projects from Central
  • Microsoft Team Foundation Server Importing a Project from Microsoft Team Foundation Server
  • Microsoft Visual SourceSafe Importing a Project from Microsoft Visual SourceSafe
  • Microsoft Visual Studio Online Importing a Project from Microsoft Visual Studio Online
  • Perforce Importing a Project from Perforce
  • Third-Party Providers Importing a Project from a Third-Party Source Control Provider
