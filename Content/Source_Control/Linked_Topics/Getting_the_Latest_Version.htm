Updating or Getting the Latest Version of Source Control Files

You can get the latest version, update, or synchronize your source control files in the following ways:

  • Apache Subversion Updating Source Control Files
  • Git Synchronizing Source Control Files
  • MadCap Central Pulling Content from Central
  • Microsoft Team Foundation Server Getting the Latest Version of Source Control Files
  • Microsoft Visual SourceSafe Getting the Latest Version of Source Control Files
  • Perforce Getting the Latest Version of Source Control Files
