Viewing Instant Messaging Users

how to view instant messaging users

    • Ribbon In the Instant Messaging section select Instant Messages.

    • Keyboard Shortcut Press CTRL+SHIFT+I.

  2. From the drop-down at the top of the window pane make sure Users is selected. Each user is listed in the window pane. You can quickly send an instant message to any users by selecting them from the list and clicking Send.
