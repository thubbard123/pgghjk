About Instant Messaging

Instant Messaging Tasks

Following are some of the tasks you can perform when it comes to instant messaging:

  • Send an Instant Message See Sending Instant Messages.
  • Reply to an Instant Message See Replying to Instant Messages.
  • Select an Instant Message Status See Selecting an Instant Messaging Status.
  • View Instant Messages See Viewing Instant Messages.
  • View Other Instant Message Users See Viewing Instant Messaging Users.

  • Request a User to Check In Source Control Files See Requesting a User to Check In Files.

  • Request a User to View Source Control Files See Requesting a User to View Files.

  • Disable If necessary, your network administrator can disable 's instant messaging feature. This can be done for , as well as any other MadCap products that use instant messaging. See Disabling Instant Messaging.

Many of the tasks listed above can be performed from the Instant Messages window pane, which can be opened by selecting Source Control>Instant Messages. However, you do not need to have this window pane open or visible to know when you have received an instant message. You don't even need to have a project open. When another user sends you an instant message, you will receive a popup on your screen.
