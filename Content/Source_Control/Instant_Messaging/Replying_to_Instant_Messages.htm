Replying to Instant Messages

how to reply to an instant message

    • Ribbon In the Instant Messaging section select Instant Messages.

    • Keyboard Shortcut Press CTRL+SHIFT+I.

  2. From the drop-down at the top of the window pane make sure Messages is selected.
  3. From the list of messages in the window pane, select the one to which you want to reply.
  4. In the local toolbar, click Reply. The Reply to Instant Message dialog opens. The original message is displayed at the bottom of the dialog.
  5. In the Message field at the top, enter the text.
  6. Click Send.
