Sending Instant Messages

 See About Instant Messaging.

how to send instant messages

    • Ribbon In the Instant Messaging section select Instant Messages.

    • Keyboard Shortcut Press CTRL+SHIFT+I.

  2. In the local toolbar, click Send. The Send Instant Message dialog opens.
  3. In the Message field at the top, enter the text.
  4. You can also use the "Send All" button to send the message to all users in the list.
  5. Click Send if you have selected specific individuals, or click Send All to send the message to everyone.
