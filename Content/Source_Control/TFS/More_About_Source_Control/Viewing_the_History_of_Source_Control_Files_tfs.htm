Viewing the History of Source Control Files—Team Foundation Server

One of the benefits of 's integrated source control is that you can view the history for a particular file, including who checked in the file and when it was checked in. You can also view differences between different versions of the file and roll back to an older version if necessary. For more information see About Microsoft Team Foundation Server, Viewing Differences in Source Control Files, and Rolling Back to an Earlier Version of a File.
