Unbinding a Project from Microsoft Team Foundation Server

Use the following steps if you have previously connected your project to Microsoft Team Foundation Server but now wish to unbind it (i.e., remove the connection).

how to unbind a project from source control
  1. Open the project.
  2. In the Properties section select Project Properties. The Project Properties dialog opens.
  3. Select File>Project Properties.
  5. Click Unbind Project.
how to unbind a project from source control—Source Control Explorer
  1. Open the project.
  2. In the Explorer section select Source Control Explorer. The Source Control Explorer opens.
  3. In the Pane section select Source Control Explorer. The Source Control Explorer opens.

  5. Click Unbind. A confirmation dialog appears.
  6. Click Yes.
