Binding a Project to Microsoft Team Foundation Server

Use the following steps if you have already created a project and want to bind ("connect") it to Microsoft Team Foundation Server. You can also automatically detect existing source control bindings if your project has been previously connected to TFS.

how to bind a project to source control

 Automatically Detecting Existing Project Bindings

You can scan a project when you open it to see if it has any bindings to source control. If a binding is detected, you can apply the binding and check the project in to source control.

 notes

You can also bind a new project to source control while creating it. See Creating a New Project.See Creating New Projects.

If you want to publish your output to source control, you must create a bind destination. See Creating Publishing Destinations.
