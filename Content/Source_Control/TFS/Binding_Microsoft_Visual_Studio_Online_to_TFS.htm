Binding a Project to Microsoft Visual Studio Online Using Microsoft Team Foundation Server

includes Microsoft Team Foundation Server support for Microsoft Visual Studio Online. If you manage your source control projects with Microsoft's online version of Visual Studio, you can bind your projects to and perform check-ins and check-outs of files.

Use the following steps if you have already created a project and want to bind ("connect") it to the Microsoft Visual Studio Online repository using Microsoft Team Foundation Server.

 What's Next?

After you bind a Microsoft Visual Studio Online project using Microsoft Team Foundation Server in , you can use the interface in the same way as you would with any other project bound to Microsoft Team Foundation Server.

You may still need to use Microsoft Visual Studio Online for some administrative tasks (in the same way as you might need to use Microsoft Visual Studio occasionally when binding a project to Team Foundation Server). For information on using Microsoft Visual Studio Online, see the Microsoft Visual Studio Online Help.
