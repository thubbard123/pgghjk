About Indexes

Index entries are also called "keywords" (or K-links). They will display in your output in an index.

Following are the main tasks you can perform when it comes to indexes:

  • Create in Project Within your Doc-To-Help project, you can create index keywords and assign topics to them. You can also create an index automatically. See Creating Index Keywords in a Project.
  • Create in Source Document As an alternative to creating an index in the project, you can create an index by inserting keywords in your Word source document. See Creating Index Keywords in a Source Document.
  • Create Links You can create links to index keywords. See Creating Links.

In addition to the tasks discussed in this chapter, you can also create links to index keywords. See Creating Links.
