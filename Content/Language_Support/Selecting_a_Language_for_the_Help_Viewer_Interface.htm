Selecting a Language for the Help Viewer Interface

how to select a language for the help viewer interface

  1. Launch the Help Viewer. The Select UI Language dialog opens.

    If you have previously selected to disable this dialog when launching the Help Viewer, you can overrule that selection by doing the following.

    1. Select Tools>Options. The Options dialog opens.
    2. Click the check box Show Select UI Language Dialog on Startup.
    4. Launch the Help Viewer again.
  2. Select the language that you want to move in the list. (The language displayed highest in the list will be the first choice in displaying the interface. If that language is not available for a particular element, the next language in the list is used, and so on.)
  3. Click the Up or Down button until that language is in the proper location in the list.
  4. Repeat steps 2 and 3 until the list displays the languages in the order that you want.
  5. (Optional) If you do not want the dialog to display the next time you launch the Help Viewer, click Show this dialog on startup to remove the check mark.
  6. The Help Viewer is displayed according to your language selections.
