Help Controls

Flare lets you use the following kinds of Help control links:

  • Concept Links See About Concepts and Inserting Concept Links.
  • Keyword Links See Inserting Keyword Links.
  • Related Topics Links See Inserting Related Topics Links.
  • Shortcut Controls A shortcut control launches a program or window in an application by passing Windows-based messages and parameters. For example, if a topic discusses a procedure that involves a specific dialog window, you can provide a link that a user can click in the topic to open the dialog in the program. Shortcut controls can make tasks easier for new users, and they can speed up complex procedures for experienced users. See Inserting Shortcut Controls into Topics.
