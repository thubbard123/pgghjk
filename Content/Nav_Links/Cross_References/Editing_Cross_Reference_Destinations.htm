Editing Cross-Reference Destinations

After you insert cross-references into topics, you can change the file to which the cross-reference is pointing.

  3. From the Link to drop-down field, select a way to identify the topic or bookmark to which you want to link. Based on the option you choose, the section below gives you a list of selections or additional fields to complete.

  4. Use the section below to choose the heading or bookmark to which you want to link.

  5. (Optional) In the Target Frame field of the Insert Cross-Reference dialog, click the drop-down arrow to select the way the linked destination will open. This option can be used to open the destination topic or file in a popup.

  6. (Optional) In the Screen Tip field, you can type a phrase that will appear when the end user hovers over the cross-reference in the output.

    When you enter a screen tip, it is added as a <title> tag in the markup. In addition, an <alt> (alternate text) tag is added with the same text. This is useful when it comes to accessibility.
