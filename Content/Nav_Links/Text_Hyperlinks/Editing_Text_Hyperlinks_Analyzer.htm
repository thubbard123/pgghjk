Editing Text Hyperlinks

You can edit text hyperlinks by changing the destination and other properties (link text, target frame, screen tip).

how to edit the destination and properties of a text hyperlink
  2. In the XML Editor, right-click inside the hyperlink and from the context menu select Edit Hyperlink. The Insert Hyperlink dialog opens.

  3. From the Link to drop-down field, select a way to identify the topic, bookmark, or file to which you want to link. Based the option you choose, the section below gives you a list of selections or additional fields to complete.

  4. (Optional) The Link text field displays the text that you highlighted in the topic, which will be used as the hyperlink. Leave the text as it is, unless you decide you would like to change it. If you want to change the link text, type the new text in the field. It will replace the previous text in the topic.

  5. (Optional) In the Screen Tip field, you can type a phrase that will appear when the end user hovers over the hyperlink in the output.

    When you enter a screen tip, it is added as a <title> tag in the markup. In addition, an <alt> (alternate text) tag is added with the same text. This is useful when it comes to accessibility.

  6. (Optional) Next to the Style Class field, click the Select button. This opens the Select Class dialog, which lets you apply one of the defined hyperlink styles (a.NameOfStyleClass) from your stylesheet to the link. After you select a style class in the dialog, click OK. The Style Class field displays the selected style. (If you do not specify a style class, uses the parent "a" style.)
  7. (Optional) In the Target Frame field, click the drop-down arrow to select the way the linked destination will open. This option can be used to open the destination topic or file in a popup.
  8. (Optional) In the Tab Index field you can enter a value to specify the order in which users can tab through hyperlinks in a topic. So when a user presses the Tab key the first time in a topic, the hyperlink with an index value of 1 will be selected first, the hyperlink with a value of 2 will be selected next, and so on. If you want to skip a hyperlink, enter -1 in that field.

  9. The hyperlink is changed in the topic. By default the hyperlink is displayed in blue, underlined font. You can change the appearance of the link by modifying the style in the Stylesheet Editor.
