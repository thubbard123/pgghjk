Inserting Text Popups

Following are steps for inserting a text popup:

how to insert a text popup
  2. In the XML Editor, highlight the text that you want to use as the link (or "hotspot") to open the popup.

  3. In the Text section, select Text Popup.

    The Insert Text Popup dialog opens, with the text that you highlighted already shown in "The Hotspot Text" field.

  4. In The Popup Text field, type the content for the popup.
  5. The link for the text popup is added to the topic. You can change the appearance of the link by modifying the MadCap|popupHead and MadCap|popupBody styles in the Stylesheet Editor.
 notes

You can also create a popup that opens a topic or external file, as opposed to opening only basic text. For more information see Inserting Topic Popups into a Topics.

You can use the built-in Analyzer and reports to find broken links. See Viewing and Fixing Broken Links and About Reports.
