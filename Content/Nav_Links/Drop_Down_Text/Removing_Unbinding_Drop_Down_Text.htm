Removing/Unbinding Drop-Down Text

If you have inserted drop-down text into a topic and want to keep the text but remove (or "unbind") the drop-down effect so that it is displayed as normal text, use the following steps. If you want to delete the drop-down text, therefore removing all of its text from the topic, see Deleting Drop-Down Text.
