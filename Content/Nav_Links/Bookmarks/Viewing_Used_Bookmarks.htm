Viewing Used Bookmark Links

how to view used bookmark links
  1. Open a project.

  2.  In the Dynamic Reports section, select Used Items>Used Bookmarks.

    The Used Bookmarks window pane opens, listing all of the bookmark links used throughout the project.

    • Bookmark Displays the name of the bookmark.
    • Link Tag Displays the tag where the bookmark link is used.
    • Link Displays the link path.
    • File
    • Folder

  5. You can open any file in the list to view or modify it in an editor. You can open the file in the Analyzer interface or in the Flare project.
