how to view incoming links to a file
  1. Open a project.
  2. Open any window pane that contains a "File" column (e.g., Broken Links window pane, Variable Suggestions window pane, Style Suggestions window pane). The majority of window panes in Analyzer include this column.
  3. Click on a row in the window pane that contains the "File" column.
  4. In the Analysis section, select Browse Links.

    If any links are shown (pointing to the file that is currently active in the interface), clicking this button will change the window pane to show the incoming links for that file.

    example

    Example

    Let's say you are currently viewing the incoming links for a file called "Topic1," and one of the files in the list is "Topic2." That means the Topic2 file contains a link pointing to Topic1. Now, let's say that you click the Browse Links button and the window pane changes, showing a file called "Topic3." This means that the Topic3 file contains a link that points to Topic2.

    The Incoming Links window pane is populated with information about the links pointing to the selected file row.
