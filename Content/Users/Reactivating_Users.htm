Reactivating Users

How to Reactivate a User
  1. Click . The Users page opens.
  2. Find the user in the grid, and click in an empty space so that the entire row is highlighted.
  3. In the toolbar click . Set User Status dialog opens.
  4. From the drop-down, select Reactivate.
  5. Click Reactivate User. On the Users page, you will notice that the row showing that user is no longer grayed out, and the status is changed to "Active."
