Reinviting Users

How to Reinvite a User
  1. Click . The Users page opens.
  2. Find the user in the grid, and click in an empty space so that the entire row is highlighted.
  3. In the toolbar click . Set User Status dialog opens.
  4. From the drop-down, select Reinvite. Another email is sent to the individual. The user must click the link in the email to activate the account. Until the user activates the account, he or she is listed as "Invited" in the system.
