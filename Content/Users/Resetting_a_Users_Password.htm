Resetting a User's Password

How to reset a User’s Password
  1. Click . The Users page opens.
  2. Find the user in the grid, and click in an empty space so that the entire row is highlighted.
  3. In the toolbar click . Set User Status dialog opens.
  4. From the drop-down, select Reset Password.
  5. Click the Reset Password button. An email is sent to the user, who can click a link in the message to reset the password.
