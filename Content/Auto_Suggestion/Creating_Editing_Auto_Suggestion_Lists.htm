Creating and Editing Auto Suggestion Lists

After you add an auto suggestion list file to your project, you can add words, phrases, or sentences to it, or make changes to existing content in it. You can add phrases directly in the Auto Suggestion List Editor, or you can right-click selected text in the XML Editor and use the context menu to add it to the auto suggestions list file.

    2. Press F2 on your keyboard.

    edit an existing row

     Maybe you want to modify the text in an existing row.

    1. Click the row and press F2 on your keyboard.
    2. Edit the text as necessary.
    3. Press Enter on your keyboard.

    cut, copy, or paste a row

     You can you use the cut, copy, or paste options to move rows from one auto suggestions file to another, or simply to create new rows based on existing ones.

    1. Click on a row (if using cut or copy).
    2. In the local toolbar of the editor, click Cut, Copy , or Paste as necessary.

 how to add text to an auto suggestion list from the context menu
  1. In the XML Editor highlight text that you want to add to an auto suggestion list file (no more than 256 characters).
  2. Right-click on that highlighted text.

  3. From the context menu select Create Auto Suggestion. If you have multiple auto suggestion list files, the Create an Auto Suggestion dialog opens.

    If the option is disabled in the context menu, it is likely that you selected more than 256 characters. Select fewer words and try again.

  4. Select the auto suggestion list file you want to use and click Create. The text is added to the auto suggestion list file.

 What's Next?

After you've added text to the auto suggestions list file, these are displayed as auto suggestions when you type matching text in the Auto Suggestion popup, as long as have this feature enabled (see Enabling and Disabling Auto Suggestion). You can then insert the any suggestions you want to use. See Inserting Auto Suggestions Into Content.
