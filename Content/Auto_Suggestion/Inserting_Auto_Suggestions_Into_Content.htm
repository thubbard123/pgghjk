Inserting Auto Suggestions Into Content

When you begin typing in a , the Auto Suggestion popup opens if finds matching text in an auto suggestion list file, snippet, or variable. You can then choose a match from the popup and insert it into your topic or snippet.

how to insert an auto suggestion into content
  1. Open a .
  2. Begin typing. As you do this, the Auto Suggestion popup is displayed if a match is found.

    • Click on the matching row in the popup that you want to insert and press Enter on your keyboard. If the row is already selected when the popup first displays, you only need to press Enter.

      OR

    • Double-click the matching row in the popup that you want to insert.
 notes

If you insert a snippet, it is added as a text snippet if the paragraph in the XML Editor already contains other content. If your cursor is on a blank paragraph when you select the snippet, it is added as a block snippet, taking up all of the room in that paragraph. See About Snippets.

Variables are displayed based on the definition rather than the file name. Therefore, if you begin typing "Friday," the match is displayed in the popup, even though the name of the variable is "LongDate."
