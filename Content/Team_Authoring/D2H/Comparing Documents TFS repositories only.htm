Comparing Documents

In TFS repositories, you can compare two versions of a document.

How to Compare Documents
  1. In the Team Authoring window, select the file.
  2. Click the Compare button. The Compare dialog opens.
  3. Choose the file versions and click the OK button.
 Notes

To specify the compare and merge tools for team authoring, use the Doc-To-Help Options dialog (File>Tools>Options>Team).

For other tasks, see Working in a Team Project.
