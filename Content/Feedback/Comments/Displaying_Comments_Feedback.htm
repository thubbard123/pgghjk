Displaying Comments at the End of Topics for Feedback

how to display comments at the end of topics
  1. Open a Standard or HTML5 skin.
  2. Select the Community tab.
  3. Click Display topic comments at the end of each topic.
 Note What's Next?

Make sure you associate the skin with the target. See Associating Skins with Targets.

After you generate and publish the output, you can open Feedback Explorer to view submitted comments and see how readers are using the output. See Opening Feedback Explorer.
