Opening Feedback Explorer

how to open feedback explorer note

You can also launch MadCap Feedback Explorer from the Tools ribbon or menu. However, doing this will not necessarily ensure that the appropriate FBCONF file associated with the target that you want to view opens in the application. If not, you would need to open the FBCONF file from the Feedback Explorer interface. To use this method for opening MadCap Feedback Explorer, select Tools>MadCap>MadCap Feedback Explorer.
