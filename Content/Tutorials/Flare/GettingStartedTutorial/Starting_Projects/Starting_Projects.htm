Starting Projects

Begin by creating a new Flare project and taking a tour of the workspace.

  1. Creating a Project The Start New Project Wizard lets you create a project from a template, an existing project, or an imported project. In this tutorial, we will create a new project from a Flare template. See Creating a Project from a Template.
  2. Touring the Workspace After creating you project, you should familiarize yourself with the Flare workspace before doing too much. This tutorial covers several major areas of the workspace. See Touring the Workspace.
