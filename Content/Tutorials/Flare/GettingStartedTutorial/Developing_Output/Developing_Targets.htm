Developing Targets

After your project is styled, you are ready to start preparing your output files. You can set up different output types (e.g., PDF, HTML5, EPUB) and define settings that tell Flare what information to include in the final documents.

The following are covered in this tutorial:

  1. Targets Before you generate your final output, you need to create a target for each output type that you want to create. See Targets.
  2. Conditions You can use condition tags to specify which content should appear in each output. For example, if you write a topic that does not need to be in your printed output, you can use a condition tag to exclude it when you build your output. You can use condition tags on any element in your project, including topics, paragraphs, images, snippets, and individual sentences. See Condition Tags.
  3. Skins Skins tell Flare how your online output should look. In this tutorial, you will learn how to set a logo in your skin. However, there are dozens of different skin settings. See Skins.
  4. Tables of Contents If you are planning to create different kinds of outputs, you may want to set up different tables of contents for each one. This makes it easier to organize your topics, especially if you will be including different information or using different outlines for each of your outputs. See Table of Contents.
