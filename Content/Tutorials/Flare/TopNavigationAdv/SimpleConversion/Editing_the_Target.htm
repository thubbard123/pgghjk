Editing the Target

Now we need to tell Flare to use our new HTML5 Top Navigation skin, instead of the older Tripane skin.

Edit the HTML5 Target
  1. In the Project Organizer, expand the Targets folder.
  2. In this particular project, the target is placed inside a subfolder. So expand Online Output.

  3. Double-click HTML5 - Tripane (Primary).

    The Target Editor opens to the right.

    Since you are converting to Top Navigation output, you can rename this target if you want. However, it's not mandatory. The target will generate the correct output regardless of what you name it.

  4. Select the Skin tab.

  5. In the Skin field, select TopNav, the name of your new Top Navigation skin.
