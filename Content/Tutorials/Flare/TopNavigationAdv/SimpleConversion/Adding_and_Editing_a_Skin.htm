Adding and Editing a Skin

Our Legacy project already has a skin in it, but it is the wrong kind of skin. It is designed for Tripane output. We need to add one that works for Top Navigation.

Add and Edit a Top Navigation Skin

  1. In the Project Organizer, right-click the Skins folder, and from the context menu select Add Skin. The Add File dialog opens.

  2. Select the HTML5 - Top Navigation factory template.

  3. In the File Name field at the bottom of the dialog, type TopNav as the name of the skin.

  4. Click Add. The skin is added in the Project Organizer next to the Tripane skin, and it opens to the right in the HTML5 Skin Editor.

  5. Select the Styles tab and expand Logo>Background.

  6. Click .

    The Insert Image dialog opens.

  7. If your image already exists in the skin, you can select it from the list. If it doesn't (and it probably won't), click , then find and select it.

  8. Your logo displays in the preview of the HTML Skin Editor.
