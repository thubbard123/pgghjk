Adding Content to Topics

Adding your own content to your topics is relatively simple. But because the output is print-based, and a bit unique, there are a few things to keep in mind as you work in the XML Editor.

Layout Modes and Style Mediums

Layout Modes and Style Mediums

The way it is displayed in the XML Editor depends on the layout mode that is selected, as well as the medium. This is especially important with this template, because you really need to see the approrpriate page layout as you add content. If you are looking at a topic in Web Layout mode, you won't see this and it will be more difficult to work.

Select the Correct Layout Mode

  1. If your topic is not already shown in the XML Editor, open the Content Explorer and double-click Front2.

    Chances are pretty good that your topic will initially be displayed in Web Layout mode with the default medium, like this:

  2. Click the Layout (Web) drop-down and select Print Layout.

    The look of the topic changes to this:

    This is the Back page layout (not the one you created, but the original page layout that came with the template). It is being shown because it is the first page layout listed alphabetically in the Content Explorer.

    But what you really need is the Front2 page layout that you created.

  3. In the local toolbar click the Page Layout drop-down and select Front2.flpgl, and in the flyout menu select Normal (the name of the only page in that layout).

    The display changes, showing the correct page layout:

    Also, notice that the medium selected is print.

    This happened automatically when you switched from Web Layout mode to Print Layout mode. Flare anticipated that you might create both online and print-based output from your project, and the default medium is typically used for online output while the print medium is used for print-based output.

  4. In the Content Explorer, double-click Back2.htm.

  5. If the topic displays in the wrong mode, click the Layout (Web) drop-down and select Print Layout.

    The topic displays with the original Back.flpgl page layout. It looks correct, but lets sync it with the new page layout you created.

  6. In the local toolbar click the Page Layout drop-down and select Back2.flpgl, and in the flyout menu select Normal (the name of the only page in that layout).

Inserting Variables and Adding Text

Inserting Variables and Adding Text

In this template, we've created some variables. On the first panel of the foldout, we used them for the company name and the title. We also used them for company information (e.g., address) on the back of the foldout. Variables can be useful if you plan to use the same page layout for multiple outputs. For one target you can use the default definitions for the variables, and you can override the variable definitions for any of the other targets you might generate (see Overriding Variable Definitions in Targets). For more about variables and overriding definitions in targets, see the online Help.

Insert Variables and Text in the Front Topic

  1. In the XML Editor, click the Front2.htm topic tab to bring it to the front.

  2. In the Front2 topic, delete the heading and paragraph that were added to the topic when you created it.

  3. With you cursor at the top of the first panel in the topic, click in the local toolbar.

    The Variables dialog opens.

  4. With the General variable set selected on the left side, choose the CompanyName variable.

  5. The variable is inserted into the topic.
  6. Press Enter to create a new paragraph and click again.

  7. This time select the Title variable.

  8. The variable is inserted into the topic.
  9. Press Enter again to create a new paragraph.
  10. In the Content Explorer, double-click Front.htm, the original topic included in the template.
  11. If necessary, click the Layout (Web) drop-down in the local toolbar and select Print Layout.

  12. Rather than making you type all of the text in the new topic, we'll just copy and paste it from the original topic. Highlight all of the text, starting with the first Heading to the last paragraph.

  13. Press CTRL+C on your keyboard to copy it.
  14. Go back to your Front2.htm topic.

  15. Place your cursor on the empty paragraph, and press CTRL+V on your keyboard to paste the copied text. Notice that the styles associated with the text were also retained, so we won't need to re-apply the styles to that text.

    The topic should look something like this:

Inserting and Positioning Images

Inserting and Positioning Images

We've added all of the content into both topics, except for a couple of images. In the page layouts we set images in the background. In our topics, however, we are going to insert them directly into the content and then use absolute positioning to drag them where we want.

Insert and Position an Image in the Front Topic
  1. Open the Front2.htm topic.
  2. In the Content Explorer, expand the Resources folder, then expand Images under it.

  3. Click the Image2.png file, then drag and drop it at the very beginning of the topic in the XML Editor.

  4. In the lower-right corner of the image, click the resize icon and drag it up and to the left to reduce the size of the image until you like it.

  5. Right-click the image, and from the context menu select Text Wrap>Square.

  6. Hover over the middle of the image until the cursor changes to a cross. Then click and drag the image, dropping it between the second and third panels. You might need to resize the image again until it is positioned between the text just the way you want.

    When you are finished, the topic should look something like this:

 Insert and Postition an Image in the Back Topic
  1. Open the Back2.htm topic.

  2. In the Content Explorer, click the Image4.png file, then drag and drop it at the very beginning of the heading in the second panel. We did not drag it to the beginning of the topic this time, because this topic is using two frames in the page layout, rather than one. And we want to make sure it is inserted into the second frame.

  3. (Optional) In the lower-right corner of the image, you can click the resize icon and drag it up and to the left if you want to reduce its size. But this image is smaller, so you may not need to.

  4. Right-click the image, and from the context menu select Text Wrap>Square.

  5. Hover over the middle of the image until the cursor changes to a cross. Then click and drag the image, dropping it above the heading in the second panel until it is positioned the way you like.
