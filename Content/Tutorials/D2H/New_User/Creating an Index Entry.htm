Creating an Index Entry

You can add index keywords (also known as K-Links) right from your Word source document. You can also do this in Doc-To-Help, which will be demonstrated later.

Steps

  1. In the SanDiegoQuickTour.doc Word file, select the words Pacific Ocean (it is in the third paragraph).

  2. On the Doc-To-Help ribbon, click the Keyword button. The Keyword dialog opens.

  3. Click OK to add "Pacific Ocean" to your index.

    "Pacific Ocean" appears in your source document with a different text color, but that color change will not be visible in your final output. It is there in your source so that you can see it.

    You can add keywords and secondary keywords from this dialog. See Adding Items to the Index and Creating Groups.

  4. Open the Doc-To-Help Build panel (Doc-To-Help>Build Panel), and in the local toolbar, click the Build button. See Building and Examining a Target.

    After the build is complete, you can view the output in your browser.

    In the "All About San Diego" topic, you will see that:

    • The project name appears in the header as "San Diego" — the name we gave the project in the wizard. We will change that in the next step using the Help Targets dialog.
    • The word "attractions" is a hyperlink that opens the "San Diego Attractions" topic.
    • "America's Finest City" is a hyperlink that will display text (Stay classy, San Diego!) when clicked.
    • "Pacific Ocean" is not highlighted in the target (but it is included in the Index; click the Index button below the TOC to see it).
    • The first two bullet points (about the "The coast…" and "Inland valleys…") are properly formatted.

  5. In the Doc-To-Help Project panel, click Index and Groups. Notice that "Pacific Ocean" now appears in the list of keywords.

    You can also view the list of keywords in the Index and Groups window pane in Doc-To-Help. Additionally, you can create keywords from this pane.

    For more information, see the online Help. For more information, see Index and Groups Window Pane (Doc-To-Help).
