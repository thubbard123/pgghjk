Adding Glossary Entries

It is easy to add entries to your glossary, and you do not need to worry about alphabetizing them because Doc-To-Help will do it for you.

Steps
  1. In Doc-To-Help, in the lower-left corner, click Documents.

  2. In the Documents window pane, double-click Glossary.doc to open it.  This document contains two entries.

    It is easy to add more entries and alphabetize the list.

  3. In Word, after the first two entries in the Glossary.doc, add the following:

    Santa Ana Winds

    Strong, extremely dry down-slope winds that originate inland and affect coastal Southern California.

    Now, we need to apply styles to the new entry. These styles not only control the look of the entries, but also the behavior.

  4. Select Santa Ana Winds and set the style to Glossary Heading.

  5. Select the definition below it and set the style to C1H Popup Topic Text.

    It should look like this:

    Fortunately, our glossary terms are already in alphabetical order. If they were not, you could fix that. To do this, select the Doc-To-Help ribbon, and in the Glossary group click the Sort button. The glossary is alphabetized.

  6. Save your document and build the target (see Building and Examining a Target).

    "Santa Ana Winds" is now part of the glossary; it is also in the list of topics in the Topics window in Doc-To-Help.
