Creating a Topic Link

You can easily create links to other topics, as well as bookmarks, link tags, keywords, groups, and glossary entries.

Steps

  1. In the SanDiegoQuickTour.doc Word file, scroll up to the first paragraph, and select the word attractions.

  2. On the Doc-To-Help ribbon, click the Link button. The Link dialog opens.

    Take a quick look at the Link dialog. You can also use this dialog to create links to keywords (i.e., index entries), groups, bookmarks, link tags, and glossary entries. From the drop-down list, click the Link type. Notice in the Options area, you can select the "Show as button" check box. This changes your link text to a button. Selecting the "Popup" check box in the Link options area makes the topic display in a pop-up window. By the way, linking will work in the PDF version of your manual target, as well as in your online outputs.

    You can select a Grid or Tree view of the "link to" topics. Click the button to the right to choose.

  3. For the Link type, make sure Topic title is selected.

  4. In the Topics area of the Link dialog, expand SanDiegoQuickTour.doc (if necessary) and from the list choose San Diego Attractions.

    Remember that does not create your topics until you first build the output. We already had you generate output in a previous step (see Building and Examining a Target); therefore, the San Diego Attractions topic was available for you to select.

  5. Click OK. This creates a topic link from the word "attractions" to the topic about San Diego attractions in the online output when it is built.

    If you have hidden text turned on in your document, you will see the information for this link tag displayed. As you can see, the tag is the topic you selected (San Diego Attractions).

    If you would like to hide hidden text, in versions of Word 2007 and later, choose File>Options, and on the left, click Display. Clear the Hidden Text check box on the right.

 Tip

When working in Word, you can create links to documents, topics, variables, index keywords, and groups by simply dragging and dropping items from the Doc-To-Help Project panel to the location where you want to create a link.

For more information, see the online Help. For more information, see Drag-and-Drop Linking.
