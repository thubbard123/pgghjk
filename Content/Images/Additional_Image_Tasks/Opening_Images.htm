 notes

You can also right-click on an image where it has been inserted and select Open Link from the context menu.

If you have inserted a PDF file as an image, the actual PDF file opens. This is different from other image formats, where the image opens by itself in the Image Viewer.

If you have imported a project that already contained images, keeps the images in the file structure of the source project (as opposed to placing the image files in the Resources folder).
