Using Image Groups

Image groups are available for pinned images and recently used images. You can access image groups from both the Insert Image dialog and the Insert Image drop-down in the local toolbar of the XML Editor.

example

Example

Let's say that you have pinned three images. You have also recently used several other images. In the Insert Image dialog, you will see the Pinned Images and Recently Used Images groups.

In the Insert Image drop-down in the local toolbar of the XML Editor, you will see the same groups with the same images in each group.
