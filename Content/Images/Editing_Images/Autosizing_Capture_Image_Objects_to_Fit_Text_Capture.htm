Auto-sizing Capture Image Objects to Fit Text

If you have inserted MadCap Capture images that contain objects with text, you can auto-size those objects automatically when the output is generated. This can be done by selecting an option in the Advanced tab of the Target Editor in MadCap Flare. The original image file and its associated properties (.props) file remain unchanged. Only the output image is affected.

example how to auto-size capture image objects to fit text
  1. Open a target.

  2. On the Advanced tab of the Target Editor, select Auto-size Capture objects to fit text.
