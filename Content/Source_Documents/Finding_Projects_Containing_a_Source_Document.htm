Finding Projects Containing a Source Document

How to Find Projects Containing a Source Document
  1. In Word, select the Doc-To-Help ribbon.
  2. In the Project section, click Find Project.
  3. In the dialog that opens, navigate to the Doc-To-Help project that contains your source document, then click Open.
 Notes

If you attempt to open a project that does not contain the open source document, you will receive an error.

You can also use the Find Project option if you open a source document and the Doc-To-Help Word add-in cannot locate the project file. In this instance, the Word add-in will open in Basic mode, and you can use the Find Project option to find a project associated with the source document. If you find an appropriate project, the add-in will switch to Full mode. See Ribbons.
