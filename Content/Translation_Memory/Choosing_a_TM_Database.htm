Choosing a Translation Memory Database

See About Translation Memory.

how to choose a tm database
  1. Make sure you have created at least one TM database. See Creating a Translation Memory Database.
  2. In the Translation Memory section, select Choose TM.

    The Manage Translation Memory dialog opens.

  3. In the Translation Memory section, select Choose TM. In the Manage Translation Memory dialog, click Add.

  4. Next to each TM in the grid that you want to associate with the project, click Use. Use the dialog to do any of the following:

    You can also use the dialog to do any of the following:

    add/remove tm databases to/from grid create new tm databases Rank Translation Memory Databases edit tm database

    You can open any TM database in the grid to edit it in the Translation Memory Editor. To do this, select the row in the grid and click Edit TM. See Editing the Translation Memory Database.

         delete tm database

    You can delete a TM database by selecting the row and clicking Delete. This not only removes the TM from the grid, but it deletes it entirely.

         import tmx files Export tmx files

    If you have a termbase file in Lingo, you can export it into a Translation Memory eXchange (TMX) file to share with others or to save for later. To do this, select the database in the grid, then click Export. Complete the fields in the dialog and click Start Export. See Exporting TMX Files.

         specify access—read & write versus read only
note

You can also choose a TM when creating a new project. See Creating New Projects.

 What's Next?

After choosing a TM, you can use it to translate your project. See Applying TM Suggestions to Segments.

After you align segments or files, you should upload them to your translation memory database. If you select specific segments and click in the local toolbar, only those translations will be uploaded. If you click in the local toolbar, all translations in the file will be uploaded. See Aligning Segments and Aligning Files.
