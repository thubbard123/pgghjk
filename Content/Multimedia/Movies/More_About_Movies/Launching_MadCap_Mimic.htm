Launching MadCap Mimic

Mimic is MadCap's movie-making and editing tool, which is tightly integrated with Flare. If you have Mimic installed on your computer, you can use it to do any of the following from within Flare.

  • Insert movie links into topics If you have used Mimic to produce a movie, you can use this feature to create a link in a topic to the movie's output. See Inserting Mimic Movie Links into Topics.
  • Insert movie links into TOCs If you have used Mimic to produce a movie, you can use this feature to create a link in a table of contents entry to the movie's output. See Inserting Mimic Movie Links into TOCs.
  • Insert movie links into browse sequences If you have used Mimic to produce a movie, you can use this feature to create a link in a browse sequence entry to the movie's output. See Inserting Mimic Movie Links into Browse Sequences.
  • Edit movies You can open a movie linked in Flare. The movie opens in Mimic, where you can make changes to it. See Editing Movies.
  • Launch MadCap Mimic You can simply open the Mimic application from Flare.
