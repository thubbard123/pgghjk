Inserting Silent Areas into an Audio File

After you record an audio file, you can edit the file. One way to do this is to insert an area of silence into the audio file.

how to insert a silent area into an audio file
  1. Open the movie.

  2. Do one of the following, depending on the element for which you have recorded audio:

    • Movie For an entire movie, select Movie>Properties, then in the Movie Properties dialog, select the Audio tab. The recorded audio file should be displayed in the Audio field.
    • Frame For a single frame, double-click the frame, then in the Frame Properties window pane, expand the Frame Sound section. The recorded audio file should be displayed in the Sound File field.
    • Object For an object, double-click the object, then in the Object Properties window pane, expand the Audio section. The recorded audio file should be displayed in the Sound File field.
  3. Click Edit. The Sound Editor opens.
  4. (Optional) There are multiple ways to insert a silent area. If you want to insert the silence at the beginning or end of the audio file, skip this step. If you want to insert the silence at a specific location or replace a portion of the audio file, do one of the following.

    • Click once in the display to show the playhead (the red vertical line that displays when you click in the Sound Editor). When you insert the silence, it will be placed immediately after the playhead.

    • Click and drag to select an area. When you insert the silence, it will replace the selected area.
  5. Click . The Insert Silence dialog opens.
  6. Complete the options in the dialog as necessary.
    • Duration Enter the number of seconds of silence that you want to insert. If you have already selected a specific portion of the audio file, this field is completed for you.
    • Position Select the location where you want to insert the silence.
      • Current selection Choose this option if you have first selected an area in the display to replace with silence.
      • Playhead position Choose this option if you have first clicked inside the display, thereby inserting the playhead (the red vertical line). The silence will be inserted immediately after the playhead.
      • Start of audio Choose this option if you want to insert the silence at the very beginning of the audio file.
      • End of audio Choose this option if you want to insert the silence at the very end of the audio file.
  7. In the Insert Silence dialog, click OK.
  8. In the Sound Editor, click OK.
  9. If using the Movie Properties dialog, click OK. The Object and Frame Properties window panes require no further action.
