Looping Audio

Use the following steps if you want the audio file to restart when it reaches the end of the playing time. This is useful, for example, if you want to play background music for the entire duration of a frame.

You can loop audio for a single frame and cursors.

how to loop audio for a frame
  1. Open the frame.
  2. Double-click the frame.
  3. Expand the Frame Sound section.
  4. Click Enable Audio.
  5. Use the options in this section to add audio. See Adding Audio to Frames.
  6. Click the check box next to Loop Sound.
 how to loop audio for a Cursor
  1. Open the frame.
  2. Double-click the cursor.
  3. Expand the Audio section.
  4. Click Enable Audio.
  5. Use the options in this section to add audio. See Adding Audio to Objects.
  6. Click the check box next to Loop Sound.
