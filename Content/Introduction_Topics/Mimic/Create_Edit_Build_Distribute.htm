Creating, Editing, Building, and Distributing Movies

Many of Mimic's features help you build your movie from start to finish.

  • (Optional) Create a Movie Collection Mimic includes several features to create a collection of related movies, such as templates and commenting. See Creating a Movie Collection.
  • Create a Movie Mimic includes many features to create standalone movies, such as tools to record your actions on screen, create a movie with a blank frame, base a new movie on a still image that you capture, create a movie from a media file, or import a Microsoft PowerPoint presentation. See Recording Movies, Creating Blank Movies, Creating Movies from Media Files, Creating Movies from Screen Captures, and Importing PowerPoint Presentations.
  • Edit a Movie Mimic has a number of movie editing features, including shapes, buttons, text, actions, effects, and audio. See Editing a Movie.
  • Build Movie Output After you create your movie, you can generate output using and . See About Movie Output Formats and Building Movie Output Using the Interface.
  • Distribute Output After your movie is generated, you can use Mimic's distribution features to let users access your movie output. This includes a feature for automatically publishing (i.e., copying) output files to a location you choose, as well as the option of uploading videos to YouTube or Vimeo. See Distributing Output.
