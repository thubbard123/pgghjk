Viewing Builds Activity

How to View Builds Activity
  2. On the right side of the page, click Builds Activity. The All Builds Activity dialog opens.

  3. (Optional) You can select a build row (do not click on the blue link) and use the buttons in the toolbar to take a number of actions:

    Button

    Description

    Removes all failed builds.

    Opens the output so you can view it. See Viewing Output.

    Opens the Build Log dialog so that you can view messages, warnings, and errors associated with the build. See Opening the Build Log.

    Downloads the output to your computer. See Downloading Builds.

    Sets the selected build as “Keep.” See Setting Builds to Live or Keep.

    Sets the selected build as “Live.” See Setting Builds to Live or Keep.

    Cancels the build if it is in progress.

    Rebuilds the target.

    Lets you delete a build, which frees space on the system. See Deleting Users.

    You cannot delete builds that are set to “Live” or “Keep.”

  4. Click Finish.
