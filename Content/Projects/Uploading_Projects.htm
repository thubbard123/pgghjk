Uploading (Binding) Projects

How to Upload (Bind) a Project to Central What's Next?

After you upload a project, you can open it. From here, you can do any of the following:

  • Add Widgets to Dashboard See About Widgets and Adding Widgets to a Dashboard.
  • Generate and Manage Output See About Builds.
  • View Repository This shows the actual files that make up the project. See Viewing a Project Repository.
  • View Activity for All Builds See Viewing Builds Activity.

Also, as you continue to make changes to your Flare project locally, you will need to push those changes to Central periodically to make sure the local and cloud files are in sync. See Pushing Content.
