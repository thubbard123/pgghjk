Editing Variables

After you create a variable, you can easily edit it in the File Properties dialog. If you change the definition for a variable that has already been inserted into objects, the changes will automatically be reflected wherever that variable has been inserted.

how to edit a variable
  1. Open the image.
  2. Double-click the image. The File Properties dialog opens.
  3. Select the Variables tab.

  4. Click in a cell that you want to edit. Then press F2 on your keyboard and type the name, definition, or comment.

    Adding a comment is optional and for your internal use only.

  5. Click OK.
 note

If you have used a linked variable from a MadCap Flare project and want to edit that variable, you need to do so from within the Flare project.
