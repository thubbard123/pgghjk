Exporting Termbase Files

If you have created a termbase in , you can export it to a TermBase eXchange (TBX) file.

how to Export TBX files—recommended

  1. In the TBX section, select Export Termbase.

    The Export to TBX File dialog opens.

  2. From the Termbase to export drop-down, select the termbase you want to export to a TBX file.
  3. Next to the TBX File field, click . The Choose file to export to dialog opens, where you can locate an existing TBX file you want to overwrite, or enter a name for a new TBX file.
  4. Click Save.
  5. Click Start Export.
 how to Export TBX files—Alternate

  1. In the Termbase section, select Choose Termbase.

    The Manage Termbases dialog opens.

  2. Click Export. The Export to TBX dialog opens.
  3. From the Termbase to export drop-down, select the termbase you want to export to a TBX file.
  4. Next to the TBX File field, click . The Choose file to export to dialog opens, where you can locate an existing TBX file you want to overwrite, or enter a name for a new TBX file.
  5. Click Save.
  6. Click Start Export.
