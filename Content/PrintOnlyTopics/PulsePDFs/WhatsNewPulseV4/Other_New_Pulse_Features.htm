Other New Pulse Features

In addition to the features already discussed, other new features are being introduced in this version.

Support for .NET 4.5, Windows Server 2012, and SQL Server 2012

Pulse now supports .NET 4.5. As a result, you can install Pulse on computers running Windows Server 2012. In addition, you can now use SQL Server 2012 when installing Pulse. For more information, see the online Help or the Installation Guide for Windows Server 2012.

Refresh Reports

Save/View Log in Configuration Wizard

In the Pulse Configuration Wizard, options have been added to view and save the log file. For more information, see the online Help or any of the Installation Guides.
