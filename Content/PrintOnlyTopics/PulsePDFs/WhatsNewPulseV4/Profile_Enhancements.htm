Profile Enhancements

Skills and Expertise

Registered users can now select skills and specify their expertise in different areas. Administrators set up the skills and categories. Users can send requests for skills that are not yet available in the interface.

Experience, Education, and Certification

Registered users can enter information about their experience, education, and certification.

New Notification Options

With the introduction of some new features (skills, moderation), a few new email notification options are available on the Notifications tab of the Profile page.

Administrators Can Adjust Profile Settings

Administrators can adjust profile settings, including which fields and elements are available and editable.

Pulse users can self-declare any of the skills you input. A great place to start your skills list is LinkedIn and other industry-specific websites. Each skill can be assigned to a skill category. Therefore, you may want to create a skill category first. Skill categories are used in Pulse to make locating and grouping skills easy. Some good skill categories are things like Human Resources, Legal, or Information Technology. Think about people’s job roles and departments for ideas on skill categories.

To edit profile settings, log into the dashboard and select Administration>Settings>Profiles.

From the Profile Settings page, you can do any of the following.

Skill Categories—Add, Edit, Remove Skills—Add, Edit, Remove Skill Requests—Approve or Reject Profile Properties (Fields)—Add, Edit, Remove Profile Features (Experience, Education, Certifications)—Enable or Disable
