Summary of New Features

Following are the new features introduced in this version of .

Dynamic Add-In Modes
  • The Doc-To-Help Word add-in dynamically adjusts the Doc-To-Help ribbon depending on the type of document you opened
  • No Features mode is used for templates
  • Basic mode is used for documents not in a D2H project or final output
  • Full mode is used for documents in a D2H project

See Dynamic Add-In Modes.

New Word Panels
  • Three new panels in the Word interface give you access to Doc-To-Help's most frequently used features
  • Doc-To-Help Build panel allows you to build, rebuild, view, and cancel builds
  • Doc-To-Help Progress panel allows you to see current build status
  • Doc-To-Help Project panel features four window panes (Documents, Contents, Index and Groups, Variables) from which you can perform common Doc-To-Help tasks

See New Word Panels.

Template Enhancements
  • Macro-enabled DOTM templates have been replaced with macro-free DOTX templates
  • Policies & Procedures templates now available in 8.5" x 11" and A4 formats, based on both the Fancy and No Margin templates
  • Source templates can now be customized so that any text in the source template is used when creating a new document

See Template Enhancements.

New Doc-To-Help Options Dialog Settings
  • Two new settings in the Doc-To-Help Options dialog
  • Option to control whether or not you will see the Doc-To-Help Progress Panel in Word
  • Option to prompt you if the Word add-in is disabled

See New Doc-To-Help Options Dialog Settings.

New Themes and Stylesheets
  • The new Accordion Logo, Tabs Logo, and Responsive Logo themes offer a built-in default logo
  • Two new stylesheets are available: jquerymobile\green for the Responsive and Responsive Logo themes; jquery-ui\green-sheen for the Accordion, Accordion Logo, Tabs, and Tabs Logo themes
  • All new projects now use the Responsive theme

See New Themes and Stylesheets.
