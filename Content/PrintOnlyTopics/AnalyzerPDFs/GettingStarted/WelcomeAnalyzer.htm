Introduction

Welcome to MadCap Analyzer. This application lets you open a Flare project to view important information about it and take action when necessary. Analyzer provides you with a wide variety of information—including general statistics, critical issues (e.g., broken links, undefined variables), and suggestions for improving the project (e.g., snippet and new style suggestions).

basic steps

Basic Steps

Using Analyzer is quite simple.

  1. Open Project In Analyzer, open a Flare project that you have created. See Opening Projects.

  2. View Current Issues See Viewing Current Issues.

    Here is an example of current issues listed in the Analyzer Summary window pane.

  3. View Information/Take Action Any action that you take is applied to the file in the Flare project, even if you are performing the task in the Analyzer interface. There are many different window panes that you can open in the interface (either from the list of current issues or apart from it) in order to see a variety of information about your project. See Key Features.Most of the rest of this guide explains the different types of information that you can view and work with in Analyzer.

    For details and steps on the various types of information you can view and tasks you can perform, see the online Help or the relevant PDF guide. See PDF Library.

    Analyzer also lets you create and print custom reports based on the information captured in your project. See About Reports.
