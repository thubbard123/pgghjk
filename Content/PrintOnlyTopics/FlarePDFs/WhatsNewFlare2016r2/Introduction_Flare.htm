Introduction

You will probably notice that this latest version of Flare has a brand new name. The most recent version of Flare is called “2016.” In fact, you might also notice an r2 at the end of the name, which is simply a release number. But don’t worry. Even though the name as changed, it’s still the same Flare you know and love.

Following are the new features introduced in this release of .

Feature

What's New?

MadCap Central Integration

  • Upload Flare projects to cloud-based platform
  • Plan, track, and manage processes, content, and teams
  • Store projects in Central and work on them locally in Flare
  • Push project changes to Central

Pinning Images

  • Make your favorite images easily and quickly accessible for future use
  • Inserted images are added to a group of your 20 most Recently Used Images

Internal Text Editor—Manage Font Size

  • Decrease, increase, or reset font size in editor (not in actual file or output)
  • Easier to see text as you work

Previous/Next Buttons in Top Navigation Output

  • Add Previous and Next buttons
  • Supported in HTML5 Top Navigation and skinless output
  • Lets users quickly navigate to the previous or next topic in the TOC

Targets—Pre/Post Build Processes

  • Run command line events before or after a target is generated
  • Type commands directly
  • Insert build-specific variables into commands
