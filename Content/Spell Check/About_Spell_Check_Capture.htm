About Spell Check

You can use spell checking features for the text in your images. These features include the following.

  • Spell Check Images This feature lets you open the Spell Check window and perform spell checking for the current file, all open files, and all files in the same folder. As questionable words are identified, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings. See Spell Checking Images.
  • Add Words to the "Ignore" Spell Check List This feature lets you manually add specific words to a list so that they are not flagged as having a questionable spelling in the file again. However, additional occurrences of the word in other files will still be flagged as having a questionable spelling if you use spell check in those files. See Adding Words to the Ignore Spell Check List.
  • Edit Custom Dictionaries This feature lets you modify the words that you have added to a custom dictionary. You can add words to a custom dictionary when you spell check files and click the "Add to Dictionary" button for highlighted words. A separate dictionary is created for each language used. See Editing Dictionaries.
