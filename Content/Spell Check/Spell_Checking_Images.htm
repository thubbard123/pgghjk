Spell Checking Images

This is a feature that lets you perform spell checking for the current image, all open images, or all images in the same folder as the current one. As questionable words are identified, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings.

how to spell check images
  1. Open the image.

    • Ribbon In the Spell Check section click the Check Spelling button .

    • Tool Strip Select Tools>Spell Check Window .
    • Keyboard Shortcut Press F7 .

    The Spell Check Window opens.

  3. Click the down arrow next to the Start Spell Check field and select one of the following.
    • current document Read more… Capture checks the spelling in the active file only.
    • all open documents Read more… Capture checks the spelling in all files currently open in the interface.
    • documents in same folder Read more… Capture checks the spelling in all files located in the same folder.

    4. If you want to skip searching in the current file and move on to the next file, click .

  4. After Capture finds and highlights the first questionable word, a list of suggested spellings is displayed at the bottom of the Spell Check Window. If you want to change the spelling of the term, follow step 5. If you do not want to change the spelling of the word, follow step 6.
  5. You can type your own corrected word in the Suggestions field or you can select a word from the list (which is then automatically added to the Suggestions field). Then click one of the following buttons.
    • Change Read more… Capture changes only that occurrence of the misspelled word.
    • Change in All Files Read more… Capture changes that occurrence of the misspelled word, as well as any other occurrences of it in the rest of the files.
  6. If you want to keep the word as it is, click one of the following buttons.
    • Skip Read more… Capture ignores the spelling of the word (this occurrence only) and moves on to the next misspelled word. If other occurrences of the word are found, they will be flagged as questionable spellings.
    • Add to Dictionary Read more… Capture adds the word to the dictionary so that it is not flagged as a questionable spelling in the future. Capture then moves on to the next misspelled word.
    • Ignore Read more… Capture ignores the spelling of the word and moves on to the next misspelled word. Capture also adds the spelling of this word to the Ignored Words dialog (for this image only). This means that if other occurrences of the word are found in the image, they will not be flagged as questionable spellings.
    • Ignore in All Files Read more… Capture ignores the spelling of the word and moves on to the next misspelled word. Capture also adds the spelling of this word to the Ignored Words dialog (for all images included in the search). This means that if other occurrences of the word are found in any other images included in the search, they will not be flagged as questionable spellings.
