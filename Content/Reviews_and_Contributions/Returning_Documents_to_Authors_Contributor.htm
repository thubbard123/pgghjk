Returning Documents to Authors

After reviewing documents sent from an author, you can return the documents via email.

how to return documents to authors
  1. Open the review package containing the documents you reviewed. If you do not see the Review Packages window pane, select the Review ribbon. In the Packages section, select Review Packages.
  2. In the local toolbar of the Review Packages window pane, click .
  3. In the email message, click Send.
 note

As an alternative to the email method, you can use SharePoint integration or the external resources feature to save the file in a location where the author can access it. If you use either of these features to return documents to authors, you will have to notify them manually through email or other means. See About SharePoint Integration and About External Resources.
