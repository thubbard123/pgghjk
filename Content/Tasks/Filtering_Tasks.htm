Filtering Tasks

How to Filter Tasks

  2. Use any of the following in the toolbar to filter the tasks:

    Filter by Project
    1. Click .

    2. In the dialog that opens, click to the left of each Flare project that you want to include.

    3. Click Save and Finish.

      The Tasks page shows only the tasks that are associated with the selected projects. Also, the toolbar button displays in blue to show that filters are set.

         Filter by Date
    1. Click .

    2. In the dialog that opens, you have the option of completing the Start Date and/or End Date fields if you want to show tasks only within a certain range.

      You can use the icons to the right of each field to choose the date in a popup calendar.

    3. You can also click to the left of any of the listed due dates as a way to filter tasks.

    4. Click Save and Finish.

      The Tasks page shows only the tasks that are associated with the selected dates. Also, the toolbar button displays in blue to show that filters are set.

         Filter by User
    1. Click .

    2. In the dialog that opens, click to the left of each user (task owner or person assigned) that you want to include.

    3. Click Save and Finish.

      The Tasks page shows only the tasks that are associated with the selected users. Also, the toolbar button displays in blue to show that filters are set.

         Filter by Priority
    1. Click .

    2. In the dialog that opens, click to the left of each priority level that you want to include.

    3. Click Save and Finish.

      The Tasks page shows only the tasks that are associated with the selected priority levels. Also, the toolbar button displays in blue to show that filters are set.

         Clear Filters
    • Click . Any filters previously set are cleared, and all tasks are shown on the page.
         All Tasks or My Tasks

    In addition to the previous filter options, you can also use a toggle button to quickly move between All Tasks or My Tasks (i.e., tasks that are assigned to you).
Note

In addition to using the filter features, you can also limit the tasks on the page by using the search field. The search field searches the task title and description.
