Deleting Tasks

How to Delete a Task
  4. Click on the task, and in the toolbar select .
  5. In the message that opens, click Delete Task. The Confirm Delete dialog opens.
  6. Type the name of the task in the field and click Confirm.
 Notes

Deleting a task also removes attachments associated with the task, thus freeing space on the server.

If you want to remove the task from the Task Board, but plan to work on it sometime in the future, you probably want to move the task to the Backlog area instead of deleting it. See Moving Tasks.
