Importing Browse Sequences

how to import a browse sequence
    • Ribbon In the Content section select New>Advanced>Browse Sequence.
    • Right-Click In the Project Organizer, right-click on the Advanced folder and from the context menu select Add Browse Sequence.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Browse Sequence is selected.
  4. Find and select the FLBRS file that you want to import.
  5. Click Open. The Source File field now contains the path to the file that you are importing. Also, the name of the file is displayed in the File Name field.
  6. If you want to give the browse sequence a different name than that for the imported file, click in the File name field and replace the text.
  7. (Optional)
  8. (Optional) See Applying Conditions to Content.
  9. (Optional) See Associating Tags with Files.
  10. Click Add. The browse sequence is added and opens in the Browse Sequence Editor.
