 how to insert a quick character
  1. Make sure you have specified the quick character in the Character dialog. See Specifying Quick Characters.
  3. Open a file in the Translation Editor.
  4. Place your cursor in a segment where you want to insert the character or symbol.
    • Ribbon In the Symbols section, click the down arrow on the Character button and select Insert Quick Character from the list.
    • Ribbon In the Symbols section, click the down arrow on the Character button and select Insert Quick Character from the list.
    • Local Toolbar In the local toolbar of the XML Editor, click the down arrow next to the Character button and select Insert Quick Character from the list.
    • Keyboard Shortcut Press F11.
 how to insert a favorite character
  1. Make sure you have added the character to your list of favorites in the Character dialog. To do this, simply right-click on the character in the Character dialog and select Make favorite.
  3. Open a file in the Translation Editor.
  4. Place your cursor in a segment where you want to insert the character or symbol.
    • Ribbon In the Symbols section, click the down arrow on the Character button and choose a character or symbol listed under Favorites.
    • Local Toolbar In the local toolbar of the XML Editor, click the down arrow next to the Character button and select a character or symbol listed under Favorites.
  6. In the Symbols section, click the down arrow on the Character button and choose a character or symbol listed under Favorites.
  7. In the local toolbar of the XML Editor, click the down arrow next to the Character button and select a character or symbol listed under Favorites.

In addition to the options for inserting a favorite, recent, or quick character, the drop-down list from the Character button also lets you insert a non-breaking space or non-breaking hyphen.
