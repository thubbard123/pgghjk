Adding Input Captions

Input captions are used in conjunction with input boxes. When a user types into an input box and then presses enter, one of three caption types can appear on the frame. These captions are triggered by setting up the "Play Action Object" action in the input box's object properties. See Selecting Actions for Objects.

how to add an Input caption
  1. Open the frame where you have added an input box.

  2. From the Home ribbon or the local toolbar in the Frame Editor, select Objects>Interactive, and then select one of the following input caption types:

    • Input Failure Caption Select Input Failure Caption Mode to add a failure caption to the frame. This is a message that displays to users if they fail to complete the input box successfully within the specified number of attempts.
    • Input Hint Caption Select Input Hint Caption Mode to add a hint caption to the frame. This is a clue that displays to users when an action is performed (e.g., when a user hovers over an input box, when a user enters an incorrect answer but is still within the number of allowed attempts).
    • Input Success Caption Select Input Success Caption Mode to add a success caption to the frame. This is a message that displays to users if they answer an input box correctly.

    The cursor changes to small crosshairs.

  3. Click and drag to draw a rectangle in the frame. A bubble shape appears on the frame and points to the input box.

  4. (Optional) Move the shape. You can do this in two ways.

    move the Bubble and Its Pointer

    Click in the middle of the bubble and drag it to the appropriate location in the frame. This moves the entire bubble, including its pointer.

         move Just the Bubble

    Click on the small bubble icon in the callout and drag it to the appropriate location in the frame. This moves the bubble, but its pointer remains fixed in its original location.
  5. (Optional) Click the point (small circle) at the tip of the pointer and drag it to the appropriate location.
  6. Double-click the bubble shape.
  7. Expand the Text section.
  8. In the Text section, enter the text that you want to display in the caption.
  9. Select the Object ribbon, then use the buttons and drop-downs in the Text section to apply formatting to the text.
 What's Next?

You can continue adding different types of input captions to the input box, or you can move on to adding actions to the input box. Adding an action to the input box is necessary to make the input captions appear when the users perform actions. See Selecting Actions for Objects.
