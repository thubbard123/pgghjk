Adding Rollover Captions

Once you create a rollover caption, you can associate it to an object.

how to add a rollover caption
  1. Open the frame.
  2. In the Frame Editor, draw a shape (e.g., rectangle, circle) over the area of the frame to be associated with the caption. This shape represents the hover area. See Adding Shapes to Frames.

    • Ribbon  In the Insert section, select Objects>Interactive>Rollover Caption Mode.

    • Local Toolbar In the local toolbar of the Frame Editor, select Objects>Interactive>Rollover Caption Mode.

    • Right-Click Right-click the hover shape. In the pop-up menu, select Add Rollover Caption.

    The cursor changes to small crosshairs. Click in the frame and draw the bubble shape, which will serve as the caption.

  4. To change the caption text, do one of the following.

    Enter text Directly In the Caption
    1. Click on the bubble and start typing.
    2. Select the Home ribbon.
    3. In the Text section, use the font options to apply formatting to the text.
         Enter Text In the Object Properties Window Pane
    1. Click the shape.
    2. Expand the Text section.
    3. In the field, enter the text for the caption.
    4. Select the Home ribbon.
    5. In the Text section, use the font options to apply formatting to the text.
  5. (Optional) Change the look of the rollover caption, the hover shape, or both. See About the Appearance of Frames and Objects.
  6. (Optional) Move or resize either shape as necessary.

  7. If you already have a rollover caption, you can right-click on the object you want to add it to and select Pick Rollover Caption. The rollover caption is now associated to the object.

 notes

If an icon like this is displayed on a rollover caption, that means the caption is not yet connected with an object. After you associate the caption with an object, the icon will disappear.

Rollover captions are not supported on some devices, such as the iPad.
