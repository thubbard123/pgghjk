Adding Images as Objects

how to add an image as an object
  1. Open the frame.

  2. In the Insert section, select Insert Image.

    The Open dialog opens.

  3. In the Open dialog, locate and double-click the image file you want to add. The image is added to the frame as an object.
  4. To move the image object, click it and drag it to a new location on the frame.
