Grouping Objects

example

Example

Let's say you have a frame with three objects—a rectangle, a line segment, and an image object, like this:

You can hold down the CTRL key and select all three objects.

Then you can either click in the local toolbar or right-click and select Group Selected Items, which converts the three objects into a single object group.

 how to group objects
  1. Open a frame containing multiple objects.

  2. Hold down the CTRL key and click the objects that you want to group.

    OR

    Click and drag across the objects you want to select. A selection box highlights the objects.

    • Ribbon  In the Advanced section click .

    • Right-Click Right-click and select Group Selected Items.
 Note

After you group objects, any keyframes, effects, and effect keyframes added to each individual object will not be editable in the Timeline or Frame Editor, nor will they display in the output. The Shape Group Properties window pane opens when you click on the group, allowing you to assign keyframes, effects, and effect keyframes to the group. See Shape Group Properties Window Pane.
