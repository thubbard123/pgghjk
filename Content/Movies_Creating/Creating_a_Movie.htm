Creating a Movie

Creating a movie can be done in various ways, whether it is a standalone movie or included within a collection.

See the Movie

methods for Creating a Movie

Creating a movie can be done in various ways:

  • Record a Movie Simulate (record) your actions in a specified application or area, creating a movie with multiple frames. See Recording Movies.The easiest way to do this is to click Record in the Movie ribbon. See Recording Movies.
  • Create a Movie from a Media File You can create movies from images or videos. After creating the movie, you can enhance the frame and insert additional frames. See Creating Movies from Media Files.The easiest way to do this is to select File>New, then select Movie From Media File, and choose an video or image file on your computer.
  • Create a Movie from a Screen Capture You can capture an image on your screen, creating a movie with a single frame of that image. After creating the movie, you can enhance the frame and insert additional frames. See Creating Movies from Screen Captures.The easiest way to do this is to select File>New, then select Movie From Screenshot, position the capture area, and click the red button.
  • Create a Blank Movie You can create a movie that consists of a single blank frame. You can enhance the frame and insert additional frames. See Creating Blank Movies.The easiest way to do this is to select File>New, then select Blank Movie, and complete the fields in the Add New Movie dialog.
  • Import a Movie If you have created a collection, you can import an existing Mimic movie into it. See Importing Movies into Collections.The easiest way to do this is to open the Collection Editor, click Import Mimic Movie in the local toolbar, and complete the fields in the dialog that opens.
  • Import a PowerPoint Presentation In addition to creating a movie by recording your actions, you can also create a movie by importing a Microsoft PowerPoint presentation. See Importing PowerPoint Presentations.The easiest way to do this is to select File>New, then select Import PowerPoint, and complete the fields in the Import PowerPoint Wizard.
 Note

You can also produce a movie collection (see Creating a Movie Collection), which is simply a group of related movies (such as a video tutorial with multiple lessons). If you create movies within a collection, you can see the first frame of each movie in the Movies window. When you double-click one of those frames, that movie is loaded into the interface. You can have multiple movies open in the interface at the same time.
