Adding Titles to Movies

You can add a title to a movie that is included in a collection. The title is used to display the name of the movie in the navigation of toolbar in the output. If you do not specify a title for the movie, the movie file name is displayed in the navigation instead.

how to add a title to a movie
  1. Open the collection containing the movie.
  2. In the Collection Editor, click the Sections tab.
  3. Select the movie to which you want to add a title.
  4. Click Change Title. The Section Title dialog opens.
  5. Enter a title and click OK. The title is displayed next to the movie file name in the Collection Editor.
