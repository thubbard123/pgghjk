Creating File Tags

After you add a file tag set, you can create multiple tags within it.

how to create a file tag

  2. Double-click the Advanced folder.
  3. Double-click a file tag set to open it. The File Tag Set Editor opens. In this editor, one file tag called "Tag1" is already provided for you.

  4. At this point, you can do a number of things, including the following.

    add a new tag

    You can add as many tags as you want, depending on your needs.

    1. In the local toolbar of the File Tag Set Editor, click the New item button . A new row is added.
    2. Press F2 on your keyboard.
    3. Replace the temporary tag name with a new one.

         change the tag type

    You can change the name of the tag type.

    1. In the local toolbar of the File Tag Set Editor, click the Tag Type: [Name] button.
    2. In the dialog, enter a new name for the tag type.

         rename an existing tag

    Maybe you want to use the existing tags, but you want them to have different names to reflect your purposes.

    1. Click the tag and press F2 on your keyboard.
    2. Type a new tag name.
    3. Press Enter on your keyboard.

         add comments to a tag

    You can add internal comments to a file tag to indicate the specific purpose of the tag.

    1. Click in the Comment cell of the tag.
    2. Press F2 on your keyboard.
    3. Type comments for the tag.
    4. Press Enter on your keyboard.
