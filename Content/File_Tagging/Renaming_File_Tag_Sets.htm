Renaming File Tag Sets

You can rename a file tag set that has been added to your project.

how to rename a file tag set
  2. Double-click the Advanced folder. The file tag sets (and other advanced files) in your project are displayed.
  3. Click on the file tag set that you want to rename.
  4. Press F2 on your keyboard. The file tag set name is highlighted.
  5. Type a new name for the file tag set and press Enter.
  6. If your file contains links to other files, you will be asked if you want to retain those links. If so, click Update Links.
