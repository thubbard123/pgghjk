Removing Inline Formatting

You can use the Remove Inline Formatting option to get rid of any local formatting in a selection or even an entire .

how to remove inline formatting

notes

By clicking , you remove all inline formatting, including any span classes that have been applied to whatever text you have selected to be unformatted.

This feature also works for local formatting inside tables. However, only the cells (e.g., background color) and their content (e.g., bold text) are affected. Any inline formatting within table tags (e.g., <table>, <tr>, <col>) is not affected.
