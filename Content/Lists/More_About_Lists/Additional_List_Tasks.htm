Additional List Tasks

In addition to the primary tasks you can perform related to lists, you can also do the following:

  • Adding Notes or Comments Between Items See Adding Notes or Comments Between List Items.
  • Continuing the Sequence See Continuing the Sequence of Lists
  • Merging See Merging Lists.
  • Restarting Numbering You can create a numbered list in a topic by using the Numbered List button in the Home ribbon. See Restarting Numbering in Lists.
  • Reversing See Reversing Lists.
  • Sorting After you have created a list, you can reorder the items in the list alphabetically. See Sorting Lists.
  • Unbinding See Unbinding Lists.
