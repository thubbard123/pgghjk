About Lists

lets you work with numbered and bulleted lists in a variety of ways.

ways to work with lists
  • Simple Lists You can create simple lists that are numbered or bulleted. See Creating Simple Lists.
  • Multi-Level Lists You can create numbered and bulleted lists with multiple levels (such as an outline). There are multiple ways to create a multi-level list. See Creating Multi-Level Lists.
  • Custom List Formats See Creating Custom List Formats.
  • Continue Sequence Use this to ensure that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content). See Continuing the Sequence of Lists.
  • Merge Use this to combine a list with another list immediately before it. See Merging Lists.
  • Notes or Comments (Paragraph Item) Use this to include comments between items without interrupting the flow of the list. See Adding Notes or Comments Between List Items.
  • Position You can position a list so that content wraps around it. See Using Absolute Positioning on Elements.
  • Restart Numbering Use this to specify a number to start a numbered list, or to start a selected item within a numbered list. See Restarting Numbering in Lists.
  • Reverse Use this to reorder the items in a list so that they appear in reverse order (i.e., first item is last, last item is first). See Reversing Lists.
  • Sort Use this option to reorder the items in a list alphabetically. See Sorting Lists.
  • Unbind (Remove) Use this to remove the list designation from the content so that it displays as regular text. See Unbinding Lists.
  • Edit List Styles You can modify the look of lists by making adjustments to the tags and style classes used for them. By editing list styles, you can affect things such as the alignment, background, bullet images, and indentation of lists. See Editing List Styles.
 quick list drop-down menu

You can also set the bullet type on a style. This includes selecting any of the types mentioned, plus more. See Specifying a List Type for a Style.

 advanced lists with autonumbers

In addition to using the quick list drop-down menu, you can use autonumbering to create more advanced types of lists. Autonumbering is an especially useful method for creating volume, chapter, section, and page numbering for printed outputs. For more information see About Autonumbers.
