Creating Simple Lists

You can create simple (single-level) numbered or bulleted lists using 's quick drop-down menu button. From this menu, you can select various list formats (e.g., circle bullet, square bullet, regular number, alpha number, Roman number). The image on the button depends on the most recent action that you have used. You can click the image to quickly apply that type of list format to the selected content. Otherwise, you can click the down arrow and select one of the other list formats available. See About Lists.

You can create simple lists that are numbered or bulleted.

  4. After providing the content for a list item, press Enter. Then enter the next list item text, and so on. To exist the list format, press Enter again on a blank list item.
 notes

If you want to create a list with multiple levels (such as an outline), see Creating Multi-level Lists.

You can also create simple lists by using autonumbering. See About Autonumbers.
