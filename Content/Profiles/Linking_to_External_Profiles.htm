Linking to External Profiles

provides you with an initial profile ("MyProfile") to help get you started. However, you can also link to external profiles. You might want to do this if you are working on a project and have a profile that is saved to an external location that is specific to that project, or if you are sharing a profile with a team.

how to Link to an External profile
