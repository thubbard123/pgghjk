Deleting Profiles

If you have created a profile that you no longer want to use, you can delete it. Deleting the profile sends it to the Recycle Bin.

However, if you are working with an external profile, such as one stored on a shared drive, you may only need it for a short period of time (see Linking to External Profiles). For this reason, you can remove the profile from Capture when you are finished using it. This breaks the link between Capture and the profile, and removes it without permanently deleting it. You can add it back into Capture at any time.

how to delete a profile
  1. Open the profile that you want to delete.
  2. In the local toolbar of the Profiles Editor, click . The Delete or Remove Profile dialog opens.
  3. From the drop-down, select Send the file to the recycle bin.
  4. Click OK.
 how to Remove a profile
  1. Open the profile that you want to remove.
  2. In the local toolbar of the Profiles Editor, click . The Delete or Remove Profile dialog opens.
  3. From the drop-down, select Remove the file from Capture's list without deleting.
  4. Click OK.

You can only remove external profiles. If you need to remove a local profile, you must delete it.
