Searching a DotNet Help System

End users have a variety of options when searching a system.

Search Experience

search experience

The user interface for the system is provided by a proprietary viewer (i.e., the Help Viewer) which is designed specifically for use with Windows .NET applications. When an end user searches a system for information, the results show both the topic rank and title. Search highlighting also lets users quickly pinpoint the matches in topics. How to Search

how to search
  1. Open the  system.
  2. In the navigation pane, click the Search pane.

  3. In the search field, type your query. Then click Search or press Enter.

  4. In the search results list, double-click one of the topics to open it. Search highlighting reveals the matches in the topic.
