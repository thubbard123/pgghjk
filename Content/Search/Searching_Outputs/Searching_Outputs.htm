Searching the Outputs

Depending on the output type of your target, there are different search features available.

  • DotNet Help Delivers output for use with Windows applications. See Searching a DotNet Help System.
  • HTML Help Delivers output in the proprietary Microsoft Compiled HTML Help format as a CHM file. This format provides a built-in search engine. See Searching a Microsoft HTML Help System.
  • HTML5 Delivers browser-based output using the HTML5 specification. See Searching an HTML5 System.
  • WebHelp Delivers browser-based output. See Searching a WebHelp System.
  • WebHelp AIR Delivers output that is similar to . It requires direct integration with Adobe AIR. See Searching a WebHelp System.
  • WebHelp Mobile Delivers web-based, XHTML output to mobile devices. See Searching a WebHelp Mobile System.

When entering a search query, each output type supports different search methods and syntax. See Search Methods and Syntax.
