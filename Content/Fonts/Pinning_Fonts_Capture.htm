Pinning Fonts

How to Pin Fonts from the Object Properties Dialog

  1. In the Capture Editor, double-click the object (not the image). The properties dialog for the object opens.

  2. Select the Rich Text tab or the Font tab of the properties dialog.

  3. In the [Font Family] field, select the font you want to pin. A pin appears to the right of the font. You can pin fonts from any font group.

  4. To pin the font or font set to the font list, click . The pin changes to to show that the font or font set has been pinned to the font list. Pinned fonts and font sets are also added to the Pinned Fonts group at the top of the font list.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

 How to Pin Fonts from the Popup Text entry window

  1. In the Capture Editor, click the object (not the image).

  2. Do one of the following:

    • Start typing content.

      OR

    • Press F2.

    A popup text entry window displays.

  3. In the toolbar at the top of the popup window, click the down arrow in the Font Family field.

  4. In the [Font Family] field, select the font you want to pin. A pin appears to the right of the font. You can pin fonts from any font group.

  5. To pin the font to the font list, click . The pin changes to to show that the font has been pinned to the font list. Pinned fonts are also added to the Pinned Fonts group at the top of the font list.

  6. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

 How to Pin Fonts from the Font Family Field

  1. In the Capture Editor, double-click the object (not the image). The properties dialog for the object opens.

    • Ribbon In the Text section click the down arrow in the Font Family  field.

    • Text Format Toolbar Click the down arrow in the Font Family field.

  3. Hover over the font you want to pin. A pin appears to the right of the font.

  4. To pin the font to the Font Family field, click . The pin changes to to show that the font has been pinned to the font list. Pinned fonts are also added to the Pinned Fonts group at the top of the Font Family field.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.
