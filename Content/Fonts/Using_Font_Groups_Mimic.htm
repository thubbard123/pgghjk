Using Font Groups

Font groups organize fonts. These groups are available in the Font Family field and on the Text section in the Object Properties window pane. Font groups are available for pinned fonts and all system fonts. You can collapse or expand any font group. This is a way to limit the list of fonts to those that you use the most and hide from view those that you do not use.

example how to Use font groups in the Object Properties Window Pane

  1. In the Frame Editor, click the object (not the image). The Object Properties window pane opens.

  2. Expand the Text section.

  3. Select a font that you want to use in the interface from a group:

    • Pinned Fonts The Pinned Fonts group is a way to create a list of your favorite fonts so they are is easier to find and select in the future.
    • All Fonts The All Fonts group displays all system font families.
