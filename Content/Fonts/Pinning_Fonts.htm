Pinning Fonts and Font Sets

How to Pin Fonts and Font Sets from the Font Properties Dialog

    • Ribbon At the bottom of the Font section click .

    • Keyboard Shortcut Press CTRL+SHIFT+B.
    • Right-Click Select the Font tab.

    The Font Properties dialog opens.

  3. Select the Font tab. From the Font field, select the font or font set you want to pin. A pin appears to the right of the font or font set. You can pin fonts from any font group.

  4. To pin the font or font set to the font list, click . The pin changes to to show that the font or font set has been pinned to the font list. Pinned fonts and font sets are also added to the Pinned Fonts group at the top of the font list.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

    Example

    Three fonts have been added to the Pinned Fonts group. You can still see the fonts in the All Fonts group, as well.

 How to Pin Fonts and font sets from the Font Family Field

  2. In the Font section click the down arrow in the Font Family field.

  3. Hover over the font or font set you want to pin. A pin appears to the right of the font or font set.

  4. To pin the font or font set to the Font Family field, click . The pin changes to to show that the font or font set has been pinned to the font list. Pinned fonts and font sets are also added to the Pinned Fonts group at the top of the Font Family field.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

    Example

    Three fonts have been pinned to the Font Family field. They are visible in the Pinned Fonts group. You can still see the fonts in the All Fonts group, as well.

 Note What's Next?

After you complete these steps, you can select fonts and font sets from the Pinned Fonts group when applying them to content. See Selecting Fonts.
