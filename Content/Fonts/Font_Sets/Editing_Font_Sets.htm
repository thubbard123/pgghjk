Editing Font Sets

You can edit any font sets you have created.

how to Edit a font set

    • Ribbon At the bottom of the Font section click . The Font Properties dialog opens. Click Font Sets.

    • Keyboard Shortcut Press CTRL+SHIFT+B. The Font Properties dialog opens. Click Font Sets.
    • Stylesheet Editor In the local toolbar of the Stylesheet Editor, click Manage Font Sets.

  4. Click . The Define Font Set dialog opens, where you can see the fonts that are currently in your font set.
  5. To edit your font set, do the following:
    1. If you want to add a font family to the font set, double-click each font family that you want to add, or select a font family and click to add it to the set. Fonts you have selected appear in the list on the right side of the dialog.
    2. If you want to reorder the font families in your font set, select it from the list of selected font families on the right side of the dialog, then use the and buttons to rearrange their order. When you are finished, the first font family should be the preferred font family that you want to be used, then the second, then the third, and so on.
    3. If you want to remove a font family from the list of selected font families, select it and click . You can also double-click a font family to remove it from the font set.
  7. Click OK in the Font Set Manager dialog.
  8. Click OK in the Font Properties dialog.
 example notes

You cannot edit the name of a font set after you create it.

Editing a font set does not automatically update styles or local formatting that use the original font set. You will need to reapply the font set if you want to use the new fonts in existing documentation.

 What's Next?

After you create or edit a font set, you can select that font set when applying fonts to content. You can also select that font set from the Defined Font Sets group in the Font Family field in the Font section of the Home ribbon. See Selecting Fonts.
